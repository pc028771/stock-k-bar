# ch3-3 — 3.3 夏季可轉債實戰案例

> 時長：10:22 | cues：311 | triggers：63

## 核心主題（1-2 句）

從可轉債角度找夏天股票的邏輯：有發債的公司更有拉抬意願；「低檔曾漲停」CB 案例演繹（昇達科二成功、汎銓一失敗）；停損設月線。

## 講師說了什麼（重點段落，含 timestamp）

- **00:07 — CB 與夏季拉抬意願**：夏天股票若有發行可轉債，拉抬意願更強，因為股價漲上去後債券可換成股票在市場賣掉就不用還錢，所以公司派希望股價往上漲。（@00:19）
- **00:46 — 每週末可轉債低檔曾漲停案例**：小哥每週末推出「可轉債低檔曾漲停案例」作為選股工具。
- **01:00 — 昇達科二成功案例**：低檔曾漲停，一開始沒量後有量（典型春轉夏），目前還沒入秋，有發債且公司派偏多，從春轉夏走到「炎夏酷夏」。（@01:00）
- **02:02 — 汎銓一失敗案例**（68301）：低檔曾漲停後沒馬上攻上去，因川普利空造成突破失敗；遇到這種狀況「設破月線就停損」。一創新高才重新進場。（@02:06）
- **03:16 — 博智一案例**：第三個案例（字幕部分，具體內容需截圖）
- **08:00 — 可轉債 CB 選股工具**：「低檔曾漲停且還沒大量轉換」是找春轉夏形態的工具。市值與 CB 委買關係：委買賣是市值的目標。（後詳述在 ch9-3）
- **09:00 — 不是每個夏天都順利**：有些突破失敗還需整理，有的很快結束；不是每個夏天都一帆風順。

## 量化條件 / 具體數字

- CB 夏季停損：破月線停損（定性）（@02:51）
- 重新進場條件：創新高即重新進場（@03:03）

## 個股案例（如有）

- 昇達科二：低檔曾漲停，春轉夏成功，走到炎夏（@01:00）
- 汎銓一（68301）：低檔曾漲停後突破失敗（川普利空），破月線停損，後創新高再進場（@02:02）
