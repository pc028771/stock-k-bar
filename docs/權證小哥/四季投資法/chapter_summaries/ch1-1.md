# ch1-1 — 1.1 市場真相：為什麼你需要四季導航系統？

> 時長：09:53 | cues：304 | triggers：54

## 核心主題（1-2 句）

用 2021 年航運股案例解說「看不懂季節」的痛點，建立四季框架的必要性；四季 = 主力資金運作的四個階段（吃貨→拉抬→出貨→離場）。

## 講師說了什麼（重點段落，含 timestamp）

- **00:45 — 四季框架介紹**：春耕夏耘秋收冬藏，透過季節框架簡化複雜市場、掌握行情結構。
- **01:14 — 利多滿天飛股票不動的原因**：以 2021 年航運股末段為例，營收持續創高但股價漲不動，因為已進入「秋季出貨期」。（@01:25）
- **02:06 — 一買就跌、一賣就漲的原因**：買到秋天或冬天的股票，賣到剛進夏天的股票，根本是看不懂季節。（@02:32）
- **02:56 — 行情崩盤**：看似大好卻無預警崩盤，也是季節轉換識別問題。（@02:58）
- **04:08 — 四季本質**：本質上是主力資金運作四個階段：春=吃貨（低量價穩，散戶無興趣）、夏=拉抬（價量齊揚）、秋=出貨（利多掩護散貨）、冬=離場（股價修正崩跌）。
- **07:00 — 3 步 SOP**：定位（Identify）→ 策略（Execute）→ 觀察（Monitor）；核心問句：這檔股票處於哪個季節？
