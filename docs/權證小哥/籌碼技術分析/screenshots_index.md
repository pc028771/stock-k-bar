## L3 SOP 執行記錄

- player VHS 已 init、master playlist 可讀（無需強制 play）
- iframe context XHR (`iwin.XMLHttpRequest`) + `responseType='blob'` + explicit `async=true` 是關鍵
- main window `fetch` 會被 403（Referer 不對）
- segment via Node local HTTP server (`localhost:18765`) POST 上載
- ffmpeg `-pix_fmt yuvj420p -q:v 2 -frames:v 1 -update 1` extract single frame
- chapter 間 cooldown 30s、第 3 章後 60s
- segment 間 cooldown 2-2.5s
