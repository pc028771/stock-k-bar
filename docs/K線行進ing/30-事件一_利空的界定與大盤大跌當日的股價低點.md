# 【K線事件判斷】(一) 利空的界定與大盤大跌當日的股價低點

## 利空的定義

**真正利空**：對經濟面影響一定程度的事件

### 突發性利空
- 市場根本來不及反應的事件
- 例：九二一地震、兩顆子彈、武漢肺炎（突發部分）

### 可預期利空
- 知道會有事件，時間未到
- 例：英國脫歐、美國總統大選、希臘破產

### 不算利空
- **美股下跌不算台股利空**——只是新聞報導，影響市場散戶心理，**不影響股市本質**

## 攻擊股遇大盤大跌的判斷

> 「對於一檔攻擊的股票來說，遇到某日台股大跌這是很正常的事。已經進入『攻擊階段』的走勢，表示這一檔股票大部分流通的籌碼都已經在攻擊者的手上，此時自己把股票砍出去當然沒有散戶願意承接，所以大盤大跌的當天往往會自己又拉上去。」

## 核心判斷：環境利空當日 K 線的低點

> 「如果出現了大盤有幅度下跌、客觀讓人以為是利空的狀態出現（其實並不是真正的利空），那麼，還有沒有要攻擊？這種特殊日子原本存在著拉抬意願的背景，**K 線當天低點就是重要的判斷關鍵**，前提是之前有過攻擊力量進駐過。」

**兩種情境同樣判斷**：
1. 大跌當日仍創新高 → 攻擊未停止；隔天破當日低 = 攻擊失敗
2. 大跌當日逆勢拉長紅 → 主力顧股；隔天破當日低 = 攻擊失敗

## 消息面同樣判斷

> 「如果突然出現了個股有代表意義的利多利空消息，這一根 K 線的低點一樣不宜跌破。」

## 對系統的啟示

1. **`market_regime_drop_day` feature**：當日大盤跌幅 > X% → 標記
2. **大跌當日的個股低點 = 隔日停損點**：features 加 `prev_market_drop_day_low`
3. **如果大跌當日個股逆勢漲**：標記為 `attack_intent_strong`，隔日跌破當日低 = 強烈出場訊號
4. **個股利多/利空當日 K 線低點同邏輯**：未來整合消息資料時可加 column `news_day_low`
5. **「美股下跌不算利空」**：禁止把美股當日跌幅當作 regime indicator
