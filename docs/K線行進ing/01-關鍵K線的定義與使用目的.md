## 關鍵K線定義

**關鍵K線 = 改變趨勢的第一根K線**

亦即：這根K線之前與之後，趨勢不一樣。

趨勢分為三類：**多頭、空頭、盤整**

任何兩類之間的切換都稱為「趨勢改變」：
- 盤整 → 多頭 / 空頭
- 多頭 → 空頭 / 盤整
- 空頭 → 盤整 / 多頭

## 關鍵K線的位置不限於買賣點

關鍵K線在各種技術分析理論都存在，本質是「價格趨勢變化的主要位置」。

例：透過「季線扣抵計算」可預判趨勢轉折位置。當季線快要從急跌段扣抵、指數撐過幾天就會由下彎轉上揚 → 那一根撐住的K線就是關鍵K線。

**重要提醒：關鍵K線的形狀並不重要**——並非長紅或長黑才算得上關鍵K線。

## 確認點：盤中 vs 收盤

「關鍵K線的確認點，是以盤中跌破或者突破為主，還是收盤價確認呢？」

**沒有標準答案**，取決於這根關鍵K線是基於「怎樣的技術分析研判技巧」：

| 結合的判斷技巧 | 確認方式 |
|---|---|
| 轉折K線中的「大敵當前」 | 盤中碰到中值價位就算，不必等收盤（但有變形，可更提前） |
| 型態學中的頸線 | 盤中突破就算（偶有收盤又回檔的狀況） |
| 季線扣抵 | 開盤跳上去就算 |

## 大盤 vs 個股的關鍵K線差異

| 對象 | 重點 | 用途 |
|---|---|---|
| 大盤 | 趨勢改變與否 | 只需判斷資金中期力量方向 |
| 個股 | 進場 vs 出場意義不同 | 進場要考慮未來走勢；出場只需眼前態勢結束 |

**進場 vs 出場的不對稱性**：
- 進場（買進、放空）：要考慮未來走勢是否如預期
- 出場（賣出、回補）：只需眼前態勢結束，不需考慮未來

對個股來說，K線判斷的深入運用目的 = **能否再提前一天反應**
- 賣的時候可以賣高一點
- 買的時候可以買再低一點

## 關鍵K線在 K線圖上的辨識

關鍵K線**不是出現了才用**，而是觀圖時要能辨識「未來可能成為關鍵K線」的位置。這些位置未來如果突破或跌破，技術分析上有特殊意義。

**範例情境**：
- 一個跳空突破的紅K（突破模式不佳）
- 隔日長黑K → **是關鍵K線**（將多方態勢直接變成整理趨勢）
- 同價位附近重複出現「紅K接續黑K」的現象 → **不是關鍵K線**（沒有改變趨勢），但代表「明顯的實質賣壓存在的位置」
- 當這種K線走勢進入「隨時一根長黑、季線就會下彎、頸線就被跌破」的狀態 → 若再出現黑K，那根才是關鍵K線

## 對系統的啟示

1. **關鍵K線不限於形狀**：我們系統目前的 `breakout_attack` 已經正確地不限制紅K/黑K，符合課程精神
2. **確認方式因 context 而異**：頸線型態用收盤（與 `neckline_break` 一致）、季線扣抵用開盤
3. **個股提前一天的價值**：這是 `ma60_neckline` STUB 要解的問題——透過 MA60 扣抵預判，能比單純頸線跌破早一天識別關鍵K線
4. **「實質賣壓位置」識別**：同價位反覆出現紅K→黑K，可作為 supply zone proxy（但不是關鍵K線本身）
