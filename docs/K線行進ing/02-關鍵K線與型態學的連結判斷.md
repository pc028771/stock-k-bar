## 型態學的力量：多方 vs 空方差別

**空方頸線跌破**：上檔的套牢賣壓是實質存在，賣壓未解套前都擋在攻擊者前方 → 頸線跌破即賣壓開始作用。

**多方頸線突破**：光是頸線突破還不夠，**重點是要有「攻擊上去的力量」才行**。

## 突破日 = 關鍵K線

> 「以型態學來說，突破就是突破，沒有等到收盤確認的必要。」

實戰上的「突破日分價量表攻擊量判斷」目的是更確定關鍵K線日有沒有攻擊力量；若等到收盤確認，又得隔天才能進場，**晚一天**。

## 多方範例：帆宣 6196（2020-07-08~09）

- 賣壓化解的區段中，季線於六月份才開始上揚（在此之前都是空方趨勢）
- 操作角度：**空方趨勢時不能進場**
- 用什麼指標或背離想「低買高賣」提前進場，最終都會失望
- 帆宣是後來有突破上漲的例子；市場上多的是低檔買了卻遲遲不漲的股票（例：神基 3005）

## 為什麼買進需要等頸線突破

> 「光只想著『買低價』的心態是不妥當的，重點應該要放在『會攻上去的股價型態』。」

頸線突破當天，才是關鍵K線的位置。在此之前，**永遠無法預期哪一天才有資金願意去拉抬這檔股票**。

## 空方型態（範例：旺宏 2337）

- 旺宏季線還沒下彎，但頸線已浮現 → 未來綠色圈圈就是可能出現關鍵K線的位置
- 空方操作要等「頸線跌破」才能進場
- 多方操作者：**雖然空方關鍵K線還沒出現，但這就是不能碰觸的股價型態**

## 預測關鍵K線位置

可以採用「事先劃出關鍵K線可能所在的位置」來幫助研判。

**結合方式遞進**：
- 型態學要點 + 關鍵K線
- + 移動平均線扣抵（更清晰）
- 唯獨**轉折K線無法事先預畫**，因為必須等到出現才能認定力竭原理的現象

## 對系統的啟示

1. **突破不等收盤確認**：型態學頸線突破應以盤中突破為準（與 §1 「確認方式因 context」一致）
2. **空方頸線跌破 vs 多方頸線突破不對稱**：空方賣壓存在已是實質，多方還需要「攻擊力量」
3. **不能碰觸的股價型態**：頸線浮現但尚未跌破 → 即使空方關鍵K線還沒出現，多方操作者也不能進場
4. **預畫關鍵K線位置**：未來可在 features 加 column「可能關鍵K線位置」標記
