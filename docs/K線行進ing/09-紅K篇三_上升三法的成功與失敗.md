# 【行進判斷】紅K篇(三) 上升三法的成功與失敗

## 上升三法定義

> **上升三法 = 股價拉出一根紅K之後，經過兩到三天的整理，再拉出一根長紅。**

整理形式可以是紅K緩慢推升、十字線、或上影線。

## 條件：必須在攻擊狀態（關鍵K線之後）

> 「行進判斷當然要背景，背景就是股價攻擊狀態中，**不能因為要滿足投資人想買低的願望，就把K線組合丟到低檔區域來運用**。」

## 上升三法成功

紅K → 兩到三天整理 → 再長紅 → 進入日出走勢（若在創新高則稱「日出攻擊」）

## 上升三法失敗 = 跌破整理低點

- 拉回跌破整理的十字/紅K低點 → 一開盤就得要處理掉
- 後續即使上去也不買回 → **這就是為什麼 N字戰法是錯誤觀念**：N字戰法只是把停損點拉開更大

## 「N字戰法」是錯誤觀念（明確禁止）

> 「請切記不能事後論的來塑造一個技術分析理論。」

N 字戰法的問題：
1. 把停損拉得更大
2. 可能誤用於非攻擊狀態的低檔
3. 事後論

## 對系統的啟示

1. **`rising_three_methods` 結構**：
   - K1：突破紅K（關鍵K線）
   - K2-K4：兩到三天整理（紅K緩慢推 / 十字 / 上影線）
   - 整理低點作為停損
   - K5：再一根長紅 → 確認上升三法成立
2. **失敗訊號 = 整理低點被跌破**：可作為入場後的早期出場條件
3. **禁止 N字戰法**：禁止「跌破整理後再買回」的回補邏輯
4. **適用範圍限制**：只在攻擊狀態（剛突破關鍵K線之後）才用，不能在低檔反彈時套用
