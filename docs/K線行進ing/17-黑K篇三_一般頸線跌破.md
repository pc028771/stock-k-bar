# 【行進判斷】黑K篇(三) 一般頸線跌破

**Source:** https://www.pressplay.cc/project/55DE90EBFBB634BE864F75703AB654DE/articles/19165ECE25C0F0C693BDCE4A215A4442
**Order:** 17 / 39 — **HIGH PRIORITY (refines neckline_break)**

## 頸線的精確定義（再次強調）

> **頸線 = 季線上揚(下跌)後的前一個高點(低點)。**

- 多方頸線：季線**上揚**後的前一個**高點**
- 空方頸線：季線**下彎**後的前一個**低點**

## 頸線跌破的判斷要件

**不一定要是長黑**：
- 季線下彎 + 頸線被跌破 → **頭部已經形成**
- 短的黑K意義也是一樣

**確認**：跌破後**有沒有站回**頸線
- 沒站回 → 頭部型態確立
- 站回 → 仍有賣壓化解的問題

## 頭部的本質

> 「頭部最大的問題並不是型態造成的，而是過往的高檔套牢賣壓出現，市場上沒有資金願意為這些人解套。」

頸線跌破 = 型態空方 + **賣壓明顯確定的意義**，從這一根黑K起點開始。

## 區間被跌破 = 頸線跌破的變形

- 短期之內頭部型態不明顯，過去一段時間有箱型整理
- 容易讓人誤以為只是區間
- **箱底跌破 + 季線下彎** = 頸線跌破的意義
- 跌破箱型同時季線下彎當然與頸線定義相符
- 過去一年的橫向整理讓人**失去戒心**

## 不攻擊 = 一種危機

> 「沒有意識到**不攻擊就是一種危機**。」

## 預判頸線跌破

> 「我們必須在季線還沒有上揚(下彎)之前，就已經知道會發生，這就得要具備**扣抵判斷的能力**。」

## 對系統的啟示

1. **`neckline_break` 需修正**（目前用 prior_low_20 是粗代理）：
   - 正確：「季線下彎後的前一個低點」
   - 需要兩步驟：(a) 偵測季線方向變化 (b) 找出該方向變化前的最近 swing low
2. **箱型整理 + 季線下彎 → 視為頸線跌破**：features 加 column `is_in_box_range`，當箱底跌破 + 季線下彎時觸發
3. **黑K大小不重要**：頸線跌破不需要長黑（修正入門中 `real_breakdown_after_range` 要求 body_pct >= 0.015 的代理）
4. **季線扣抵預判頸線跌破**：用 `ma60_rolling_off_close` 預判季線方向，可在跌破前一天就標記為「即將頸線跌破」
5. **長期橫向 = 警訊**：「不攻擊就是危機」可作 scanner 扣分（超過 N 天無新高 + MA60 即將下彎）
