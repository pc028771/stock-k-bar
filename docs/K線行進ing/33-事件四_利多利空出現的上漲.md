# 【K線事件判斷】(四) 利多利空出現的上漲

## 利多 + 紅K 的判斷

> 「假如是利多，股價卻是黑K那就不用多問，肯定是藉著這個消息出脫持股；如果是利空卻走出長紅，那利空出盡的可能性比較高。」

但有例外。

### 利多伴隨紅K 的進一步判斷

> 「當利多出現才跟著出現的紅K，要考慮的面向很多。」

要點：
- 不能用「利多一定會被出貨」這種想法誤導
- 還是要回到攻擊角度判斷
- 利多紅K之後**不應該回拉讓人有更低買進機會**

範例：神基(3005) 利多殖利率消息 + 逆勢長紅 → 後續無回檔 + 上升三法 → 攻擊持續

### 反例：利多後遇壓或隔日黑K

範例：為升(2231) 毫米波利多 + 逆勢漲停 → 但遇前高壓力 + 隔日黑K → 不應有太多期待

## 利空 + 紅K 的判斷

> 「當利空發生的時候，當然一開始就是下跌，可是總會有一天止跌……而這個日子往往就會是紅K。」

關鍵：
- **進場 = 投資 vs 交易角度的選擇**
  - 投資角度：不宜進場（風險太大）
  - 交易角度：停損明確（再破底）→ 可考慮
- 連續跳空下跌段 = **賣壓中空區段**

範例：旭富(4119) 失火四根跌停 → 紅K 止跌 → 低點作為明確停損

## 高價股的利空特例

> 「高價股在利空事件的 K 線表現又會跟一般中低價股有很大的差別，因為如果就這樣下跌，也不會有散戶願意去承接，反倒只是法人自己殺自己而已。
> 股價再往上拉之後才給人一種**利空出盡的感覺**，等到人們不擔心了往往才是下跌的出現。」

範例：漢微科 利空後反而拉抬，但後續下跌。

## 對系統的啟示

1. **「利多紅K 不可回拉」**：features 加 column `news_bullish_red_low`，後續日 close 跌破即出場
2. **利空後紅K = 停損明確的進場時機**：與「人去樓空」例外。但限「交易模式」非「投資模式」
3. **連續跌停跳空段 = 賣壓中空**：可作為將來突破時的「無套牢」優勢
4. **「利空出盡感」的警示**：若高價股在利空後拉抬但無攻擊力量，反而是出貨陷阱
