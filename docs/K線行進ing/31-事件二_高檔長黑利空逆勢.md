# 【K線事件判斷】(二) 出現攻擊的股價並非隨機漫步：高檔長黑、利空逆勢

## 觀念釐清

> 「沒有大資金要攻擊的股價，往往是**隨機漫步**的，無法預測股價的方向。」
> 「**『多空循環』只是一種說法**，實際上股價在隨機漫步的狀態下，沒有多空循環，股價在有攻擊力量存在其中時，也不會是多空循環出現。」

## 高檔長黑伴隨利多 = 刻意為之

> 「如果出現了高檔長黑，當然就是刻意為之。」

範例：申豐(6582) — 月營收 YOY 大幅成長 + 高檔長黑 → 拉抬資金已出脫 → **人去樓空**，未來只剩隨機漫步。

## 利空逆勢的兩種情境

### 情境一：利空當下還在攻擊狀態
- 主力會繼續拉
- 原因：必須造成「利空鈍化」假象，未來才能出脫

### 情境二：利空發生後馬上止跌
- 不是隨機漫步
- **刻意的力量拉抬** → 反彈會有力量

範例：欣興(3037) 失火兩次 → 利空後股價馬上止跌 = 資金意圖。

## 利空後沒有反彈 = 真的人去樓空

範例：旭富(4119) 失火之前股價已弱勢盤整 → 失火後連續跌停 → 反彈很難回原價位。

## 核心判斷

> 「對比沒人要的股價，**不用任何利空消息，照樣可以走進空頭趨勢**。」

→ K 線可以告訴你「有沒有資金力量身在其中」，不一定要有消息。

## 對系統的啟示

1. **「利多+高檔長黑」= 主力出貨**：features 加 `news_bullish_with_high_long_black`
2. **利空後個股 K 線力量**：
   - 利空後馬上止跌反彈 + 量價配合 = 有資金（保留持股）
   - 利空後連續跌停或反彈乏力 = 人去樓空（出場 / 避開）
3. **「多空循環」是錯誤概念**：禁止用「均值回歸」、「乖離率」、「KD 黃金死叉」等基於循環概念的指標
4. **「沒人要 = 隨機漫步」**：scanner 排除「成交量極低 + 無方向」的個股
