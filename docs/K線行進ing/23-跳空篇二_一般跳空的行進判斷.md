## 跳空在 K 線中的核心地位

> 「就連轉折組合都得要用到這個力量，例如**雙鴉躍空、跳空反轉、母子雙星、夜星、晨星、島狀反轉**……」
> 「學會了跳空的意義與行進延續的判斷，已經等於學會了一半以上的 K 線判斷了。」

## 「跳空」討論範圍

- 收盤後依然有缺口的 K 線（不是盤中跳空、收盤回補）
- 向上跳空但盤中回補 → 屬於黑K範圍

## 一般跳空（不具攻擊意義）的兩個背景

1. **消息背景**：利多消息帶動的跳空，不具備力量意義
2. **環境背景**：大盤氣氛樂觀帶動的跳空，意義微弱但至少還在

→ **K 線無法直接看出消息/環境背景**，需要額外對照新聞。

## 攻擊缺口的條件

> 「**沒有套牢賣壓的向上跳空才稱之為攻擊缺口**。」

範例：駐龍(4572) 三天前跳空漲停一條線，但 90 元以上有大量套牢 → 不算攻擊缺口

## 跳空後的行進判斷

1. **缺口不能回補**（最簡單觀察）
2. **接下來的高低點**：採取緩步推升 → 拉抬力量並不明確（沒有強烈積極的意願）

## 「賣壓化解」的重要性

> 「我們不應該在賣壓化解之前猜測這檔股票有沒有能力越過，而是應該要**等到越過了之後才能確定有攻擊意願**。」

## 上影線在套牢區的意義

> 「**剛**創新高的上影線才代表攻擊，如果遇到了過往的套牢區段，不管是不是上影線，都有壓力存在。」

→ 上影線 + 創新高 = 攻擊；上影線 + 套牢區 = 壓力

## 向下跳空的兩種

> 「向下跳空並不區分為攻擊與否，但行進的判斷上，向下跳空有：
> - 一般的跳空
> - **消除攻擊意義的跳空**——當這個向下跳空把過去的攻擊跳空給消滅了。」

→ 攻擊跳空被消滅 = 沒有攻擊意願 → 警示「明顯幅度的回檔」

範例：智邦(2345) 攻擊跳空後橫向，再出現向下跳空消除攻擊跳空 → 可以避開回檔

## 對系統的啟示

1. **攻擊跳空精確條件**（補入門 E1 的 `gap_open` 粗代理）：
   - 向上跳空（open > prev_high）
   - **且該跳空上方無套牢區**（用 `overhead_supply_layer` 為 0 過濾）
   - **且該跳空使股價創新高**（close 或 high > prior_high_60）
2. **「消除攻擊跳空」出場訊號**（新增 exit condition）：
   - 過去 N 天內有攻擊跳空 = `gap_attack` 標記
   - 後續若出現向下跳空使該攻擊跳空的位置被回補 → 出場
   - 與目前 `gap_fill` (同日 close < prev_close) 不同：這是**「跨日消除」**
3. **緩步推升 = 沒有強烈意願**：scanner ranking 中，連續紅K但 body 縮小（緩步）扣分
4. **上影線雙重意義**：features 加 column `is_at_overhead_zone`，組合 `upper_shadow` 才能判斷
