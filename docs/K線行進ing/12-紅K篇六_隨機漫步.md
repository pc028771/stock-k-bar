# 【行進判斷】紅K篇(六) 隨機漫步

**Source:** https://www.pressplay.cc/project/55DE90EBFBB634BE864F75703AB654DE/articles/9CA1A326EC68E69E01092868C005B0ED
**Order:** 12 / 39

## 隨機漫步定義

> **隨機漫步 = 突破紅K後股價上上下下、無跡可尋，看起來像有攻擊但不積極。**

定義二：「市場上沒有大資金打算要來好好拉這一檔股價。」

## 判斷依據

1. **無故意做量**：有新高卻沒做量 = 沒有要拉抬的意願
2. **量增是「做量」的假象**：拉抬資金會自己賣給自己創造價漲量增；後續沒量 = 沒打算拉
3. **時間成本不划算**：現在買沒人要攻擊，以後再突破風險變高（本益比偏高）

## 量的真相

> 「一檔真的有意願拉抬的股價，往往會做量。做量很簡單，就是自己賣給自己創造價漲量增的假象。」
> 「市場散戶沒這麼厲害，一檔本來冷門的股票一漲了就都跑去當沖，這不可能。」

→ **可以從當沖張數看出做量的假象**。

## 進入隨機漫步後的應對

> 「如果在這個隨機漫步的過程中，我會找個有漲的紅K來出場，把資金投入到真的有打算攻擊的股票。」

## 對系統的啟示

1. **隨機漫步偵測**：
   - 突破紅K 後 N 天（如 5-10）內
   - 沒有再創新高 或 創新高無量
   - 上下波動但無方向
2. **量的角度**：
   - 突破當天無量 → 沒有攻擊意願（warning）
   - 突破後創新高無量 → 隨機漫步（出場訊號）
3. **時間成本**：scanner ranking 可加入「持有 N 天無進展」的扣分
4. **當沖張數**：未來可整合 intraday 資料偵測「做量假象」
