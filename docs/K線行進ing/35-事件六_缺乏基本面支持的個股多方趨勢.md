## 四種「沒基本面但K線多方」的類型

### 一、短期看似有基本面（業外收益）
範例：正文(4906) — Q3 EPS 3.54 元主要是業外釋股，本業只有 1.05。
- K 線：紅K 之後緊接黑K 屢屢出現在同價位 → 實質賣壓
- 等到月營收真正成長後才開始攻擊

### 二、中期沒有營運轉機的多方
範例：車王電(1533) — Q4 月營收 YOY 169% 只是單月特例，1-2 月恢復原狀。
- 量越來越低 → 籌碼集中在主力手上但**主力無散戶接盤**
- 「沒有量，表示散戶沒有參與，所以主力應該頭很痛」

### 三、突發事件讓本質下降的多方
範例：和碩(4938) — 立訊聯手好消息 → 1 月營收公布大幅衰退 → 向下跳空 → 攻擊結束
- **「不適合拉回承接」**：拉回 = 股價已轉弱
- 向下跳空缺口 = K線事件中重要性最佳證明

### 四、只因題材而拉抬（無基本面）
範例：浪凡(6165)、青雲(5386)、撼訊(6150) — 比特幣概念、AI 題材
- 股價漲到 90 元 EPS 卻是負數
- 「沒有每股盈餘更不用談長期投資」
- 多頭市場主力被出貨完，接下來幾年不一定能解套

## 重要規則

> 「**獲利趨勢就是股價的趨勢**，完全不需要把你的資金投入在還看不到基本面變化的個股之上。」

> 「**價穩量縮**——技術分析不是用成語來判斷的。」（禁止）

> 「不要再相信市場有人教的什麼價穩量縮。」

## 量越來越低的判斷

- 量越來越低 + 股價持續創新高 = 主力籌碼集中 + 散戶不參與 → **危險**
- 與「跳空漲停一條線」的量小邏輯**不同**：那邊是當天的量小，這邊是趨勢的量縮

## 對系統的啟示

1. **基本面整合的需求加強**：scanner 應加入：
   - 連續多季 EPS > 0
   - 月營收 YoY 持續成長（至少 2-3 個月）
   - 排除「單季業外收益」
2. **量縮 + 股價創高 = 警示**：當前 `volume_ratio` 因子只看當日量，應加入「過去 N 天量縮趨勢」（如 vol_ma20_slope < 0）
3. **「拉回不買」鐵則**：scanner 不應有「拉回買進」型訊號（如 RSI 超賣、Bollinger 下軌）
4. **「價穩量縮」禁止**：明確 `forbidden_signals` 清單應加入此項
5. **題材股檢測**：features 加 `is_topic_stock`（EPS < 0 但股價漲幅大）警示
