## 多空波動原理

> **多方狀態 = 在波動的過程中產生的每一段高點、低點，都比前一次的高低點來得高。**

→ 只要股價沒有跌破前一次的低點，就表示股價依然在多方狀態。

## 短線 vs 中期持有的差別

| 屬性 | 短線 | 中期 |
|---|---|---|
| 優點 | 持有最飆那一段 | 大波段利潤 |
| 缺點 | 不能忍受回檔 | 必須忍受回檔 |
| 出場 | 攻擊結束 | 多空波動原理（前低跌破） |

## 多方結束 ≠ 空頭開始

> 「**多方結束不代表股價就是空頭了**，往往有很大的機率只是轉為中期的盤整。」

頸線完整定義（再次強調）：
> **頸線 = 季線下彎後的前一個低點，且此低點上方有三個月的套牢。**

## 中期持有的真正挑戰：第一次回檔

> 「真正的困難是出在股價突破後往上攻擊出現，第一次的往上可以看得出來……而波段操作者卻會遇到回檔，**第一次波動下的低點會到哪？沒人知道**，也很難忍受。」

對比：
- 世紀鋼：第一次回檔 = 4 根跌停板，但後續大漲
- 上緯：第一次回檔直接跌回起漲點

→ **無法事先確定回檔幅度** = 中期持有的根本挑戰

## 突破前高的進場意義

> 「**突破前高（不是型態突破）是確認多方持續進行中**，因此股價的第一次突破前高，表示之前已經先有第一個拉回的低點了。」
> 「**突破前高是給之前有做到第一段的人使用的**。」

範例：亞信(3169) 突破前高 → 跳空攻擊 → 第一次回檔 → 再突破前高 → 完美波段

## 對系統的啟示

1. **`prev_low_break_exit` 補強**：中期持有的出場條件 = 跌破前次低點（多空波動）
2. **「第一次回檔」識別**：features 可加 column `is_first_pullback`，計算「進場後第一次低點」
3. **「下一個買點」=突破前高**：與入門「下一個買點」概念一致——適合在出場後等待再次突破才回進
4. **頸線完整定義**：除了「季線下彎後的前一個低點」，還需「該低點上方有 3 個月套牢」—— 補強 `neckline_break` 的精確判定
5. **「轉折K線+高檔長黑」優於多空波動**：對於明顯反轉訊號，可不需等到多空波動觸發
