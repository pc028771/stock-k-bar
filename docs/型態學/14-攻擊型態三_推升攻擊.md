## 推升攻擊 = 最重要的攻擊型態

> 「**推升攻擊型態的判斷可以說是攻擊型態中最重要的環節**……比起日出攻擊的強勢、跳空攻擊的明確，真正應該要花時間理解與運用的，就是推升攻擊型態。」

理由：日出攻擊、跳空攻擊強勢且明顯但**不常見**；推升攻擊最常見但容易迷惑投資人。

## 推升攻擊兩個面向

### 面向 A：盤中江波圖層次（日內）
> 突破隔日沒有向上跳空，但是江波圖呈現了多方波動狀態。

→ 短線進場條件：**早盤第一個高點要越過**

### 面向 B：日K線層次（多日）
> 股價K線在突破日出現之後，並沒有立刻往上攻擊，而是連續K線的低點推升上去。

= **低點推升型態**

兩個面向都採用「多空波動原理」。

## K 線層次的推升攻擊辨識

範例：直得(1597) 106-06-01
- 突破後短期沒回檔
- 「真心打算要拉上去的股票，**沒有理由突破之後又跌回去讓散戶有再次低接共同參與的機會**」

範例：聯陽(3014) 109-06-05
- 突破後中間有一根黑K但未跌破突破價
- 連續 K 線低點呈推升
- 造成「好像隨時來一根長黑就會跌破」的假象
- 實際上：**真正的攻擊意義是不會大幅區間震盪**

## 高檔狹幅整理 = 推升攻擊的一種

> 「股價突破的紅K之後，並沒有呈現出拉抬的跡象，而是橫向的狹幅走勢，股價就是不回檔，過一段時間之後再拉出一根紅K，形成推升狀態。」

範例：潤泰新(9945) 110-07-06
- 紅K突破後連續 6 天狹幅橫向
- 低點沒跌下
- 屬於推升攻擊的醞釀類型

## 對應入門「中樞型態」

高檔狹幅整理 = 中樞型態 = 推升攻擊的一種。

## 對系統的啟示

### 1. 多種推升攻擊識別

```python
def is_low_recovery_attack(df, lookback=5):
    """
    低點推升型 = 連續 N 天 low 不斷往上 + 沒跌破突破價
    """
    g = df.groupby("ticker")
    is_higher_low = df["low"] > df["prev_low"]
    streak = is_higher_low.groupby(df["ticker"]).rolling(lookback).sum()
    return (streak >= lookback - 1) & (df["close"] > df["prior_high_60"])
```

```python
def is_high_zone_narrow_consolidation(df, lookback=6):
    """
    高檔狹幅整理 = 突破紅K後 N 天小區間，低點不破突破點
    """
    g = df.groupby("ticker")
    range_pct = (g["high"].rolling(lookback).max() - g["low"].rolling(lookback).min()) \
                / g["close"].rolling(lookback).mean()
    is_narrow = range_pct < 0.05
    low_above_breakout = g["low"].rolling(lookback).min() >= df["prior_high_60"].shift(lookback)
    return is_narrow & low_above_breakout
```

### 2. 早盤第一個高點越過 = 短線進場條件

> 「對於短線交易者的進場基本條件來說，當日早盤的第一個高點要越過。」

→ 這是 intraday 條件，目前系統用 daily K 線無法直接實作，但可以：
- 用 `close > open` + `close > prev_high * 0.5 + prev_low * 0.5` 作粗代理
- 或留待未來整合 intraday 資料

### 3. 早盤高點沒過 = 弱勢警示

> 「黑K的形成，多數都是早盤高點沒過，股價當天越走越低。」

→ 對應已實作 high_long_black 出場條件，邏輯與「早盤高點沒過」匯流。

### 4. 「攻擊型態 = 持有判斷」鐵則

> 「對於攻擊型態來說，是交易者一定要學習的應用環節，因為人們在獲利狀態中常常耐心不足，虧損時期反而很有耐心的等待解套。」

→ scanner 不應該過早把 candidate 從持有清單中移除，只要還在推升狀態。

### 6. 「醞釀型」推升

高檔狹幅整理屬於推升攻擊。系統應該**辨識並繼續持有**，不應該因為「股價不動」而出場。

→ 增強 ranking：高檔狹幅整理 + 量縮（不是大量散戶進入）= **籌碼安定信號**，加分。
