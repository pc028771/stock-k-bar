## 定義：中繼、過渡

> 「『中樞』指的就是中繼、過渡，意思是原本已經先有了一個明顯的趨勢之後，接下來短時間之內，只呈現了狹幅整理的狀態。」

對應其他課程概念：
- 多方強勢中的中樞 = **高檔狹幅整理型態**
- 空方弱勢中的中樞 = 中繼之後延續的整理
- 非轉折組合中的「**咬定型態**」 = 還沒最後長紅/長黑前
- 上升/下降三法中的兩到三天整理 = 短期中樞

## 判斷重點：「原本的趨勢」

> 「趨勢是最根本的方向所在，不能在短時間內的中樞型態中出現判斷錯誤。」

## 上升中樞型態（多方）

**人性陷阱**：
- 中間略為休息 → 短線客以為漲不動了 → 出場
- 然後突破上去創新高
- 持有者錯失整段

範例：
- 中光電(5871) 107-08-28：三週震盪，最後帶量紅K出現
- 鴻海(2317) 108-04-12：權值股中樞快速通過
- 聯發科(2454) 109-07-08：高價股的中樞（非上升三法、非咬定型態）

**重點**：
> 「中樞型態的重點在於**不要因為過去幾天的小小震盪一下，就自己想太多把股票獲利了結了**。」

## 權值/高價股的中樞特性

- **權值股**：股本大難控制震盪幅度，中樞震盪小且快
- **高價股**：散戶追買少，不需要做太久假動作

## 下降中樞型態（警示型）

**兩種狀況**：

### A. 本應上升卻長時間中樞 → 一根黑K跌出中樞範圍
- 等於整理區間被跌破
- 整理時間過久 → 黑K還帶動季線以下均線全下彎
- = 趨勢完全改變

範例：大立光(3008) 108-05-09
- 中樞 2 個月
- 黑K未到能讓季線下彎程度，但意義已不對
- **股王整理姿態 2 個月沒必要再洗盤**
- 中價股的話可能只是不拉，但高價股需注意法人籌碼疑慮（可能財報有問題）

### B. 空方趨勢已強，中樞區間放大
- 跌深者以為反彈
- 實際只是「把中樞區間放大」

範例：富邦媒(8454) 111-04-25
- 破底 → 整理 → 再破底 → 再整理
- 「空方氣焰最高的一種空頭走勢」
- 季線扣抵還在高檔 → 只要再破底，季線下彎難止

## 不能用於買賣點

> 「中樞型態並非用於買賣點……作為中繼與過渡的一種基礎型態，雖然沒有買進或者賣出的意義，卻足以幫助投資的角度**不要買賣在錯誤的時機點上**。」

> 「空方趨勢進行時，**怎樣都不能賭在中樞的型態之中**。」

## 鐵則：等關鍵K線

> 「如果股價可以改變空方趨勢，那我們就得等改變的關鍵K線出現再說，**不要用過去一段時間的走勢來當作是判斷的依據**。」

## 對系統的啟示

### 1. 中樞型態 = 持有期間的「保持冷靜」訊號

對應已實作的：
- `sunrise_attack_end` 才出場（不要因中樞小震就出）
- `prev_day_low_break_gated`（必須有攻擊意義 K 線才適用）

### 2. 「整理時間過久 + 黑K跌破」= 強警示

新出場條件可考慮：

```python

    """
    高檔狹幅整理 N 天後出現黑K跌破整理區間
    """
    # 條件 1: 過去 N (10-30) 天 high/low range < 5-8%
    # 條件 2: 已在攻擊狀態（趨勢正向）
    # 條件 3: 今日黑K + close < min(prior N days low)
```

→ 比 `breakout_low_break` 更精確（中樞範圍小，跌出比突破K低點晚但更明確）

### 3. 區分「中樞型態」vs「箱型整理」

- **箱型** ≥ 3 個月
- **中樞** 通常 < 1 個月（短期中繼）
- 處理方式不同：箱型突破 = 型態突破訊號；中樞突破 = 攻擊延續

### 4. 高價股 vs 中低價股的中樞時間差

- 高價股中樞 2 個月不對 = 嚴重警示
- 中低價股中樞時間長 = 較常見

→ 加 column `is_high_price_stock`（如 close > 200）作為中樞判定的乘數
