## 箱型整理的嚴謹定義

> **股價在一定的價格區間中震盪超過三個月之久時間，受限於這個範圍，波動並未呈現越來越高或者越來越低的狀態。**

- 有越來越高/低 → 屬於**楔形型態**（不同處理方式）
- 嚴格分類有意義

## 箱型位置決定意義

**情境 A：箱型前有過明顯有幅度的拉抬**
- 箱底跌破 → **直接形成頭部型態**（已經有過炒作 = 套牢者眾）
- 箱底突破 → **不能稱為底部**，只是「多方再次型態突破」

**情境 B：箱型前無主力拉抬跡象**
- 為了出貨目的而整理的可能性低
- **事後檢視可被視為築底**

## 突破箱頂的真實意義

> 「真正的關鍵還是在於**過去股價是否有被刻意拉抬過**。」

突破箱頂時的兩個檢視角度：
1. **是否真的有基本面的更大誘因**？（月營收 / 獲利數據）
2. **這個突破在攻擊 K 線範圍內被稱為「再次突破」**

範例：南帝(2108) 110-03-05
- 箱型突破 + 季線上揚 + 過去高檔曾有空頭吞噬（力竭意義）
- 必須先**忽略突破箱型上緣的意義**
- 等到半個月後真正越過高檔黑吞噬位置才算
- 從基本面看：月營收的確更大誘因，股價從此再漲一倍

## 箱型 + 季線 = 頸線位置

> 「如果加上季線，箱頂與箱底就變成頸線的位置。」

但實務上**不只考慮箱型突破跌破**。

## 「箱底買進、箱頂賣出」的災難

> 「股價跌下來又給人一種機會再來一次的假象，要知道沒有資金會幹這種事情，**放手讓股價掉下來只為了給散戶機會進場做價差**。」

範例：瑞軒(2489) 110-12-16
- 箱型上方有過去帶量的高檔長黑 → 賣壓所在
- 股價突破箱頂後**快速回到箱底**
- 「跌下來又給人機會再來一次的假象」
- 真正攻擊的走勢不會這樣

「箱底買進，箱底跌破停損」**是矛盾的說詞**：
- 前面說箱底買進 = 利用箱型作為買低依據
- 跌破才停損 = 已經承認原來判斷錯誤
- 同時持有兩個矛盾立場

## 箱型誤判的根源

1. **過去基本面驚喜未注意**（月營收推論能得到答案，但人性已箱頂賣出不再追高）
2. **過去主力拉抬未注意**（拉抬過 → 箱型是出貨誘餌）

### 4. 與「賣壓化解」概念結合

突破箱頂後不能直接視為攻擊起點，要看：
- 上方賣壓是否化解
- 是否有「更大誘因」
- 是否在 N 天後再次創新高（攻擊延續）

### 5. 與「型態突破」+「中繼」對立

行進ing 已強調：3 個月整理才算型態突破，否則是中繼。
箱型整理是型態突破的**標準前置**，符合這個邏輯。

### 6. 楔形 vs 箱型的分別**重要**

入門系統目前完全不區分這兩種，需要：
- 箱型 = 高低點不變
- 楔形 = 高低點漸高 or 漸低
- 觸發 entry filter 的判斷不同
