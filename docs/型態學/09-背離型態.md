## 背離 = 現象，不是訊號

> 「背離是**現象**，**不是訊號**。」

定義：「理論上應該如何，但是卻沒有如此」。

## 背離 ≠ 乖離率

> 「背離與乖離率不是同樣的意義，因為乖離率過大，這是很虛幻的說法，那麼多大的乖離率叫做過大？**沒有辦法能有一定的標準**。」

→ 「乖離率」是禁止項目（與「多空循環」、「KD 黃金死叉」等並列）。

## 背離的兩種分類

### 籌碼背離
- 股價上漲但法人連續賣超 → 漲不動的警示
- 股價下跌但法人站買方 → 不必悲觀但不必預期創新高

範例：
- 華碩(2357) 110-03-08：股價上揚但法人賣超 → 110-07-26 才漲不動
- 微星(2377) 110-05-12：外資從買轉賣 → 股價跟著跌

### 指標背離（MACD 範例）

**頂背離**：價格創新高，但指標高點比前次低 = 「上漲乏力」
**底背離**：價格一波低，但指標高點越來越高 = 「強烈買進訊號」（傳統說法）

範例：
- 凌群(2453) 108-08-19：股價創新高，MACD 柱狀明顯短
- 金居(8357) 108-10-01：股價創新高，MACD 在零軸下，快慢線往下

## 為什麼背離不能當作訊號

> 「**台股是淺碟型態**，表示在多頭市場熱烈進行的時候，股價是**輪動表現**的，那麼就算出現了背離型態……最後股價還是會在績優股上再重演一段漲勢，這就不是當初的背離型態可以預測到的。」

## 背離的真正用途

> 「背離型態的功用就在此處，**或許是不適合繼續持有，也或許是空手者判斷不宜介入**，並不是用來作為放空股票的標準。」

對比生活案例：開車上高速公路看到很多車轉向避開塞車 → **不繼續前進的考慮**，不是「往反向走」

## 對系統的啟示

### 1. 完全不實作「背離訊號」

明確禁止：
- ❌ MACD 背離
- ❌ KD 背離
- ❌ RSI 背離
- ❌ 任何指標背離作為訊號

理由：課程明確說「背離是現象不是訊號」、「淺碟市場下不可靠」。

### 2. 但「籌碼背離」可作為持有警示

```python
def is_chip_divergence_warning(df):

    股價創新高但法人連續賣超 → 漲不動警示
    
    Requires: 法人籌碼資料（目前系統未整合）
    """
    is_new_high = df["close"] > df["prior_high_60"]
    institutional_selling = ...  # 連續 N 天法人賣超
    return is_new_high & institutional_selling
```

→ 對應入門「成交量真正用途」：volume 用於識別套牢區 + 籌碼意圖，**不是訊號**。

### 3. 不適合進場 ≠ 應該放空

> 「並不是用來作為放空股票的標準。」

→ scanner 設計上，「背離警示」只是把該股**排除候選**，不是反向訊號。

### 4. 「乖離率」明確禁止

加入 ai-context forbidden patterns 表格：
- 乖離率（過大、過小）= 沒有客觀標準的虛幻說法

### 5. 「股價輪動」的概念

台股淺碟 + 主流輪動 → 即使某股出現背離，其他股可能接棒。

→ 不能因「整體市場背離」而停止 scanner 篩選（與「不要事後論」一致）。
