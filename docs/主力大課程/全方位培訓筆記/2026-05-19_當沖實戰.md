## 4. Spec 沒寫的實戰細節（這份解讀的核心價值）

以下是 Ch5 spec 完全沒寫、但實戰課反覆出現的執行細節：

### 4.1 「兩個價格」教條 ⭐⭐⭐
原句：「就是幾個價格就是幾個價格，就是開盤價、昨天的收盤價，就是兩個價格是很重要的」
- **昨日收盤價 = 第二停損線**（除了 5 分 K 低點外另一個錨點）
- **今日開盤價** = 多空分水嶺

### 4.2 「前 10 分鐘不切入」紀律 ⭐⭐⭐
原句：「我們都不會去衝一開始前 10 分鐘，前 10 分鐘是讓你通知給你之後你就去看一下他後續 10 分鐘走的狀況」
- **Ch5-3 spec 寫「突破第一根 5 分 K 高點切入」，實際是等 9:10 後**
- 切入點：**回踩到 10MA 附近守住**（不是直接過高）
- **這條對 5/27 4526 案例直接打臉**（見第 7 節）

### 4.3 「前 5 分鐘漲 5%」一票否決 ⭐⭐⭐
原句「一開始開盤前 5 分鐘如果他已經拉到拉了 5%以上的基本上你這個就是有點堵了」
- 奇鋐 3017、滑統（推測 信驊 5274 或漢磊）、藥華皆觸發此規則被 skip
- **Ch5-3 spec 只寫「開盤幅 < 4%」，實戰更嚴格用「前 5 分鐘 < 5%」**

### 4.4 「右上角第幾段」視覺判斷 ⭐⭐
原句「走到第 3 段的基本上啊都是紅黑拉上去就丟掉了」「走到第 4 段了知道嗎」
- **完全 spec 外的視覺判斷法**：日 K 圖右上角已經三段以上 → 開高就被丟
- 取代量化「距 20ma < 30%」的口語化版本

### 4.5 「掉人線」（吊人線）的反覆強調 ⭐⭐
原句「下影線太長了有點掉人線的味道我自己就不喜歡」「分 K 裡面不管其實不管分 K 還是日 K 啦我都很討厭這種掉人線」
- 出現 ≥ 6 次
- **Ch5-3 spec 完全沒提**，但這是老師過濾的常用工具

### 4.6 進場「試一下」哲學 ⭐⭐
原句「就進進試一下出出丟一下試一下這樣子」（九暘示範）
- 不是一次梭哈，而是分批 ping-pong 試溫
- 加碼門檻：**「軍線是交叉的我先不要進去」→ 等月線/5MA 確認**

### 4.7 爆大量先出一半 ⭐⭐
原句「這邊爆大量注意，這邊爆大量前面的先丟掉一下除非他已經快進漲停了」
- **Ch5-3 spec 沒寫**：拉抬末段爆大量是減碼訊號（不是加碼）
- 例外條件：**快進漲停就抱**

### 4.8 「紅吞噬會累」 ⭐
原句「紅吞噬黑吞噬黑黑下一根可能就會收不會這麼快拉出去因為他很累他吃完這個黑帮他休息一下」
- 大紅 K 吞噬黑 K 之後必定回踩，不要在吞噬完當下追
- 等下一根回踩同樣支撐位再進

### 4.9 多空鏡像對稱
原句「掉人線的相反有沒有掉了你把你把剛做多的那個這個鏡面圖找出來」
- 老師示範蒙利（推測誤譯，可能為茂林 4935 或其他）做空，**結構就是做多形態翻過來**
- 紅 K 上來空下去（空方控盤判斷）

### 4.10 「沒人要拉」的撤退訊號 ⭐
原句「最後階段最後拉沒有拉上去就全數離開」「沒人拉或者是拉完知道那個梗圖嗎就沒有人要拉了」
- 兩三根 K 不再創高 → 主動掀傘出場（不等停損）
- **Ch5-3 spec 沒寫「主動退場」**，只寫被動停損

## 6. Ch5 Spec 修正/補強建議

### 6.1 Ch5-1-1 補強（量化條件 → 加入口語判斷）

新增條目：

```
### 視覺優先過濾（盤中即時用）
- 「右上角第幾段」原則：日 K 圖已連續上漲 3 段以上 → 開高紅 K 都是給你賣的、不切入
- 「離均線太遠」原則：股價偏離 20ma 太遠時，視覺上判斷已「走到右上角末端」即過濾，不必再算 30%
```

### 6.2 Ch5-3 補強（停損 + 進場時點 + 達標）

新增條目：

```
### 進場時點細節（spec 缺）
- 開盤前 10 分鐘（09:00-09:10）不切入，只觀察
- 9:10 後等回踩 10MA 守住 → 切入
- 前 5 分鐘漲 > 5% → 整檔放棄（比「開盤幅 < 4%」更嚴格）

### 停損雙錨點（spec 只寫一個）
- 主錨：第一根 5 分 K 低點（原 spec）
- 副錨：**昨日收盤價**（老師反覆強調 ≥ 15 次）
- 跌破任一錨 → 停損

### 達標進階規則（spec 缺）
- 達標 1.5-2.5% → 走（原 spec）
- 例外 1：拉抬末段爆大量 → 先出一半（不是加碼）
- 例外 2：爆大量且快進漲停 → 留倉轉短線
- 例外 3：兩三根 K 不再創高 → 主動掀傘出場（不等停損價）

### 進場「試溫」哲學（spec 缺）
- 不一次梭哈，分批進出 ping-pong 試溫
- 加碼門檻：5MA 與 10MA 不交叉糾結 → 才加
- 軍線交叉時：先撤、等月線/5MA 重新表態

### K 棒形態過濾（spec 缺）
- 上影線 > 實體 → 不做
- 下影線 > 實體（掉人線/吊人線）→ 不做
- 紅吞噬之後一根：必回踩，不追、等回踩支撐再進
```

### 6.3 Ch5-2 補強

```
### 量的反向使用（spec 缺）
- 「量越大越好」只在突破當下成立
- 拉抬末段同樣的「大量」是減碼訊號（除非快漲停）
```

### 6.4 新增 Ch5-X：當沖做空 SOP（spec 完全沒寫）

```
### 做空形態（多方做多形態的鏡像）
- 紅 K 上來 → 黑 K 灌下 → 紅 K 上來 → 黑 K 灌下：空方控盤
- 切入：彈上後守不住均價線 → 空
- 紅吞噬之後也會「累」，下一根可能反彈再空更好
- 出場：殺到打底、走不動 → 回補
```

### 6.5 新答 5/27 法人籌碼 12 點殺盤是否適用？

- 5/19 課強調「只做前半小時、前一小時」→ **當沖根本不會做到 12 點**，所以 12 點殺盤對純當沖無感。
- 但若採 Ch5-1-2「思維切換」留倉的單子，12 點殺盤就直接觸發本日尾盤觀察 → 與「沒人拉就掀傘」邏輯一致。
