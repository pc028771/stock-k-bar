## 1. 進場規則（做多）

### 1.1 4707 磐亞「均線順向多頭」標準長相（9:02 通知）
老師明說「其實應該不會操作（因為不熟）」，但**特地拿來講「位置是對的、圖是對的、可以用在其他長相身上」**。這是當沖做多的教科書 setup：

> 逐字引用：「均線很棒、均線在這個地方往上揚、然後沒有發散、全部均線**包含季線**也在這個地方、所以一個紅K下來洗一下黑K灌下去再紅K拉上去、基本上這個長相就已經成立做多的姿態了、就慢慢這樣拉上去。但是你的位置點可能就在這邊 282 附近拉上去、然後洗一洗震盪一下、觀察一下有沒有往下蹦、結果還是沒有都還能撐住、那就繼續留著。」

拆解成條件：
1. **均線往上揚**（上彎）
2. **沒有發散**（均線糾結、未拉開）
3. **季線也聚在同一個位置**（含季線的多均線匯聚 = 強）
4. **紅K → 洗一根黑K灌下 → 再紅K拉上去**（洗盤後再攻的型態）= 做多姿態成立
5. 進場後**洗一洗震盪、觀察有沒有往下蹦；撐得住就續抱**

> 補充引用：「還有是有弓喔、今天整場都有弓」「這個軍線是不錯的、你可以稍微記一下」（軍線 = 均線；「弓」疑為「攻/弓型」、逐字稿未明確、不硬解）

### 1.2 開高拉下 → 10日線與月線之間找多單（會吃損的反例示範）
某檔開高衝上後黑K灌下，老師示範「在 10 日線附近做小短多」、結果失敗停損：

> 逐字引用：「我們之前的教學是**開高拉下來、在十日跟月線之間你可以去找一個多單的機會**、所以 554 / 558 進去。但是呢灌下來了、在這一根**破掉了月線**了、月線破就該閃人了、所以你這個地方就會吃損、吃一塊錢左右。」
> 「但注意你**多單虧損你可以用空單來補**」（多單停損後反手空、把損失補回來）。

→ 印證 spec「踩回進場 SOP（10MA/月線之間）」+ 停損「破月線就閃」。

## 2. 隔日沖倒貨識別 + 順勢做空（本課重點）

### 2.1 為什麼「昨漲停今開平/開低」很恐怖（老師講第三、四次）
> 逐字引用：「昨天漲停、今天開低、這就完蛋了⋯**漲停然後隔天開低是很恐怖的**、就是所有漲停的那些人、他隔天完全賠錢去賣做這件事情、他連拉都不想拉、代表說就很危險了⋯漲停隔天開低很可怕喔、都是要出事情的喔。」（老師強調「我們好像講第三次第四次了」）

> 開平盤版本（西華 ~$76.8）：「**昨天漲停然後開平盤也很恐怖喔**⋯往下刷一下、然後隔日沖趕快自救、自救起來、我趕快丟貨了⋯讓我可以賣在昨天的漲停附近、鎖漲停的位置趕快跑、趕快一直倒給這些散戶。然後黑K灌下來再拉一次我再丟一次、結束、倒貨完畢。」
> 「**趕快拉上來吸引散戶進來、之後我再倒一個更好的價格**、所以非常的險惡。」

機制（隔日沖交易對手邏輯）：
- 昨天漲停買進的隔日沖客，今天**賠錢/平盤就急著倒貨**（連拉都不想拉 = 最弱）
- 開低/開平後可能有「5秒、10秒的自救機會」拉一下解套 → 但**解不了**
- 拉上去就是給你**加空的點**、不是追多的點

### 2.2 「加高」案例 — 收破昨收 = 隔日沖出貨完畢 → 順勢空（賺 10%）
這是本課最完整的做空示範（「加高」= 諧音、代號未知）：

> 進場觸發：「昨天漲停、加高昨天漲停、然後結果第一根開一個小高 OK、**但是收盤是往下破、收破了昨天的收盤價、這個就是代表隔日沖已經出貨完畢了**。只要守一守、沒有人要往上繼續攻的話、他就會慢慢盤下來。」
> 「只要早上的紅K一直被黑K吞噬、代表說就很可怕了⋯今天的主力們都一直被打下來、那他就覺得算了今天盤勢不太正確。」

> 不馬上空、等趨勢成形：「**這個地方不代表馬上就要空、等待一下、他這邊只有 50/50 的機會**。下一根下下根紅紅之後、第三根又再出現一個黑K、那這個空單的機會就大增了⋯但是一根黑黑又再灌下來了、好、開始真的偏空了、533 空下去、53.3 / 52.7 開始去點空單。」

> 季線是關鍵錨：「注意看這邊所有均線開始往下了⋯**季線只要一沒有往上揚、開始變成走平的話、它就會形成土石流**⋯空在哪個位置、空在破季線這個附近 527。」

> 結果：「這是跌停喔今天收跌停⋯53.3 / 52.7 合起來平均價是 53 塊、一下來 48 塊、你想怎麼補就怎麼補⋯都是 8% 9% 很漂亮的地方、一整天賺這 10%、日內賺 10% 是超級厲害。」

關鍵：**空單回補不在反彈處追補、反彈是「下一個空點/加空位」**：
> 百榮案例引用：「你空單都會補在⋯彈上來這個地方 54 啊、上來這邊都不會去做回補、因為**都很可能會是一個下一個空點**⋯碰到季線了、這邊不會是讓你去追多的、是空單看有沒有機會再做一次的空單、在加空的位置。」

### 2.3 「百榮/白龍」案例 — 兩根大黑K吞噬前紅K = 賣壓很大、破多重套牢就空
> 逐字引用：「攻了之後下一根馬上收一根大黑K、這一根其實就不太妙了⋯**兩根大黑K就吞噬了前面的紅K、這個就是賣壓很大、準備要閃人了**。」
> 三重套牢壓力 = 空點：「它只要沒有守住昨天的收盤價、剛剛好季線也在這個地方、所以只要一跌破那完蛋。**前面累積了兩次的套牢壓力（當日套牢 + 昨天收盤 + 季線）、只要一破這個位置就可以空下去了**。」
> 切入時點：「慢慢下去之後抽一個上來下影線、我就會開始空在 558 附近⋯沒有力 561 空一下⋯**只要插上來就空下去**、就一路下去了。」

### 2.4 順勢空的「連動」教學
> 西華案例引用：「因為剛剛的那個（加高）跌停了、所以你看到**加高跌停、你就趕快順勢去空西華**、這個有沒有撐住啊、這也沒撐住、也是一路的下去。」

→ 同型態（昨漲停今開弱）的標的會**一起倒**、看到一檔確認破位、可順勢掃同型態的其他檔。

## 3. 不碰的標的特徵（篩選門檻）

> 逐字引用：「景明這個就做不了、他這個**沒有在我的範圍內**、所以我就不會去操作。雷迪克光學也沒有操作、這種都是**量很小的**、不管他、那量超級小。」
> 「百貨業很奇怪今天跳出這些東西就很奇怪了、所以你看今天跳的東西都是這種百貨業、**你就知道今天盤怪怪的、你不要亂去做多、不然又會小賠一些錢。**」

門檻：
1. **量太小不碰**（景明、雷迪克、華融、48元小股票等）—— 量小 = 不在操作範圍
2. **不熟不做**（4707 磐亞、配亨老師都明說「不熟啦、不熟」即使圖對也不操作）
3. **跳出來的清單性質本身是訊號**：跳出一堆「百貨業/冷門小股」= 今天盤偏弱、別亂做多

## 4. 當沖獲利預期 / 部位管理

> 逐字引用：「我們說平均日內做當沖的話、**1.5% 1.6% 這個獲利就已經很強了、2% 算今天大賺**。你賺到了 10%、這個是放鞭炮了。」

- 日內 1.5%-1.6% = 強
- 2% = 今天大賺
- 10% = 放鞭炮（極罕見、需碰到當天跌停鎖死順勢空滿）
- **回補/出場由你決定**：「跌停諜聽（鎖死）的、看你想補在哪個位置都是 8% 9%」—— 跌停鎖死後在任一價位回補都是賺。

→ 印證 spec line 466「達標 1.5%-3%/每段」、但這課給了更完整的**心理錨**（2% 已是大賺、別貪、別預期 10%）。

## 5. 停損 / 出場

### 5.1 做多停損 = 破月線/破關鍵均線就閃
> 「在這一根破掉了月線了、**月線破就該閃人了**、你這個地方就會吃損、吃一塊錢左右。」（10張吃一萬、三張吃三千）

### 5.2 做空進場 = 收破昨收 + 黑K吞噬 + 季線走平
- **收破昨天收盤價** = 隔日沖出貨完畢的確認訊號（反覆出現）
- **早上紅K一直被黑K吞噬** = 多方力量不夠
- **季線從上揚 → 走平 → 下彎** = 土石流啟動點、空在破季線處

### 5.3 多空都「只做前段、等趨勢成形不搶第一根」
> 「這個地方不代表馬上就要空、等待一下、他這邊只有 50/50 的機會⋯第三根又再出現一個黑K、那這個空單的機會就大增了。」

→ 等「破位 + 反彈不過 + 第三根確認」、不在 50/50 處搶空。

### 5.4 「通知密度」判斷今日盤性（本課老師自評最重要的教學點）
> 逐字引用：「我覺得今天要教大家很重要的這個**通知的觀念**⋯我們這個通知（2分K/3分K 過高）的今天的數量超級少、你就知道一件事情、**今天盤空了、日內要做空了**。因為我們這個通知是有往上突破往上攻他才會通知你⋯你發現今天往上衝突破的通知很少、代表大家都是在往下走、那你就知道今天不要去做多、而且是要有反彈上來你可以找位置去做空。」
> 「**9.5 分（早盤時段）才 9 折、不到 10 折、你就知道今天都是放空盤**。下次去觀察一下、我們這通知只要 9.5 分之前不到 10 折的、今天都是空、放空盤。」

→ **新規則（spec 沒有）**：用「早盤過高通知的累積數量/密度」判斷今日多空盤性。通知少 = 大家往下走 = 做空日，別逆勢做多。

## 7. 對 user 持倉 / 操作的啟示

> user 持倉：元大金 / 聯電 / 日月光 / 台積電 / 亞電。**本課是當沖技術、不直接對應波段持倉**、以下為戰術通則：

1. **這課對持倉的直接價值有限** —— 是分K戰場（當沖）技術、跟 user 的波段/中長持倉是「兩個戰場」（spec B-3 已明示日K vs 分K 不混用）。別把當沖空訊號套用到波段持倉上做減倉決策。

2. **可借用的通則 = 「昨漲停今開弱 = 危險」**：若持倉中有標的昨日漲停、隔日開平/開低 + 收破昨收 → 這是 spec 既有的轉弱訊號（總整理 line 172/201），本課再次印證。**亞電（4939）若出現此型態要警覺。**
