## B. 法人籌碼課程 →【敘事 + 族群方向 + 法人框架】

### 排序方法論（老師明示）

> 「我們的排序是用**外資買超**、買超越小、量能一定越小。」
→ 名單由**外資買超金額由大到小**排列、越往後量越小（小錢區）。今日標題「法人外資預兆先買車」＝**跟著外資買超 + 軍線順向 + 自救點**找布局。

### 老師選股邏輯框架（六軸）

1. **外資買超排序**：先看外資今天買哪些（即使收大黑 K、只要外資仍買超就記下來、等回測）。
2. **軍線順向**：老師有一組「軍線順向的股票」清單（居祥、加幣期/嘉碁期等）— 守均線、慢慢往上走的型態最愛。
3. **融資自救點（核心新框架）**：逐字——
   > 「如果明天開盤很差、直接到**十日線**的話、這個融資、外資都會**自救**、所以你的成本會比他們低。」
   > 「（加聯益）這邊**融資一直增加、今天也沒跌很多**、明天如果真的暴跌下去注意一下…台南、台中、元大草屯都是一些買方、融資很多、記一下。」
   → **融資集中 + 外資同向買 + 回到十日線 = 隔日自救反彈點**、散戶成本被卡在上方、主力/外資成本更低 → 你買十日線成本比他們低。
4. **十日線 / 五十日線（軍線）支撐**：回測均線是進場前提（守住才做）。
5. **大錢 vs 小錢看成交量**（部位邏輯、逐字）：
   > 「你說大錢小錢怎麼分？就看他成交量。**七八萬甚至更高就算 OK；如果成交量只有一萬張以下、那種你基本上也不太能壓太大。**」
   > 「小錢的票可以**觀察他的狀況**、之後把同樣道理**印證用在大錢上面**。」
   → **倉位上限由成交量決定**：7~8萬張+ 才敢壓大；<1萬張只能小單/觀察驗證股性。
6. **盤中相對強弱**：大盤往下灌但個股守住 = 簡單對應關係（老師盤中做「聯軍」的理由）。

### 「凱基戰前（站前哥）買黑隔天做」打法（逐字完整）

**打法 SOP**：分點凱基戰前（站前哥）**在黑 K（下跌/收黑）當天買超某檔 → 隔日做多該檔**、勝率高。買紅當天不算特別訊號（屬常態）。
