## 3. 法人籌碼框架

### 外資（本週主要動作）
- **反手回補**：聯電、新唐（連三日）、強茂（8,000 張）、康舒（持續挺）

- **老師解讀**：外資成本不會買在最低、會在「相對高位」（如新唐 65–66）=> 跟外資不是抄底，而是**在外資成本區附近等回**

### 投信
- 字幕未直接點名投信動作，但 L92–L108 老師強調**「證券業創新高」「我特別提醒的」**=> 暗示投信會布局券商（兆元結構性受惠）
- 投信偏好邏輯（從老師強調的）：**結構性、有實質數字、有兆元背景的族群**——券商即為標準範本

### 自營
- 字幕無直接著墨

### 三者交叉訊號（從這份字幕推導）
- **Bullish 組合**：外資反手大買 + 老師圈內持有 + 均線靠攏 + 缺口結構（例：聯電、康舒模型）
- **Bearish 組合**：外資大買但「均線未排整」+「離越線很遠找不到支撐」（例：泰鼎、奇鈦）—> 不追
- **避開組合**：今天大漲 + 已連三根（順打、蒙利）—> 老師明說「我們都會先撇掉今天大漲的股票」（L74–L75）

### 老師教的新方法論（這堂課的精華）
1. **外資成本區操作法**：外資成本不在低點、在相對高，例新唐 65–66；策略 = 「明天回的話、尾盤回到外資成本區、可看」（L29–L39）
2. **缺口 → 小結構迴圈法**：「他只要有一個學（缺口）我就進行下一個小結構的整理」（L67）—— 一個缺口 = 一個新觀察波段啟動
3. **撇掉今日大漲股原則**：「我們都會先撇掉今天大漲的股票」（L74）—— 法人籌碼分析時，**先過濾掉「外資買 + 今日大漲」**，因為敘事已被反映
4. **三符合缺一不可（敘事 + 均線 + 籌碼）**：泰鼎是反例——籌碼符合但均線未到，不切

## 5. 對應操作建議（從敘事推導、不是個股推薦）

### 加碼方向（族群）
- **成熟製程**：明天若回，外資成本區（如新唐 65–66）尾盤可承接；持有聯電者繼續抱
- **券商**：兆元結構性改變受惠族群，**「不要小看」**——這是配置邏輯，非短打
- **記憶體（華邦電系，非南亞科）**：跟外資同向，戰前哥已介入

### 減碼方向（族群）
- **第一排機器人/傳動**：三根 + 外資連賣 + 老師明說「不可能叫你明天買」=> 已持有可考慮降溫，空手不切
- **IC 設計文曄系**：套牢賣壓未消、避開直到均線排整

### 觀察方向（什麼訊號出現 → 翻盤）
- **缺口 + 小結構整理**（康舒模型）：等下一個缺口 + 整理完畢 = 切入訊號
- **均線靠攏**（泰鼎、奇鈦、旺宏）：等均線到位 = 切入訊號
- **外資連續買 N 天**（新唐三天）：訊號明確，但要配合均線

### 風險（什麼事件 → 退場）
- **美股今晚大漲 → 明天台股早盤拒追**（觀盤重點補充）
- **主流族群出現「10 萬張出量」**（如 L58–L60 金元店示範）=> 主流出貨
- **套牢賣壓蔓延**（文曄系訊號擴散到別族群）=> 敘事破
