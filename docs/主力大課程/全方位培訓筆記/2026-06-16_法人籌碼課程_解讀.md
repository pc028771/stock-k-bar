## 五、籌碼觀念補充（課程教法）

### 1. 用權值股姿態判斷指數方向（戰略框架）

> 「**你就看台積電長相**、如果他是均線順向慢慢走上去的話、你就大概知道這個指數、不要去預設高點。但是如果他姿態很醜、有各種破線下來的話、那你就會有一個印象、這指數都已經在做頭了、台積電都已經在做頭了、那指數可能就很慘。」(~14:30)

→ **這是課程級教法、可入 memory**：「**用權值股姿態（特別是台積電均線）判斷指數方向、不靠數字高低點預設**」。

### 2. 自合證 = 自營商（觀念釐清）

詳前述、提醒 user 在籌碼面看到「自營商」買超時、不一定是市場敘事、可能是投資公司自有部位。

### 3. 末端收斂 = 進場觀察點

> 「均線收斂到末端、寫起來、後面拉一根你不會意外。」(~12:00)

→ 對應 K 線力量「均線扣抵亮燈 + 收斂」、跨課程驗證。

### 4. 篩選方法：均線順向掃 28 檔

> 「**28 比均線順向、連強就有出來、瑞昱也有出來、聯詠也有出來**、刚刚我們重點的那幾個都有出來。」(~12:30)

→ 老師親口示範**用「均線順向 scanner」做 first pass**、再用籌碼層做 second pass。
