### 1. 🔴 「選股設定」概念（直播第二大主軸、文章未提）

**「大口咬」的股票**：
- 是**全集股**（有期貨可搭、可用槓桿）

**「放眼睛、玩票」的股票**：
- 小股票（5000 張就可以漲停、東普 / 環科這類）
- 漲 30% 看起來爽，但絕對金額遠小於全集股 +10%
- 老師說「就放一點小錢、有上去稍微點一下」

**對 user 的直接意涵**：
- 已有 memory `feedback_put_eyes_position.md`（放眼睛 = 1 張保底）
