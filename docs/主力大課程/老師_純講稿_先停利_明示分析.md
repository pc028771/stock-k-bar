**是否課程明示「先停利」？答案：YES，但有強烈情境分流。**

- **當沖／極短線**：完全明示「滿分 3% 走人」、「大量先丟一半」、「上影線快閃」、「沒延續性出一半」
- **波段／中長線**：明示「抗波動、最小單位不亂停損」，**反對**過度先停利
- **核心持股有 30%+ 利潤**：「明示不用緊張、不用調整」（5/26 東捷案例）

### 5/19 當沖復盤操作策略課

#### A. 明示「先停利」

**[L215-227] 當沖滿分 3% 就走**
> 「所以記得 / 我們當沖 / 當沖來講啊 / 最好的情況滿分是 3% / 懂嗎 / 有人賺到 0.8% 0.6% 的 / 量很大的那種 / 其實就算很厲害了有沒有 / 但是我們小小部位的話你可能賺個 2% 3% 算很 OK / 好所以這打到往上走 / 就離開了 / 有賺到 3% 3.多% 就離開了 / 後面有沒有漲停不管我們的事 / 懂意思嗎 / 後面就不關我們的事我們就做這一段而已」

→ **極明確**。當沖賺 2-3% 就先停利，「後面漲停不關我們的事」。

**[L624-635] 爆大量先丟一半**
> 「大量 / 好爆大量這地方爆大量注意 / 這邊爆大量前面的先丟掉一下除非他已經快進漲停了 / 是對他是快進漲停了 / OK 好所以 / 這我們剛後面都都都沒有看到只是 / 沒有細看所以你不會記得我這樣所以這是我平常 / 會去做判斷 / 一開始紅 K 大紅 K 這小心一點但是下一根有回踩了 / 支撐確認進場 / 7.89 / 拉上去到這個地方的時候下一個沒有延續性先出一半」

→ 明示：**爆大量＋下一根沒延續性 → 先出一半**。除非快漲停才不丟。

**[L656-665] 大量見高點先出一半**
> 「但是你認知如果這邊有出現大量的話前面的都要先走一半 / 我現在因為大量 / 後面會見高點 / 見高點先出一半但是如果是漲停的話 / 如果這邊大量是漲停的話 / 那就除外 / 懂嗎 / 所以就很漂亮的 / 去做了這一檔」

→ **同上強化**：大量見高點先出一半，唯一例外是漲停。

**[L573-584] 高位上影線快閃（順德／盤勢）**
> 「每次都是紅 / 十字紅十字紅十字紅 / 在這個高點拉出去到 140 出了這個下影線 / 其實我可以幫上面很討厭這種弔人線 / 分 K 裡面不管其實不管分 K 還是日 K 啦我都很討厭這種弔人線」

→ 暗示偏明示：見上影線／弔人線就出。

**[L515-520] 太多下影線先閃**
> 「太多下影線也很奇怪 / 我先叫各位看到這太多下影線也很奇怪 / 知道嗎 / 我就趕快閃了太多下影線很奇怪 / 這個我應該也做不到我這邊 / 會捏著然後等到兩三個下影線出來的時候我就會閃了 / 我就會閃了」

→ 明示：訊號不對先閃，不等停損。

#### B. 暗示

**[L443-462] 黑 K 反轉就撤**
> 「好黑黑 / 灌下來這注意了 / 又開始準備要反轉了我們再往下看 / 好最後階段 / 最後拉沒有拉上去就全數離開 / 懂嗎 / 好有點不行了 / 有點沒力了 / 要要撤了這我會撤了 / 這沒人要拉了好這就是沒人要拉了 / 就掀傘」

### 5/26 觀盤重點

#### A. 明示「先停利／先砍」

**[L160-178] 早盤先砍三分之一觀察**
> 「我認為今天其實今天盤 / 早盤的時候 / 早盤半小時是在砍東西 / 先砍一下下來 / 然後觀察一下有沒有發現 / 先砍一下 / 然後等了一下 / 全部成組沒有到貨 / 再把它 / 剛剛砍了三分之一再補回來 / 對因為我自己認為 / 今天好像 / 可能 / 早盤的時候有一點那種出貨的那種味道」

→ 明示：漲一根後要小心，**休息就是要先處理部位**。

| 強度 | 句數 | 適用情境 |

## 結論

### 1. 純講稿中「先停利」教法明確度

**明確度：HIGH（明示，有具體執行案例）**

- 共 **12 句直接明示**，老師親口示範至少 **3 個具體案例**：
  - 5/19 當沖 3% 走人、爆大量出一半
  - 5/23 自己 40 張「沒工先賣一半」
  - 5/26 早盤砍 1/3 觀察
  - 5/27 抗合宿「出量減碼一點點」、「紅 K 丟丟丟」
- 但**強烈情境分流**：先停利只用在「短線確認失敗 / 出量見高 / 紅 K 上影」，**反對**對「真心相信的好東西／成本 30%+ 核心倉」亂停利。

### 2. 5/28 Line 群是「新教學」還是「強化已教過」？

**答案：強化既有教法（YES）**

理由：
- 「先停利／先賣／減碼」在純講稿已有 12 句明示句、11 句暗示句
- 老師親口示範**至少 3 個自己的執行案例**（包括 40 張對賬單）
- 「滿足就走」「沒工先賣一半」「出量減碼一點點」「紅 K 丟丟丟」全部都在錄影課程中明示

→ Line 群即時提醒同學「先停利」**不是新概念**，是對既有教法的即時情境套用提醒。

### 3. 對應 spec 該怎麼更新

#### 建議納入 spec（已有講稿明示，可寫進主力大課程文件）

| Spec 條目 | 出處 | 適用 |
|---|---|---|
| 當沖滿分 3% 出場 | 5/19 [L217-225] | Ch5 當沖 |
