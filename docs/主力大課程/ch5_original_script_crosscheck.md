## 結論摘要

**原講稿就有、漏寫最多的 3 條：**

1. **A-1（守住/踩住）**：ch5-3.txt 出現 10+ 次，是整章最核心的盤中判讀動作，spec 術語段完全漏收，整理時疏漏最嚴重。
2. **S-7（離月線 30% + 視覺替代）**：ch5-1-1.txt 出現 4 次，量化條件已在原講稿，spec 有收但沒補「視覺替代判斷」說明，造成 spec 與實戰口語用法脫節。
3. **B-8（均線發散 → 硬性過濾）**：ch5-3.txt line 456 明確有，但 spec 定性為「理想條件」，而非「硬條件」，等級判斷錯誤。

**整體結論：**
- 18 條補強中，**5/19 新增占多數（12 條）**，代表 5/19 復盤課有實質教學新增，繼續抓類似復盤課有價值。
- **原講稿漏寫（9 條，含 spec 錯誤）**中，A-1「守住」最嚴重，其餘多為細節執行動作缺失。
- S-4（達標上限）屬原講稿自身前後不一致，建議以最後一次提到的「3%」為準。
