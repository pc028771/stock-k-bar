# F 當沖智慧單每日 SOP（單進單出版）

> 配合 daily_scanner_job F 當沖前夜篩 Rank 1、智慧單全自動執行
> 預期 EV：+0.44% per pick（含 Ch5-2 量能 filter）/ +0.73%（無 filter）
> 預期勝率：78%（含 filter）/ 71%（無 filter）
> 月均 trade：9（含 filter）/ 17（無 filter）

從輸出找「🎯 F 當沖前夜篩 (Top 10)」段、抄錄 **Rank 1**：

| 欄位 | 值 |
|---|---|

⚠️ **排除條件**（任一中即 skip 今日不做）：
- [ ] Score < 30（過低、紀律不夠）
- [ ] 距前高 > 9%（接近 filter 邊緣）
- [ ] 今日漲幅 > 7%（明日跳空鎖死風險）

## 2️⃣ 開盤前 8:30 試撮觀察 — 3 分鐘

- [ ] 看試撮價、若 **試撮 > prev_close × 1.05**（跳空 +5%）→ **skip**（紅線 #1）
- [ ] 試撮量大小（粗略判流動性）

## 3️⃣ 9:00 開盤 — 立刻記

```
open_price = __________ (9:00 開盤瞬間)
```

- [ ] 確認 open_price vs prev_close 跳空 < 5%（再次紅線 #1）

## 4️⃣ 9:05 第 1 根 5K 收完 — 算 4 個價

| 變數 | 公式 | 值 |
|---|---|---|
| H1 (第 1 根 5K 高) | 從券商 5K 圖讀 | __________ |
| L1 (第 1 根 5K 低) | 從券商 5K 圖讀 | __________ |
| **進場觸發價** | H1 + 0.01 | __________ |
| **雙錨停損 stop** | max(L1, open_price, prev_close) | __________ |

⚠️ **若 stop 距 H1 < 0.5%** → **skip**（停損距離太小、噪音容易打到）

⚠️ **若前 5 分鐘 high > open_price × 1.05** → **skip**（紅線 #9）

## 5️⃣ 9:05-9:09 掛智慧單組合（4 單）

### 智慧單 #1 — 進場（觸價買）

```
類型: 觸價買進
觸發條件: 成交價 >= __________ (進場觸發價)
生效時間: 09:10:00 至 11:00:00  ← 進場時間窗
動作: 限價買進 N 張、上限價 = __________ (entry × 1.05、防滑價)
失效時間: 11:00:00 之後自動取消、今日 skip
```

⚠️ **進場時間窗 09:10-11:00 來源**：
- Ch5「9:10 後通知品質下降、最佳訊號 0-10 分內」
- backtest 證實：實際 entries 全部 < 11:00、11:00 cutoff EV 不降
- 11:00 後仍未過 H1 = 弱勢盤、強行進場 EV 變負

部位 N = __________ 張

### 智慧單 #2/#3 — 進場後自動 OCO

```
條件: 智慧單 #1 成交後立即掛
類型: OCO (One-Cancels-Other)

【停利限價】
   價格 = 成交價 × 1.025 (達標 +2.5%)

【停損觸價】
   觸發價 = __________ (= stop)
   動作: 觸發後市價賣出

任一成交、另一單自動取消
```

⚠️ 部分券商要進場成交後**手動**掛 OCO、設定鬧鐘提醒

### 智慧單 #4 — 13:25 強平

```
類型: 時間單
觸發時間: 13:25:00
動作: 市價賣出剩餘部位
有效期: 當日有效
```

## 6️⃣ 9:10-13:25 — 不看盤

智慧單跑、user 不動作。可去做其他事。

⚠️ 例外手動介入時機：
- 大盤突發崩盤（跌 > 200 點 / 小時）→ 手動全平、別等智慧單
- 個股出現「兩三根 5K 不創高 + 量縮」(掀傘訊號) → Ch5-3 例外、可手動先出半張
