## Ch1 — 投資前你必須要知道的三件事（投資三大原則）

### 核心概念
- 操作的勝率組成（主力大自定義拆解）：
  - **選股 50%**（基本面 + 籌碼面）
  - **進出場規劃 30%**（均線、量價、缺口、型態四件事）
  - **運氣 20%**（老天爺、無法控制）
  - 課程裡的「100% 賺錢」是不存在的，只追求高勝率。
- **第三件事：心態與觀念** — 「活下來比什麼都重要」。
- 選股不對，操作白費。

### 進場/出場/停損規則
- 本章節為「總論」，不直接給進場/出場規則，而是預告後續單元會講：
  - 進出場規劃會用 **均線、量價、缺口**（最重要 3 個）+ **型態**（次要）。
  - 不使用 MACD / KD / RSI（互相抵觸時無法決斷）。

### 重要術語定義
- **基本面**：分為「題材」（漲價、缺貨、轉單）與「族群」（面板、航運、鋼鐵、記憶體、被動元件等）。
- **籌碼面**：外資（波段）、投信（短線）、主力分點（內資）。
- **地緣分點**：證券商與公司地址相同地區（如高雄分點大買高雄公司），代表可能有內部資訊。
- **大道至簡**：主力大的核心哲學，越簡單越能勝出。

### 可量化參數
- 勝率拆解：50% / 30% / 20%（選股 / 進出場 / 運氣）。
- 主要技術判讀只有 3+1 個：均線、量價、缺口、（型態）。
- 不使用：MACD、KD、RSI。

## Ch2-1 — K棒的意義

### 核心概念
- 紅K：收盤 > 開盤；綠K：收盤 < 開盤。
- 上影線 = 賣壓；下影線 = 買盤支撐。
- 紅K 一定比綠K 好：紅K 代表當天多數人賺錢，綠K 代表多數人套牢。

### 進場/出場規則
- 需要等股價突破密集上影線高點才視為「賣壓消化完畢，才有進場理由」。
- 高機率有人賣出的位置 = 壓力；股價未突破壓力前不進場。

### 重要術語定義
- **真空地帶**：未在本章定義（為 ch2-3 缺口教學用）。
- **長下影線**：盤中跌到低點被買盤拉起的型態，代表有多方防守。
- **長上影線**：盤中漲到高點被賣下來，代表上方賣壓沉重。

### 可量化參數
- 沒有具體數字參數；判斷依肉眼框出「上影線涵蓋區間」。

## Ch2-2 — 均線的實務應用

### 核心概念
- 均線 = X 天平均買進成本（用收盤價計算）。
- 均線天期決定交易屬性：
  - 短期：5ma 穿 20ma → 持有 3-5 天
  - 中期：20ma 穿 60ma → 持有 1-3 個月
  - 長期：60ma 穿 120ma → 持有 3-6 個月
- 均線上彎對股價有支撐；下彎則為壓力。
- 黃金交叉：短均上穿長均 → 多方力道；死亡交叉相反。

### 進場/出場規則
- **進場：** 等待均線完成「理想多頭排列」：
  - 必要條件：5ma > 10ma > 20ma > 60ma（價格排列正確）
  - 理想條件：4 條均線皆上彎
  - 加分條件：均線尚未發散（仍糾結）
- **長天期版本：** 把參數改成 20 / 60 / 120 / 240ma（適合 3-6 個月以上波段）。
- **慣性支撐進場：** 股價回到某條均線（如 20ma）後隔日馬上站回 → 確認該均線為慣性支撐，可在該位置進釋單。
- **出場：** 收盤跌破慣性支撐均線 → 出場避免更大虧損。
- **加碼：** 每次回到慣性支撐均線守住，可加碼或釋單。

### 重要術語定義
- **多頭排列**：5ma > 10ma > 20ma >(60ma)，且皆上彎。
- **發散**：均線之間距離拉開，相對於糾結而言。
- **慣性支撐**：股價回到該條均線後反彈，反覆驗證該條均線為支撐。需自行從歷史觀察驗證。
- **理想條件**：可以分為「必要 / 理想 / 加分」三層。

### 可量化參數
- 均線組合：5/10/20/60（短中期）或 20/60/120/240（長期）。
- 跌破慣性支撐用「收盤價」判斷。

## Ch2-3 — 缺口的實務應用

### 核心概念
- 缺口 = 兩根 K 棒中間的真空地帶（不含上下影線）。
- 缺口形成原因 = 主力對該檔不計價的買進（多方缺口）或賣出（空方缺口）。
- 多方缺口未被封閉前，缺口區為支撐；空方缺口未被封閉前，缺口區為壓力。

### 進場/出場規則
- **多方缺口進場：** 量縮回到缺口附近，未跌破缺口低點 → 可在缺口高點附近切入。
- **多方缺口停損：** 收盤價跌破缺口低點 → 視為轉弱，出場。
- **空方缺口進場（回補轉強）：** 收盤價站上空方缺口高點（封閉缺口）→ 視為轉強，可進場。
  - 最強訊號：封閉紅K的低點在缺口高點之上（勝率 60-70%）；
  - 一般訊號：封閉紅K 整根穿過缺口（勝率約 40%）。
- **空方缺口進場停損：** 封閉紅K 的低點。

### 重要術語定義
- **缺口高點 / 缺口低點**：缺口上下兩個邊界，不含上下影線，必須真空。
- **封閉缺口**：收盤價跨越缺口的邊界（多方缺口被跌破低點 = 失守；空方缺口被站上高點 = 轉強）。
- **轉強K**：封閉空方缺口的紅K，收盤價 ≥ 缺口高點。

### 可量化參數
- 缺口判定：收盤價為準（不看盤中）。
- 強訊號 vs 普通訊號的經驗值：60-70% vs 40%（主力大實戰經驗，非回測）。

## Ch2-4 — 量價關係判讀

### 核心概念
- 「量是因，價是果」。
- 健康量價：價漲量增 / 價跌量縮。
- 不健康量價：價跌量增（大量黑K）→ 籌碼凌亂、多數人套牢。
- 紅K 出量 = 攻擊；後續最好量縮整理。
- 大量 K 棒（紅或黑）在股價左上方 = 壓力；左下方 = 支撐。

### 進場/出場規則
- **波段進場（量縮整理）：** 攻擊紅K 後出現量縮 + 回跌至支撐（缺口/均線/大量低點） → 切入。
- **大量黑K 出現後處理：**
  - 若股價跌破大量黑K 低點 → 出場（避免長期整理或下跌）。
  - 重新進場條件：收盤站上前波大量K 棒（含黑K）的高點 → 轉強訊號。
- **支撐結構（站上大量高點後）：**
  - 第一支撐 = 大量K 棒高點（最強，不能跌破）
  - 第二支撐 = 轉強K 棒（突破紅K）低點
- **加碼時機：** 股價回測大量高點時，下影線在大量高點附近守住（第一強確認）→ 可加碼。

### 重要術語定義
- **攻擊紅K**：帶大成交量的紅K，啟動段。
- **量縮**：成交量 ≤ 前一根的 1/2。
- **出量**：比前一根 K 棒成交量多。
- **第一支撐 / 第二支撐**：突破大量高點後的兩層防守。
- **量增**：跟前一天比較成交量是否變大。

### 可量化參數
- 量縮定義：≤ 前一根 K 棒成交量的 1/2。
- 量增定義：跟前一天比較。
- 比較對象 = **前一天**。

## Ch3-1 — 大波段選股方程式

### 核心概念
- 上漲背後原因 = 市場認為未來營收/獲利會比現在更好。
- 波段交易看「外資」（外資 = 波段；投信 = 短線）。
- 族群性是關鍵：法人同日大買同一族群 → 後續兩週至一個月仍看好該族群（漲勢延續性高，較不會一日行情）。

### 進場規則（廣篩）
- 從外資（或投信）買賣超清單篩出族群性。
- **籌碼量化條件（族群篩選）：**
  - 外資/投信對個股買超 ≥ 當日總成交量的 1/3
  - 或買超 ≥ 兩萬張
  - 滿足任一即可，最理想為三者皆滿足（族群 + 量 + 比例）。

### 重要術語定義
- **族群性**：法人同日大量買進同類股（金融、鋼鐵、面板、記憶體等）。
- **外資**：波段佈局，看好長期。
- **投信**：短線佈局，有作帳/績效壓力。
- **延續性**：族群同步發動，不易一日行情。

### 可量化參數
- 買超佔成交量 ≥ 1/3
- 或買超 ≥ 兩萬張（或最低 2-3 萬張）

## Ch3-2 — 波段的關鍵因素（波段四個篩選條件）

### 核心概念
- 三面分析：①族群/題材 ②籌碼面 ③技術面，皆需滿足才進場。
- 耐心是波段交易者必要的心理素質。

### 進場/出場規則（完整 SOP）
- **籌碼面（前置條件）：**
  - 外資/投信買超佔成交量 1/3 或 ≥ 兩萬張
  - 並非符合籌碼隔天就進場 → 等待技術面就位。
- **技術面（必要 + 理想）：**
  - 必要：20ma 與 60ma 皆呈現上彎
  - 理想：當前股價離 20ma 在 5% 以內
- **進場：** 籌碼 + 技術面同時滿足當日尾盤切入。
- **加分進場：** 若同時封閉空方缺口（收盤站上缺口高點），多方因子疊加。
- **停損：** 跌破月線（20ma） → 離場；或用大量紅K 低點防守。
- **加碼：** 走出波段後拉回到月線（仍離 5% 以內）→ 可再加一筆。

### 重要術語定義
- **三面分析**：族群 / 籌碼 / 技術。
- **5% 以內**：當前股價與月線距離。

### 可量化參數
- 20ma / 60ma 皆上彎（必要）
- 當前股價離 20ma ≤ 5%（理想）
- 籌碼門檻：買超 1/3 或 2 萬張（沿用 ch3-1）

> **注意：** Ch3-2 講稿全部討論「**單一波段策略**」與其進場條件（包含外資/投信族群篩選 → 月線季線上彎 → 5% 距離 → 缺口加分），**並未介紹「波段四形態」**。前一輪 Haiku 若把這節標為「波段四形態」是誤讀；講稿從頭到尾就是一套 SOP（一個策略），不是四個形態。

## Ch4-1 — 飆股基因方程式

### 核心概念
- 短線交易者屬性：個性急、可盯盤、風險承受度大。
- 短線參照「投信」籌碼（投信有作帳/績效壓力，買進後通常會有發酵）。
- 投信買的標的需深入研究、call 公司、寫報告，因此買進後 1-2 週內有高機率波動。

### 進場/出場規則
- 本章節為「概念引入」，具體進出規則放在 Ch4-2。
- **強化條件：** 外資 + 投信同買 → 加乘效果，短線套牢仍有長線翻身機會。

### 重要術語定義
- **投信認養**（後續 ex2-1 細談）：投信對個股連續性買超。
- 投信多買上櫃（中小型）；外資多買上市（大型）。

### 可量化參數
- 投信效應時間窗：1-2 週內高機率波動。

> - 形態二：反轉形態（突破下降趨勢線 + 兩高點連線）— line 217-356 — **完整內容**

### 形態一：奇形（旗形 + 旗杆）— Ch4-2 line 9-216 ✅

#### 使用情境
- 70% 的情況我們抓不到第一根起漲 K 棒（看到時往往已漲 7-8%）。
- 怕追高 → 用奇形觀察方式安全切入。

#### 進場條件
- 第一根（旗杆）：紅K，可以有上影線但不要太長。
- 第二、三根（旗子）：紅綠不拘
  - 紅K → 需量增
  - 綠K → 需量縮
- 收盤價條件（最關鍵）：
  - 必要：兩根 K 棒收盤價需在 5ma 之上
  - 理想：兩根 K 棒收盤價在第一根（旗杆）收盤價之上（最強勢）
  - 細節：兩根的下影線需在旗杆 K 棒中間以上

#### 進場時機
- 兩種切入方式：
  1. 第三天尾盤 → 模擬切入（積極）
  2. 第四天打到 5ma 位置 → 模擬切入（穩健）

#### 停損
- 跌破 5 日線 → 出場。

#### 風險
- 最強勢的情況下，第四天會直接開高拉走，吃不到 5ma 切入點。

### 形態二：反轉形態（突破下降趨勢線 + 兩高點連線）— Ch4-2 line 217-356 ✅

#### 使用情境
- 抓趨勢反轉的安全切入點。
- 不接下跌中刀；等出現反轉紅K。
- 一般做隔日衝（尾盤切入、隔日開盤賣），因左方壓力大、易回跌。

#### 左方壓力大小取決
- 下降趨勢線斜率（越斜壓力越大，肉眼判斷）。
- 左方大量 K 棒越多 → 壓力越大。

#### 均線條件
- 5 / 10 / 20 ma 至少三條在反轉紅K 之下（不能切到 K 棒）。
- 反轉紅K 必須整根站上所有均線。
- 短均線開始上彎（用扣底值判斷，原本就為多頭排列最佳）。

#### 進場條件
- 紅K 底部站上下降趨勢線（兩高點連線下移後的趨勢線）。
- 紅K 底部站上所有均線。

#### 進場時機
- 安全切入點：反轉紅K 的下方 1/3 處（隔日操作）。
- 越靠近紅K 低點切入 → 風報比越好。

#### 停損
- 收盤跌破反轉紅K 低點 → 離場。

#### 後續延伸
- 反轉紅K 之後常見奇形延伸，可繼續沿用形態一規則加碼。

### 形態三：布林上軌進出策略 — Ch4-2 line 357-493 ✅

#### 使用情境
- 標股齊漲策略；具備追高風險，須評估自身承受度。
- 此策略不需提前佈局，等突破才確認。

#### 前置條件 — 通道形狀
- 通道不可為下降趨勢（向下不適用，請用反轉策略）。
- 最佳：橫盤壓縮的通道。
- 次佳：上升通道（爆發力較弱）。
- 通道窄度量化：(上軌 − 下軌) / 中軌 < 10% → 起漲爆發力強。

#### 通道組成
- 上軌（通道頂）、中軌（20ma 月線）、下軌（通道底）— 任何券商軟體 BB Band 皆可顯示。

#### 進場條件（起漲 K 棒）
- 收盤價 > 上軌
- 必須帶量（出量）
- 突破前一天通道窄度 < 10%（事後確認）

#### 進場時機
- 突破當天尾盤 → 模擬切入（一買點）
- 次日上軌位置 → 二買點（不能盯盤者：上軌 × 0.99 作為預估參考）

#### 出場
- 收盤實體綠K 跌入上軌之內 → 賣出。

#### 雙面刃特性
- 股價必須跳出上軌；若停在上軌之下，會被上軌往下打。
- 「股價齊漲齊跌的特色 → 通道由窄變成喇叭狀」。

### 形態四：布林回測策略 — Ch4-2（從 PDF p.127 補完）

> **講稿狀況：** ch4-2 講稿在 line 494「啊，我們就做一個」斷掉（影片實長 72:08，講稿只到 44:10）。本節從簡報 PDF 補完。
> **資料來源：** `stock-analysis-system/docs/materials/1627272942273-主力大全方位操盤教戰守則.pdf` p.127

#### 策略目的
- 抓強勢股的「第二波攻擊」。
- 避免「買在突破前高」的獲利賣壓風險，改用「量急縮 + 技術面/籌碼安定」時切入。

#### 前置條件
- **必須先符合形態三（布林上軌）**：曾經跳出布林上軌。

#### 進場核心條件
- 急漲後回跌至 **中軌 = 20MA（月線）**。
- 回跌期間 **量縮**。
- **量縮理想定義：** ≤ 急漲段大量紅K 的 **1/5（20%）以下**。
- 最高原則：**不能跌破中軌**。

#### 切入時機（兩種）
- **積極者：** 當日尾盤切入。
- **穩健者：** 確認次日低點比前一日高後，尾盤切入。

#### 停損
- **跌破中軌（20MA）→ 停損。**

#### 停利
- 沿用形態三出場邏輯：**K 棒離開上軌（收盤實體綠K 跌入上軌之內）→ 賣出。**

## Ch5-1-1 — 動能仔的選股設定（當沖前夜篩選）

### 核心概念
- 大盤每天 3-4 成成交量來自當沖。
- 當沖需要「波動」與「量能動能」才沖得起來。
- 主力大不鼓勵新手做當沖，但需具備基本認知。

### 進場（盤前篩選）— 量化參數設定
- 均線排列：5ma > 10ma > 20ma（多頭排列）
- 三條均線皆上彎
- **成交量：兩天皆 > 兩萬張**
- **三天區間振幅 > 8%**（平均一天 2.5%）
- **三天周轉率 > 20%**
- 當前股價離月線（20ma）< 30%

### 進階優化
- 若篩出太多 → 提高周轉率至 30-40%
- 加入投信買賣超條件 → 提高發動機率

### 視覺優先過濾（盤中即時用）

> 補強來源: 5/19 當沖復盤實戰課

- **「右上角第幾段」原則**：日 K 圖右上角已連續上漲 3 段以上 → 開高紅 K 都是給你賣的、不切入
  - 老師原句：「走到第 3 段的基本上啊，都是紅黑拉上去就丟掉了」「走到第 4 段了知道嗎」
- **「離均線太遠」原則**：股價偏離 20ma 視覺上已走到右上角末端 → 直接過濾，不必再算 30% 量化值
  - 老師原句：「他已經離均線非常遠了」「右上角已經長很多的了開高就是丟掉」

### 重要術語
- **動能仔**：有量能、有波動、可當沖的標的。
- **周轉率**：人氣指標；越高代表有人撿/丟換手活絡。

## Ch5-1-2 — 案例帶入（前夜篩 → 隔日操作）

### 核心概念
- 用 ch5-1-1 的 6 個參數設定每日篩出 5-10 檔當沖候選。
- 思維切換：當沖選到的標的若跳空漲停 → 不要賣，轉為短線/波段思維留倉。
- 留倉是動態調整：例如 2 張中先出 1 張達標 2-3%，留 1 張看後續波段。

### 進場規則
- 沿用 ch5-1-1 參數設定篩出標的。
- 隔日盤中若拉出漲停且解鎖次數少（1-2 次以下）→ 不賣，留倉。
- 開開關關 3-4 次 → 考慮減碼。

### 重要術語
- **思維切換**：當沖 → 短線 → 波段的彈性轉換。
- **鎖漲停 / 鎖不住**：漲停掛單能撐稱「鎖」、被吃光稱「鎖不住」；解鎖次數即上述「開開關關」。

### 時段紀律與訊號品質（spec 補強）

> 補強來源: 5/19 當沖復盤實戰課 audit 2026-05-28

- **最佳訊號時段**：開盤後 0–10 分鐘訊號品質最高；9:10 後通知品質明顯下降
  - 老師原句：「集中都在 9 點 10 分以內通知集中 9 點 10 分以內好其他都沒有什麼好的通知了」
- **只做首發訊號**：同一檔當日第二次以上的訊號跳過、只做第一次
  - 老師原句：「都是第二次第二次第二次那就我們都不看這種就看第一次出現的時候」
  - 邏輯：首發 = 臨界突破點；後續 = 已發展中、追單勝率低

## Ch5-2 — 兩大精選關鍵（從 5-10 檔篩到 1-3 檔）

### 核心概念
- 從前夜跑出的清單再精選；兩大關鍵：
  1. **準突破（位置）**：準備突破前面高點，距離前高 < 10%（隔日漲 10% 以內可過高）
  2. **量（動能）**：右下角近期量 > 左邊前高的量

### 進場規則
- 同時滿足：
  - 距離前高 < 10%
  - 近期量 > 前波高點量

> ⚠️ **實戰場景化補強 (audit 2026-05-28)**：5/19 實戰課中「距前高 < 10%」的具體場景多為「**昨日漲停 + 今日不跳太高（< 5%）**」、老師原句：「漲停後的那一天很重要，漲停然後你隔天不知道怎麼操作很簡單，然後你就把我們這課程的精髓記起來」

### 與 ch2-4 連動
- 呼應 ch2-4「衝擊前高需要更大量」的原則。

### 量的反向使用

> 補強來源: 5/19 當沖復盤實戰課

- 「量越大越好」只在突破當下成立
- 拉抬末段出現同等級「大量」= **減碼訊號**（不是加碼訊號），除非快進漲停
  - 老師原句：「這邊爆大量注意，這邊爆大量前面的先丟掉一下，除非他已經快進漲停了」

## Ch5-3 — 當沖進出場 SOP（盤中）

### 核心概念
- 進場勇氣 = 出場勇氣。
- 盤中三大判斷工具：
  1. **均價線（加權平均線）** = 當日大部分人的買進成本
  2. **5 分 K 高點 + 5 分 K 的 5ma**
  3. **江波圖**（一波比一波高 / 低判斷）

### 進場規則（買進前）
- 股價必須在均價線之上
- 突破第一根 5 分 K 的高點 → **訊號觸發**（非切入點，切入時機見「進場時點細節」段）

> ⚠️ **spec 寫不精準修正 (audit 2026-05-28)**：原 spec 寫「突破第一根高點 → 切入」、實戰中老師示範是「過高 = 訊號、等 9:10 後回踩 10MA 守住才切入」。

### 出場/停損規則
- 停損雙錨點（任一跌破即停損） — 詳見下方「停損雙錨點」段
- 跌破 5 分 K 的 5ma → 轉弱出場

### 江波圖判斷（買進後）
- 上漲時：一波須比前波高，否則轉弱要減碼
- 下跌時：一波沒比前波低 → 止跌訊號
- **雙向流程（spec 補強）**：賣出後若價格站回前波高（5MA 收復）→ 可再買回繼續做下一段
  - 老師原句（強茂案例）：「我原本賣的位置，站回來之後我再再買回來」
  - 不視為破紀律，江波圖是雙向觀察工具

### 5 分 K 理想條件（早盤 15 分鐘判斷）
- 均線（5/10/20ma）向上糾結、尚未發散；**發散 = 硬性過濾**
  - 老師原句：「軍心發散了…不符合我們喜歡的長相」
- 開盤幅 < 4% 為一般條件；**前 5 分鐘漲 > 5% 為一票否決**（嚴格升級）
- 第一根 5 分 K 為實體紅K（實體 > 上影線）
- 第二根紅K 不跌破第一根 K 棒
- 切入：直接過第一根高點 OR 等第二、三根回到 5ma 位置接

### 停損 / 留倉策略
- 跌破 5 分 K 的 5ma → 全出
- 留倉策略：可保留一張（如 5 張裡留 1 張）用長紅K 低點做防守 → 轉為隔日留倉思維

### 達標
- 當沖達標：**1.5% – 3% / 每段**（3% 為老師口中「滿分標準」）

> ⚠️ **spec 寫不精準修正 (audit 2026-05-28)**：原 spec 寫「2.5%」、老師實戰口頭「2-3% 算很 OK」「滿分是 3%」、上限應為 3%。

### 進場時點細節（盤前 spec 補強）

> 補強來源: 5/19 當沖復盤實戰課

- **開盤前 10 分鐘（09:00–09:10）不切入**，只觀察方向與量能
  - 老師原句：「我們都不會去衝一開始前 10 分鐘，前 10 分鐘是讓你通知給你之後你就去看一下他後續 10 分鐘走的狀況」
- 9:10 後等**回踩 10MA 守住** → 才切入（不是直接過高就衝）
- **前 5 分鐘漲幅 > 5% → 整檔放棄**（比原 spec「開盤幅 < 4%」更嚴格的一票否決）
  - 老師原句：「一開始開盤前 5 分鐘如果他已經拉到拉了 5% 以上的，基本上你這個就是有點堵了」

### 停損雙錨點（spec 補強）

> 補強來源: 5/19 當沖復盤實戰課

- **主錨**：第一根 5 分 K 低點（原 spec）
- **副錨：昨日收盤價**（老師反覆強調 ≥ 15 次，為第二道防線）
  - 老師原句：「就是開盤價、昨天的收盤價，就是兩個價格是很重要的」「跌破昨天的收盤價，所以你說收盤價這麼說超級重要」
- 跌破任一錨 → 停損出場

### 達標進階規則（spec 補強）

> 補強來源: 5/19 當沖復盤實戰課

- 達標 1.5–2.5% → 離場（原 spec）
- **例外 1**：拉抬末段爆大量 → 先出一半（不是加碼）
- **例外 2**：爆大量且快進漲停 → 留倉轉短線思維
- **例外 3**：兩三根 K 不再創高 → 主動掀傘出場（不等停損價觸發）
  - 老師原句：「沒人要拉了，就掀傘」

### 進場「試溫」哲學（spec 補強）

> 補強來源: 5/19 當沖復盤實戰課

- 不一次梭哈，分批進出 ping-pong 試溫
  - 老師原句：「就進進試一下出出丟一下試一下這樣子」
- 加碼門檻：5MA 與 10MA 確認不交叉糾結 → 才加
- 均線交叉糾結時：先撤出、等月線 / 5MA 重新表態再進

### K 棒形態過濾（spec 補強）

> 補強來源: 5/19 當沖復盤實戰課

- **上影線 > 實體** → 不做（原 spec 已有實體紅 K 條件，此為確認）
- **下影線 > 實體（掉人線 / 吊人線）→ 不做**
  - 老師原句：「掉人線掉人線不要做，掉人線很可怕」（出現 ≥ 6 次）
- **紅吞噬之後下一根**：必回踩（「吃完這個黑幫他休息一下」），不追、**等回踩前波支撐再進**
  - 老師原句（強茂案例）：「我的認知是黑黑灌下去隔天下一根收復回來的時候可以會在這個地方 130 附近再踩一下，其實有踩到」
- **5 分 K 掉人線**：分 K 連續出現 2-3 根下影線過長 → 即使整體上漲也出場
  - 老師原句：「分 K 裡面不管其實不管分 K 還是日 K 啦我都很討厭這種掉人線」
- **區間盒過濾**：5 分 K 同時出現長上影線 + 長下影線 → 視為區間盤整、畫盒等待突破/跌破再進、不在盒中追
  - 老師原句：「他常常上一線常下線然後本身可以帮幅度很長的很寬的你就把畫兩條上下畫兩條畫過去他就會在這個區間裡面正」

### 踩回進場 SOP（spec 補強）

> 補強來源: 5/19 當沖復盤實戰課 audit 2026-05-28

- 吞噬大黑 K 後若隔天 / 下一根**開低 + 踩到前波支撐 + 收紅** → 進場
- 不買最低點、買 **+0.5% 確認位**
- 老師原句（強茂案例）：「他其實這個地方有踩，有開低然後踩一下…我說這個地方就可以上車在這個可能不會買在最低的可能就 130 點 5、131 附近我慢慢上車」

### 加碼前置條件（spec 補強）

> 補強來源: 5/19 當沖復盤實戰課 audit 2026-05-28

- **5MA + 10MA 排列清楚**（不交叉糾結）
- **連續兩根站穩 5MA**
- 上述未滿足前只做試溫部位、不加碼
- 老師原句：「軍事還是很混亂我沒辦法加碼，等一下…好這個地方可以進去加碼，站上無日線兩根了」

### 大盤環境前置條件（spec 補強）

> 補強來源: 5/19 當沖復盤實戰課 audit 2026-05-28

- 大盤大跌日（如 5/19 跌 700 多點）→ **只做開盤前 30-60 分鐘**的多方機會
- 9:10 後幾乎沒勝率（與「最佳訊號時段」段呼應）
- 老師原句：「我們這邊都是只做前半小時、前一小時」「盤勢今天真的很差…尾盤很爛大盤很爛就下去了」

### 停損距離合理性檢查（spec 補強，示範性質非硬性）

> 補強來源: 5/19 當沖復盤實戰課 audit 2026-05-28
> ⚠️ 老師只**示範計算**、未明說「必須 R:R = 1:X」；此條為計算示範、非硬性比率

- 進場前快速心算：「停損點 vs +3% 達標」距離合理性
- 老師示範（環科案例）：成本 484、停損 478.5（差 1.5 元 ≈ -1.1%），目標 +3% ≈ R:R 1:2.7
- 老師原話：「停損是 47.5（即 478.5），我覺得這還 OK，因為我們希望後面可以賺個 3% 嘛」

### 重要術語
- **均價線**：加權平均線，當日大部分人的買進成本。
  - ⚠️ 註（audit 2026-05-28）：5/19 實戰課中老師多以「昨日收盤價 + 5 分 K 5MA」為主要錨點，均價線為輔。
- **江波圖**：上漲下跌的波段比較圖（觀察一波比一波高 / 低）。
- **5 分 K 切入法**：早盤 15 分鐘判斷一日方向。
- **掉人線（吊人線）**：下影線遠長於實體的 K 棒，當沖一票否決形態。
- **兩個價格**：昨日收盤價 + 今日開盤價，盤中多空最重要的兩個錨點。
- **守住（踩住 / whisper 譯「手助」）**：盤中 K 棒回測支撐位有人接（紅 K 或下影線買回）→ 支撐有效。
- **掀傘**：拉抬主力收手、流動性消失、主動出場訊號（不等被動停損）。
- **右上角第 N 段**：日 K 圖已連續上漲段數；三段以上紅 K 拉上去都是給你賣的（取代量化「距 20ma < 30%」的視覺判斷）。

## Ch5-X — 當沖做空 SOP（多方做多形態的鏡像）

> 補強來源: 5/19 當沖復盤實戰課
> 老師原句：「掉了你把你把剛做多的那個這個鏡面圖找出來」「多空在一念之間」

### 核心概念

- 做空形態 = 做多形態翻轉（鏡像對稱）
- 不是隨時能做空；須盤中確認**空方控盤**才啟動

### 做空切入條件

- 日 K 均線空頭排列（或盤中均線由多轉空糾結）
- 盤中 5 分 K 走勢：紅 K 拉上 → 黑 K 灌下 → 彈上守不住均價線 = 空方控盤確認
  - 老師原句：「紅 K 上來空下去（空方控盤判斷）」

### 空方停損

- 停損錨點：**昨日收盤價**（回補）+ **今日開盤價**（同多方邏輯對稱）
- 反彈過均價線且站穩 → 停損回補

### 空方出場（達標）

- 殺到支撐底部、走不動 → 回補
- 老師原句：「最後拉沒有拉上去就全數離開」（空方同理：最後殺沒再破 → 回補）
- 多空均只做**前半小時至一小時**：「我們這邊都是只做前半小時、前一小時」

## Ch6-1 — 隔日沖提升勝率心法

### 核心概念
- 隔日沖 = 今天尾盤買、隔天開盤賣。
- 目標：每日 1-2% 達標即滿足，不求大賺只求穩定勝率。
- **隔日沖不是每天都能做**。
- 紀律：今天尾盤買，隔天開盤一定要賣（不管賺賠）。

### 進場前判斷（最關鍵）
- 大盤 / OTC 指數須為多頭架構：
  - **當日紅K**
  - **量增紅K**
  - **站上 5ma**
  - 尤其櫃買指數（OTC）必須在 5ma 之上

### 進場時機
- 1 點 開始選擇標的
- **1 點 20 到 1 點 25** 進場
- 越早進場勝率越低

### 出場規則（盤前預掛）
- 8:30-9:00 開盤前預掛跌停賣 → 用開盤價賣出
- （漲停買、跌停賣 = 以開盤價成交，不是真的漲跌停價）

### 進階出場規則
- 跌破第一根 5 分 K 低點 → 出場（適用國際利空或期待續強）
- 持有 2 張 → 1 張開盤賣、1 張用 5 分 K 低點觀察續強機會

### 重要術語
- **盤前預掛**：8:30-9:00 預掛漲停買/跌停賣，以開盤價成交。
- **續強條件**：開盤未馬上賣的續留條件，看 5 分 K 低點防守。

## Ch6-2 — 尾盤隔日沖標的速篩

### 進場速篩參數
- 漲跌幅：紅K 大於 0.1 + 一萬張以上
- 月線斜率 > 0.4（很陡的上升）
- K 棒突破布林高軌（用還原日K）
- 距上軌 < 6%

### 精選人工判斷
- 出量（比前一天量增）
- 紅K 站上布林上軌
- **理想型態：** K 棒底部（開盤價/實體）在上軌之上；最佳是整根 K 棒包含下影線都在上軌之上。

### 進場時機
- 盤中 1 點 20 確認進場（不要過早，會套牢）。

### 續留條件（進階融合當沖）
- 第二天 15 分鐘判斷法（沿用 ch5-3）：
  - 跳空開高未攻擊 → 開盤就走
  - 守住第一根 5 分 K 低點 + 出量突破第一根 5 分 K 高點 → 加碼或續留
  - 開高直接拉走 → 用跌破 5 分 K 的 5ma 出場

### 三種開盤情境
1. 跳空開高後 15 分鐘弱掉 → 開盤離場
2. 守住第一根 5 分 K 低點 → 出量突破第一根 5 分 K 高點 → 加碼
3. 開盤就跳很多 → 第二根直接拉出 → 停利用跌破 5 分 K 的 5ma

### 可量化參數
- 漲跌幅紅K > 0.1，量 > 1 萬張
- 月線斜率 > 0.4
- 距上軌 < 6%

## Ch7-1 — 沒有停損點就沒有進場點

### 核心概念
- 風險不可怕，可怕的是不知風險在哪。
- 進場前必須確認停損點，且確認你能接受該停損距離。
- 越靠近關鍵支撐進場 → 試錯成本越低、風報比越好。

### 課程教過的「關鍵支撐」清單
1. 大量低點
2. 慣性均線
3. 缺口支撐
4. 轉強K 棒低點

### 觀念性原則
- 有勇氣按買進，就要有勇氣按賣出。
- 投資是一輩子的事，不要把它看短。
- 要有能力判斷盤勢（多頭做多、空方做空或空手、盤整低買壓力賣）。
- 不是每天都要做股票。
- 選你會做的盤勢。
- 困難時降低部位（< 30-40%）。

## Ch7-2 — 資金配置

### 核心概念
- 沒有標準答案；依交易屬性（積極/穩健、有/無耐心、能/不能盯盤）調配。
- 投資屬性決定資金分配比例。

### 範例配置
- 無耐心、想賺快錢、可盯盤：短線 50%、隔日沖+當沖 30%、波段 20%。
- 上班族無法盯盤：波段 60%、短線 20%、隔日沖預備金 20%；可再加存股。

### 重要術語
- **股性**：標的的波動幅度（短線屬性 vs 波段屬性）。
- **波段標的**：股性溫吞、適合長抱。

## Ch7-3 — 收高開低 / 收低開高（主力意圖判斷）

### 核心概念
- 開盤價決定主力今天的意圖。
- 不去歸咎原因，只看現象、看訊號、做動作。

### 收高開低（轉弱訊號）
- 定義：前一天出量實紅 K 收最高（或收相對高點），次日開平甚至開綠。
- 動作：當天看到 → 不硬作多，甚至可作空。
- 主力意圖：不計價賣出，思維上對該股轉變。
- 拉到前一天收盤價 = 強壓力（需更強買盤才回得去）。
- 若已持有：開盤離場（後續真有突破訊號 = 帶量突破前一天收盤價，才接回）。

### 例外（不適用）
- 國際性利空（道瓊、夜盤大跌）→ 普遍性開低，不是個股因素 → 不適用收高開低法則。

### 收低開高（轉強訊號）
- 定義：前一天出量實黑K 收最低（甚至跌停），次日開平甚至開紅。
- 動作：視為轉強訊號，可進場。

### 開平盤的恐怖
- 漲停的標的隔天開平盤（沒有開高）也是危險訊號。

### 重要術語
- **開盤價 = 主力意圖**。

## Ex1-1 — 窒息量的意義

### 核心概念
- 「新手看價、老手看量、高手看籌」。
- 窒息量出現 = 主力已沒有籌碼可再賣（再跌下去傷主力成本）。
- 窒息量必須前一波有攻擊過（有主力進貨量在前）。
- 窒息量是「等待 + 積蓄能量」的過程，一出量就出方向。

### 重要術語
- **窒息量**：成交量縮到極低（搭配價格平穩），代表主力沒有籌碼可再倒。
- 對應「下跌量縮、橫盤量縮」的健康量價。

## Ex1-2 — 高勝率窒息量參數設定（最重要的 SOP）

### 進場條件
- **成交量條件**：當日成交量 < 20 日內最大量的 10%
- **月線必須上彎**（必要條件，避免真的窒息掉的股票）
- **理想搭配**：5 / 10 / 20 / 60ma 排列正確且皆上彎
- **窒息量後一根 K 棒（出量 K）：**
  - 紅K（含實體紅K、紅K加少量上影線）
  - 或下影線長於實體的綠K（下影線必須長於實體）
  - **不能是實體黑K**
  - **不能是實體綠K**
- **出量定義**：比前一根窒息量多就算出量

### 停損
- 窒息量後出量 K 棒的低點 → 跌破出場

### 重要術語
- **窒息量**：< 20 日最大量的 10%
- **出量 K**：窒息量後第一根放量 K 棒

### 可量化參數
- 成交量 < 20 日最大量 × 10%
- 月線（20ma）必上彎
- 出量 K 限定：紅K / 長下影綠K（下影 > 實體）

## Ex1-3 — 窒息量進出實戰應用

### 案例彙整
1. **3533 嘉澤**：窒息量 → 出量黑K 不進（不符 SOP）→ 等下一根出量紅K → 進。
2. **8150 南茂**：窒息量 → 出量紅K → 停損 0.6 元換 8 元獲利。
3. **6284 佳邦**：窒息量 → 出量紅K → 停損 2.7 元換 17 元獲利。
4. **2338 光罩**：弧形整理後窒息量 → 出量紅K → 短期翻倍。
5. **1590 亞德克**：弧形 + 凹洞量 → 出量綠K（下影長）→ 進場。

### 如何發現窒息量（操作流程）
- 第一步：回想前一波拉抬過的標的，看成交量是否開始縮。
- 第二步：正在拉抬中、覺得沒有切入點的標的 → 放入自選等待 SOP。
- 第三步：追蹤與等待（需要耐心）。

### 最後叮嚀
- 月線必上彎（避免真的窒息掉）。
- 5/10/20/60 都排列正確會更完美。

### 重要術語
- **凹洞量**：弧形整理時量縮到極低，形成「凹洞」。
- **弧形**：底部呈弧形整理，主力大喜歡的型態。

## Ex2-1 — 投信的習性

### 核心概念
- 法人三類：外資（長線）/ 投信（短線、本單元主角）/ 自營商（過短，主力大不用）。
- 投信 = 國內證券投資基金（元大、野村、安聯、富邦等）。
- 投信特性：資金較小 → 鎖定中小型個股；操作週期數週至 1 季。

### 投信績效壓力
- 每季結算：1-3 / 4-6 / 7-9 / 10-12 月。
- 6 月底（半年底）、12 月底（年底）衝刺最明顯 → 投信作帳。

### 重要術語
- **投信認養**：投信對個股連續性買超。
- **投信作帳**：季底前衝刺，推升持股股價。
- **結帳**：累積到一定水位後反手賣出獲利了結。

### 進場/出場規則
- 作帳要提前一個月開始（不要在最後一週才佈局，否則容易被結）。
- 季底最後一週投信才開始大買的標的 → 可能是下一季重點。

### 量化警戒
- **投信持股 > 12%** → 隨時可能轉手賣出
- **進場最理想**：投信持股 3-5% 仍處於佈局期
- **必須小心**：> 12%、15%、20% 區間

### 案例
- 茂達：庫存 17% → 一波下殺。
- 弘塑：庫存 24% → 拉高出貨 → 從 400 腰斬到 250。

## Ex2-2 — 投信策略參數設定 + 進出規劃

### 進場條件
- **籌碼面：** 投信買超佔股本比 5 天 > 1.5%（「近 5 日 + 1.5%」）
- **精選條件：** 必須是「剛上榜」（前幾天未上榜過）
- **技術面：**
  - 必要：5ma > 10ma > 20ma，皆上彎，尚未發散
  - 理想：加上 60ma

### 進出規劃
- **十日線進出**（通用性）：跌破十日線出場、出量站回十日線接回。

### 操作原則
- 第一天上榜不進場（除非積極派可隔天 5 分K SOP 切入）。
- 穩健派：等回到十日線附近排好、出量站上十日線再切入。
- 黑K 吞噬前一天漲停 = 出場訊號。
- 「籌碼 + 量價 + 窒息量整合」：窒息量 + 投信首買 = 高勝率組合。

### 可量化參數
- 投信買超 5 天累計 ≥ 1.5% 股本
- 十日線進出

## Ex2-3 — 投信首買（加碼內加碼）

### 核心概念
- 「投信首買」= 新手也能用的高勝率單一策略，課綱外加碼內容。
- 利用投信的內部資訊（每天 call 公司、看報告，廣度與深度都比散戶高）。

### 進場邏輯（雙條件）
1. **投信突然買第一天，至少 200 張**
   - 可調寬到 50 張，但勝率會降低；不可低到 5 張（不算首買）。
   - 前面至少 2 個月（最好 3 個月）沒有投信買超 → 越長空白越好。
2. **當日個股收黑K，但投信買超（價籌背離）**
   - 黑K + 投信買超 → 可能是被散戶賣下去，主力/法人接 → 多方因子。

### 搭配條件
- 籌碼 + 技術面：均線多頭排列、未發散 → 勝率加成。
- 技術面不好但夠低檔（回到前波起漲點）→ 仍可進場。

### 篩選法
- 用「投信買超大於 N 張 + 投信連 1 日買超」篩。

### 重要術語
- **價籌背離**：價跌 + 籌增 = 好事（散戶賣、法人接）。
- **投信首買**：前 2-3 個月無投信買超，今日首次 ≥ 200 張。

### 可量化參數
- 投信首日買超 ≥ 200 張（可降至 50 張，勝率降低）
- 前 2-3 個月無投信買超
- 當日 K 棒：價跌 + 投信買超 = 價籌背離
