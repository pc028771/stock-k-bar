# 主力大 Pattern：老師預告 → 確認進場 → 整理守住 → 突破

> **來源**：2026-05-22 Line 群兩個成功案例
> - 力積電 6770（煒璁）— 早盤看 MA5/MA10 預備 + 群創概念
> - 晶技 3042（陳慧芬）— 老師週日課程點名，週一進場，三天守住，今天漲停

---

## ⚠️ 核心原則（最重要）

**老師講完不是要馬上進場，而是要密切觀察、自己判斷。**

### 老師 vs 自己的分工

| 角色 | 負責 |
|---|---|
| **老師** | 大盤趨勢、族群輪動、題材脈絡、籌碼概觀 |
| **自己** | 個股功課（K線結構、MA、扣抵、量價、進出場時機）|

**判斷 = 應用老師的教學方法在個股上**，不是等老師告訴你進場價。

老師幫你**過濾族群方向**（這禮拜看 ABF / 連接器 / IC設計），
個股的具體進場、停利、停損**都要自己依課程框架算出來**。

### 三個禁忌

- ❌ 老師喊就買 → 沒用課程方法確認 = 賭
- ❌ 進場後不看 → 「撐住觀察」是每日工作
- ❌ 跟著群組情緒走 → 同學說「飛天」不代表自己該追

老師的話：「我希望可以幫大家建立獨立的個體就辦法有自己的判斷能力」

---

## Pattern 五階段

### Stage 1：老師預告（Watch List 進入）

**訊號來源**：
- 週日 PressPlay 培訓課程（個股名單）
- Line 群直播提示（族群、扣抵值預判）
- 盤後文章「下波族群」「圈圈內」提示

**這個階段的正確動作**：
- ❌ 不是直接買
- ✅ 把這檔加入觀察清單，開始每日追蹤
- ✅ 記錄老師點名當下的價位、形態、為什麼點名

**特徵**：
- 老師當下**不一定**已經攻擊
- 通常處於底部/整理區
- 給出「為什麼這檔」的故事（族群、題材、籌碼）

---

### Stage 2：確認進場（Entry）

**用老師教過的方法確認強勢，不要單純買「老師有講」**

| 確認項目 | 課程依據 | 訊號 |
|---|---|---|
| **MA5/MA10 預備** | 群創 N字概念 | MA5 接近 MA10、即將糾結向上 |
| **扣抵值上彎** | 扣抵原理 | 隔日扣抵 close < 今日 close → MA 將上彎 |
| **小結構整理到末端** | 群創 N字案例 | 高點橫盤 5-7 天，量縮但價守 |
| **第一根紅K + 量增** | K棒解讀 | 整理區末端的攻擊訊號 |

**力積電案例**（煒璁 5/22 早盤）：
- 5/19、5/20 連續下跌至 58.5、57.8
- 5/21 開始三陽開泰（三根紅K）
- 早盤看 MA5、MA10 即將糾結 → 進場
- 「使用了您解說的群創概念」= N字上攻形態

**晶技案例**（陳慧芬 5/19 週一進場）：
- 老師週日課程點名
- 5/19 開盤後依老師方法進場（收 148，在 MA10 附近）
- 進場 K 低約 143

---

### Stage 3：整理守住（Hold Through Noise）

**特徵**：
- 進場後 2-5 天**不會直接噴**
- 大盤可能不好（同學說「盤爛」）
- 個股守住關鍵支撐：進場 K 低 / MA5 / 前低

**晶技案例**：
- 5/19 進場 148
- 5/20 收 148（盤整）
- 5/21 收 148（盤整）
- 三天都守在 148 附近，最低觸 143（5/20 low）但收回
- 同學原話：**「前三天盤爛還守住」**

**老師課程依據**：
- 收盤確認原則 → 盤中跌不算
- 「進場 K 低」是結構底，沒跌破就還在
- 「成本高的人會緊張」 → 守住代表有人撐

**晶技案例的關鍵啟示（撐住觀察的重要性）**：
- 老師週日點名 ≠ 週一買進就會漲
- 5/19 ~ 5/21 三天「盤爛」，但個股在 143-148 區間守住沒跌破
- 如果這三天有人因為「沒漲」而出場 → 錯失 5/22 的漲停
- **「撐住觀察」是每日例行工作**，不是設好停損就放著

### 撐住觀察的每日 checklist（每日盤後）

對每檔「老師預告 + 已進場」的個股做：

| 檢查項 | 課程依據 | 標準 |
|---|---|---|
| 1. 是否收盤跌破進場 K 低？ | 結構支撐 | 跌破 = 出場訊號 |
| 2. 是否收盤跌破 MA5？ | 趨勢追蹤 | 連兩日跌破 = 警示 |
| 3. 是否出現大黑K完整包覆前段漲幅？ | K棒型態 | 出現 = 停損訊號 |
| 4. 量能變化 | 量價結構 | 量縮整理 = 正常；量增收黑 = 警示 |
| 5. 是否仍在族群同向動能？ | 族群輪動 | 族群一起跌 = 整體 risk-off |
| 6. 老師有沒有更新看法？ | 訊息追蹤 | Line 群 / 文章新訊號 |

**沒有任何一項紅燈 → 續抱，等 Stage 4 突破**
**有任何紅燈 → 評估是否仍在課程框架支持下**

---

### Stage 4：突破（Breakout）

**訊號**：
- 量增爆發 + 突破整理區高點
- 紅K body 大、上影線小
- 通常一根衝出整理區，甚至漲停

**晶技 5/22**：漲停（最高 162.5，從 148 一根攻出）
**力積電 5/22**：三陽開泰後當天進場

---

### Stage 5：續抱或停利（已超出本 pattern）

依課程框架另行判斷：
- 沿 MA5 續抱
- 上影線 > 5% → Ch2 警示
- 收盤跌破 MA5 → 退場第一訊號

---

## 為什麼這個 Pattern 比「追老師喊單」勝率高

| 對比項 | 純追單 | 預告 + 確認 |
|---|---|---|
| 進場時機 | 老師喊那天追 | 等個股自己「準備好」 |
| 成本 | 老師喊單後常開高 | 整理區低點，成本低 |
| 套牢風險 | 高（追上影線）| 低（守不破前低）|
| 心態 | 焦慮（怕老師沒喊）| 安心（自己確認過）|
| 課程契合度 | 低 | 高（用扣抵、N字、MA 排列）|

## SOP 應用

下次有新標的時，依序問：

1. **Stage 1 ✓？** 老師有沒有在課程/文章/Line 群明確提到？
2. **Stage 2 ✓？** 用課程方法（MA5/10 排列、扣抵預判、N字形態）確認個股自己「準備好了」？
3. **Stage 3 守得住嗎？** 進場後幾天，關鍵支撐有沒有破？
4. **Stage 4 出現了嗎？** 量增 + 突破前高才算成立

四個都打勾才是課程框架認可的進場，否則就是賭。

## 補充：老師講的個股「沒有明確支撐」怎麼辦？

老師點名只代表「趨勢/族群」對，**個股結構不一定到位**。
個股若沒明確支撐（剛跌完、上影線多、量爆），不能直接追，但可以：

### 等回檔 → 下試單觀察

| 步驟 | 動作 | 課程依據 |
|---|---|---|
| 1. 不追當前高 | 等大盤雜訊洗回 | 「不要開盤開高追進去」|
| 2. 確認潛在支撐 | 找前低、MA5/10/20、缺口 | K線結構 |
| 3. 下試單（小部位）| 不到平常部位的 1/3-1/2 | 資金控管替代停損 |
| 4. 觀察支撐是否真撐住 | 收盤是否守住，量縮還量爆 | 收盤確認 |
| 5. 守住 → 加碼至正常部位 | 確認後才補滿 | 老師「邊看邊加」 |
| 6. 跌破 → 小傷出場 | 試單部位小，損失可控 | 課程停損 |

## 三個案例對比

### 案例 A：8064 東捷（DB 5/21 收 120.5，5/22 漲停 132.5）

**老師背書**：5/21 L3「沒出貨跡象」

**結構問題**：
- 5/19 高點 143.5 + 上影線 4.7%（上有壓力）
- 5/20 暴跌 -9.9%，body -11.2%（大黑K）
- 5/21 收 120.5，跌破 MA5（5/20 MA5 = 125.6，5/21 MA5 = 126.6）
- 量比 5/20、5/21 持續放大（1.75）→ 不是窒息

**正確做法**：
- 5/22 漲停 132.5 不該追（已經 +10%）
- 等回檔到 120-125 區間（MA5 + 5/21 收盤附近）
- 下試單觀察是否守得住
- 守住 → 加碼，跌破 120 → 出場

### 案例 B：8027 鈦昇（DB 5/21 收 241.5，5/22 即時 254.5）

**老師背書**：5/21 L3「沒出貨跡象」

**結構問題**：
- 連續 4 根長上影線（5/15 4.3%、5/19 7.0%、5/20 6.9%、5/21 7.2%）→ **賣壓非常重**
- 5/19 高點 290 = 短期天花板
- 5/20 大跌 -9.8% body -7.7%
- 5/22 即時 254.5 仍在 MA5（253.2）附近徘徊

**正確做法**：
- 上影線連 4 根是課程明確警示（Ch2 賣壓）
- 比 8064 更危險，不建議追
- 即使試單，也要等收盤確認站回 270+ 才有意義
- **這檔我會選擇 pass**

### 案例 C：8358 金居（DB 5/20 收 460，5/22 即時 523）

**老師背書**：未在最近的 line/文章明確提過（族物面：CCL / 玻纖布）

**結構問題**：
- 5/14、5/15 連跌 -6.4%、-8.8%（兩根大黑K）
- 5/15 量比 0.15 → 窒息量出現
- 5/18 反彈 +9.9% body +13.1%，但量比 0.17 極縮（量價背離）
- 5/19、5/20 都在窒息量區（量比 0.16、0.00）橫盤
- **5/22 一根直接衝 +13.7%** → 突破

**正確做法**：
- 5/15 窒息量出現時就該注意（shakeout 訊號早期）
- 5/18-5/20 量縮整理是「老師也許還沒講，但形態自己準備好」
- 試單時機在 5/19-5/20 量縮整理區（價 428-460）
- 5/22 +13.7% 已經是 Stage 4 突破，**現在追是追在突破後**

## 🛑 進場前 30 秒提醒（每次按下買進前默念一次）

```
1. 我是「老師講了所以追」還是「個股結構準備好了」？

   → 結構準備好 → 繼續

2. 這檔離我已經進場的價格 +3% 以上嗎？
   → 是 → 停手，這就是追高
   → 否 → 繼續

3. 今天是這檔啟動的第幾天？
   → 第 1-2 根漲停內 → 可以
   → 第 3 根之後 → 風險急升，停手

4. 如果今天買完明天 -10%，我承受得了嗎？
   → 不行 → 部位減半或不買
   → 可以 → 進場

5. 市場上還有沒有其他機會？
   → 一定有 — 老師清單 + 自己觀察清單還有 N 檔
   → 沒有這檔不會死
   → 錯過一檔 vs 套住一檔 → 永遠選錯過
```

## 不要追高的三條底線

1. **同一檔不在 3 天內加碼超過一次**
   - 連續加碼 = 把均成本推高 = 自己挖坑

2. **加碼價不能高於前一筆 +3%**
   - 想加 → 等回檔到前一筆價以下
   - 不回檔 → 那就讓它走，放棄這次加碼

3. **市場永遠有下一檔**
   - 老師每週新焦點、Line 群同學每天有發現
   - 錯過一檔不會餓死，追錯一檔會傷筋動骨
   - 真的「不能錯過」的股票，**老師會講第二次第三次**
