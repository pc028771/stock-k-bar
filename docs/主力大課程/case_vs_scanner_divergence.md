## 背景

講師講課時手挑案例展示概念，但「scanner 完全按 spec 量化條件」未必跟講師案例 100% 重合。
原因：講師可能用調整後股價、不同最大量窗、概念為主而非嚴格規則、或 FinMind 資料跟富邦軟體有 ±1-2 天時差。

## 各案例詳細診斷

### 3533 嘉澤（suffocation_only，確認）

- 課程定位：僅作為「窒息量長什麼樣子」的視覺範例，沒有標進場價
- FinMind 資料：2020-12-30 volume=200,774；max_vol_20d=2,862,624；**vol_ratio=7.0%** ← 符合 < 10% 定義
- 結論：窒息量出現確認；不查出量 K 屬預期行為（此 case 的 entry_type=suffocation_only）

### 8150 南茂（full，已知落差）

- 課程標進場：35.43（2021-03-10 附近）
- FinMind 資料（2021-03-08 ~ 2021-03-12）：
  - 2021-03-08 vol_ratio=22.1%
  - 2021-03-09 vol_ratio=23.0%
  - **2021-03-10 vol_ratio=12.2%** ← 最低，但仍 > 10% 閾值
  - 2021-03-11 vol_ratio=29.9%
- 落差原因：講師軟體上看起來像窒息量，但依 FinMind 資料計算不符合 spec 的 10% 閾值；
  可能講師用的 max_vol 窗口不同（例如更長的歷史窗口讓分母更大）
- 最近 signal：2021-01-05 scenario=B（約 2 個月前）
- 結論：資料差異導致落差，非 scanner 邏輯錯誤

### 6284 佳邦（full，已知落差 — doji 阻擋）

- 課程標進場：73.9（2021-01-22 附近）
- FinMind 資料（2021-01-20 ~ 2021-01-29）：
  - 2021-01-27 vol_ratio=**9.3%** ← 符合窒息量 < 10%
  - 2021-01-28 open=76.0, close=76.0 → **doji（實體=0）**
- 落差原因：窒息量確實出現（2021-01-27），但隔日 2021-01-28 為 doji，既非紅 K 也非下影線 > 實體綠 K，因此不算出量 K
- 依 user 拍板：保持 doji 不算出量 K 的邏輯不動
- 結論：scanner 邏輯依課程 spec 正確；講師課程時 doji 是否被視為有效出量 K 屬課程未明說範疇

### 2338 光罩（full，已知落差）

- 課程標進場：~45-48（情境 B，2021-02-18 附近）
- FinMind 資料（2021-02-17 ~ 2021-02-22）：
  - 2021-02-17 volume=34,883,472；max_vol_20d=18,424,202；**vol_ratio=189%** ← 量最大值
  - 2021-02-18 volume=33,641,050；vol_ratio=96%（對新 max_vol）
  - 2021-02-22 volume=9,746,281；vol_ratio=**19.1%**（仍 > 10%）
- 落差原因：2021-02-17/18 是這段期間的最大量日，根本不是窒息量；課程標記的日期可能是在更早的窒息量後爆量段落，或講師看的是更短的量窗
- 最近 signal：2021-03-11 scenario=A（約 3 週後另一次機會）
- 結論：課程案例日期可能不精確，或講師定義的量窗與 spec 不同

### 1590 亞德客-KY（full，基準成功）

- 課程標進場：874（還原）
- FinMind 資料：2020-12-28 vol_ratio=7.1%（窒息量）；隔日出量紅 K；price > ma20（情境 A）
- Scanner 找到：2020-12-28 scenario=A，stop=871.00
- 結論：完整機械命中，確認 scanner 核心邏輯正確

## 結論

- **5 個案例中 1 個完全機械命中**（1590）、1 個窒息量出現符合（3533）、3 個不命中（8150/6284/2338）
- Scanner 邏輯正確；講師案例本身的 spec 量化邊界並非 100% 精準
- **Scanner 不應為了拉高命中率而放寬條件** — 課程定義為準

## 後續

- 真實驗證 scanner 勝率要靠 backtest（Phase 2）
- 講師案例僅作為**邏輯流程驗證**，不是「scanner 必中目標」
- 若 backtest 顯示 scanner 勝率異常低，可重新審視這些 divergence 是否暗示 spec 量化定義需要調整
