## 二、新補充的操作守則

### 資金控管替代無謂停損（引自張志超）

> 「資金控管做到極致，去替代掉無謂的停損」

**適用條件**：確定個股在「主賽道圈圈內」
- 主賽道：ABF 載板、玻璃基板、面板、記憶體、被動元件、機器人
- 方法：**縮小部位**扛住震盪，而非直接停損出清

**重要**：這是針對「你確定是好股、好族群」才適用。如果族群判斷錯誤就不適用。

### 盤中期貨操作節奏（老師自述）

1. 開盤上去打（大部位進）
2. 停一下出現黑K → 先稍微減一點
3. 盤面掛單鋪上來 → 繼續進
4. 守住紅K → 可以再加一點，往上拉打
5. 3 秒鐘內判斷：日K 確認 → 均價線確認 → 分K 確認 → 五檔掛單確認

## 六、學習心法（末段教學）

老師說明課程的目的不只是給標的，而是幫學員建立**從 0 到 1 的獨立判斷能力**：

- 第 1-2 年：單方面吸收（看、聽、記）
- 第 2-3 年：吸收內化，變成自己的東西
- 第 3-4 年：開始輸出，自己判斷後來找老師確認

> 「最後的目標是你來找我確認說我找到這個東西，判斷是這樣，你覺得呢？」
> 「比昨天強一趴，一年就強 3.6 倍了」
