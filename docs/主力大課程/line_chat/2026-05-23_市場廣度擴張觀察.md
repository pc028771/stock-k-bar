# 老師 LINE — 2026-05-23 市場廣度擴張觀察

## 原文

> 這個假日把台股市場看了一圈 用不同策略掃了大概4~500檔吧 其實有一個很明顯的結論 除了原來就在的（被動元件 ABF 記憶體 老玻璃們） 我們發現以前爛很久的東西都開始從底部反轉向上 ：原本空頭的》轉成築底整理 原本在築底的〉開始轉成多頭 比方說 特用化材 IC設計 工業電腦 網通 矽晶圓 功率元件 二極體 BMC BBU 甚至還有一些權值股型也是標準型態正在醞釀 這是市場給的訊號 最直接也最客觀 用這種方式判斷行情是我比較喜歡的方式 比每次都主觀預設指數高低來的實際 現在的問題已經不再是判斷哪一檔會漲 而是判斷哪一檔更有效率 看看明天有沒有辦法整理出來
