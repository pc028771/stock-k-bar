**補充發現（PDF vs 講稿對比）：**

### 4. 飆股形態四（布林回測策略）— 重大發現

**這是本次 PDF 萃取最重要的成果：形態四完整定義被找回。**

PDF 頁面 127（型態四，300 dpi OCR，原文照錄）：

```
型態4：布林回測策略

前置條件 => 符合布林策略（上）（即形態三：曾經跳出布林上軌）

此策略目的：為抓到強勢股的第二波攻擊
（通常市面上都是要你買在突破前高進場，但其實風險很高，因為通常過前高就會有"獲利賣壓"，
 因此就讓你買在最高點！所以！此策略是利用量急縮，在技術面和籌碼都相對安定時切入）

進場核心條件：
- 急漲後回跌至中軌（20MA = 月線）且量縮
- 最高原則 => 不能跌破中軌
- 切入後停損就是跌破中軌

量縮理想定義：
=> 達急漲段大量紅K的五分之一以下（即 ≤ 20% 的急漲量）

切入時機：
- 積極者切入點 => 為當日尾盤
- 穩健者切入點 => 確認次日低點比前一日高後尾盤切入

停利：
- 形態三出場邏輯沿用：K 棒離開上軌（收盤實體綠K 跌入上軌之內）=> 賣出
```

**與講稿 ch4-2 對比：**
- 講稿在 line 494 截斷，僅說「啊，我們就做一個」，形態四內容完全缺漏
- **PDF 補完了 100% 的形態四定義**（名稱、前置條件、量縮門檻、停損、兩種切入點）
- 量縮門檻「急漲段大量紅K 的五分之一以下（≤ 20%）」是 PDF 獨有的關鍵數字
