## §2 Exit Rules [R-EXT]

### R-EXT-001 C6 組合（production default）🔬
`tools/backtest_standard_workflow.py`、backtest +45.44%（5/1–6/4、報告 `backtest_production_workflow_*.md`）。
**「C6」是內部命名、非課程術語。**

- **Rule A — MA10 容忍 + 量縮例外**（行 242-260）：
  - close ≥ MA10 → 不出
  - -2% < close < MA10 且量縮 → 觀察
  - -2% < close < MA10 且放量 → 出
  - close < MA10 達 -2% → 出
  - 連 2 天容忍區 → 出
- **Rule C — 倉位水位 partial exit**（行 315-341）：水位 > 15% trim 到 13%、≤ 15% 全出
- **轉倉**：倉滿 + 有 P3 + 有 partial 倉 → 砍最弱換 P3

### R-EXT-002 Core 持倉停損 = 結構底 🎓
收盤跌破關鍵低點/頸線 → 停損出清。**不用 MA5**（8064 東捷教訓、老師親口）。

### R-EXT-003 出場 detector（daily）🎓+🛠️
`exit/detectors.py`
- **掀傘**（行 34-113）：連 3 不創新高 + 量縮 × 0.7 + **無連續紅 K** + 浮盈時 → 全出
- **高檔長黑 K**（行 182-309）：實體 ≥ 4% + 高檔區 + ≥ 2 項意義（缺口回補/包覆創高/吃下前 5 根）
- **分批停利里程碑**（行 323-383）：+10% / +20% / +30% 提示
- **隔日跳空急殺**（行 390-447）：≤ -5% 緊急出、-3%~-5% 警示

### R-EXT-004 殺盤日 5 分內出 ⚠️→🔬待補
跳空 + 族群整跌 = 殺盤日、09:00-09:05 立即出（6/5 ABF 案例）。**樣本少、列觀察**。

### R-EXT-005 出清前 checklist 🎓+⚠️
出清前必跑過早出清案例庫 checklist（累積 ~80-90 萬學費）。人工紀律。
