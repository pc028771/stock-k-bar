# 主體規則

## 1. 選股與族群

### 1.1 選股 universe（圈圈內才做）
- 【規則】**八大家族框架**：全市場想買的約 30 檔以內、每家族最多 2-3 檔；不脫離這圈圈。圈外標的不評論、不做。（2026-04 / 全期重申）
  - 原話：「八大家族⋯每個家族裡面大概三檔以內，最多兩檔⋯其實就那 30 檔以內」「它不在我的圈圈裡面，講完了」
- 【規則】**主賽道清單**（隨期更新）：ABF 載板 / 玻璃基板 / 面板 / 記憶體 / 被動元件 / 機器人 / 光通訊 / 連接器（5/21）；6 月聚焦光學鏡頭 / 工業電腦 / 廠務工程（6/9）。
- 【規則】**只做圈內 + 主賽道**：「這個不是我們圈圈內的股票你就不用理它」「丟出來的股票不是你每個都要全盤接受」（4/30）。

### 1.2 三符合缺一不可（進場前置）
- 【規則】**題材關鍵字 + 均線 + 籌碼三者皆具、缺一不可**（5/26）。
  - 原話：「關鍵字符合，但是你軍線不符合；你關鍵字符合軍線符合但是你籌碼不符合」→ 任一不符即不切。
  - 反例：泰鼎外資連買三天但「均線還沒準備好」→ 不切。

### 1.3 族群輪動 SOP
- 【規則】**第二棒/第三棒才是大錢**：主題啟動後去做第二、三棒、不死守第一棒。（6/2）
  - 原話：「紅海之餘為創(?)、技嘉之餘華新、群創之餘彩晶 — 如果我把同樣精神放在彩晶、省事很多」
- 【規則】**一批一批的概念**：已持倉者重複操作同一檔；空手者看「下一排」新標的、不追第一排漲多的。（5/26）
- 【規則】**同族群選對的子弟兵**：同題材挑技術/籌碼乾淨、無套牢壓力的子標（彩晶 vs 群創教訓：群創卡 50 元被洗、虧 70-80 萬學費）。（5/31）
- 【規則】**不用母子貴概念**：小朋友很強就直接買小孩。（2026-04）

### 1.4 選股偏好與過濾
- 【規則】**盤底久 + 洗刷過 + 特定分點買**（如新竹人）= 偏好標的。（5/23）
- 【規則】**量太小不主推**：族群幾百人各買 3 張就能拉漲停的標、正派經營不主推。（5/31）
- 【規則】**避險不靠買股票**：船/油/塑化避險工具老師不做、用降水位避險。（2026-04）
- 【規則】**整理未完畢 = 不切**（旺宏/文曄案例、均線還沒排整好）。（5/26）

### 1.5 籌碼量化選股門檻
- 【規則】波段籌碼（前置）：外資/投信對個股買超 ≥ 當日成交量 1/3，或買超 ≥ 兩萬張；滿足任一即可。（核心 Ch3）
  - ⚠️ 外資 = 波段、投信 = 短線；投信多買上櫃中小型、外資多買上市大型。
- 【規則】日常掃描門檻（當日方法、4/29 起慣用）：外資買超 ≥ 200 張 + 成交量 ≥ 3000 張 + 均線往上順 + 從處置中找。

## 2. 位階判斷（第 N 段 / 右上角 / 放風箏）

### 2.1 位階紅線（現行統一框架）
- 【規則】**第 1-2 段可進、第 3 段灰色（給你賣的）、第 4 段不進**。（5/19 + 6/9 印證）
  - 原話（5/19）：「右上角走了兩段⋯之後開始拉上去都是給你賣的」「走到第 3 段的基本上都是紅黑拉上去就丟掉了」「他是走到第 4 段了」
  - 原話（6/9）：「第一根第二根已經是拉很用力了、第三根就要小心」
- 【規則】**長多是最大的利空 — 第三段不可買到紅 K 上面**。（5/23）
- 【規則】**離均線太遠就「堵」（堵 = 賭）= 勝率低**。（5/19）

### 2.2 放風箏（距均線 +30%）
- 【規則】**距均線 +30% = 放風箏、離太遠不追、只能沿路操作既有部位**。（5/27）
  - 原話：「連電到這已經是放風箏了已經超過 30% 了⋯沿路去做」
- 【規則】（核心 Ch5）股價離月線（20ma）< 30% 為當沖前夜篩選硬門檻；實戰以「右上角第幾段」視覺判斷取代量化值（5/28 spec 審查升級）。

### 2.3 五日線上的都可以看
- 【規則】**沒跌破 5 日線的標都在可看範圍**（位階下緣判定）。（5/27）
  - 原話：「今天沒有破無人線（5 日線）的都是你可以看的東西」
- 【規則】**先撇掉今日大漲 + 已連三根**的標（外資買 + 大漲 = 敘事已反映、進去剛好被出貨）。（5/26）

> **演進註記（術語）：** 4/30「擊掌段 123」（長紅拉三根結束、沒切入機會）與 5/19「右上角第 N 段」描述相近位階概念、老師**未明示等號**→ 分別記錄、不合併（見〈待確認清單〉#3）。

## 3. 進場（波段 / 留倉）

> 當沖、隔日沖另見專章；本章為日 K 戰場（波段 / 留倉）。

### 3.1 波段進場 SOP（核心 Ch3）
- 【規則】**三面分析皆需滿足**：① 族群/題材 ② 籌碼 ③ 技術。
- 【規則】技術面必要：20ma 與 60ma 皆上彎；理想：當前股價離 20ma ≤ 5%。
- 【規則】進場：籌碼 + 技術面同時滿足當日**尾盤切入**；加分 = 同時封閉空方缺口。
- 【規則】「**沒有停損點就沒有進場點**」；越靠近關鍵支撐進場 → 試錯成本越低、風報比越好。（核心 Ch7）

### 3.2 尾盤才動作（時點鐵則）
- 【規則】**尾盤才進場、不在開盤進**。（6/2 再確認）
  - 原話：「不可能是在開盤這個地方就進去⋯你要動作的話、等到尾盤再動作」
- 【現象】「開盤賣、尾盤賣、中間波動不用參與」「11 點半 12 點常常是最低點」。

### 3.3 回均線 / 回踩支撐進場法
- 【規則】**回到均線守住「兩天有手」才試單**（5/23）：止跌必須「止在線上停住、兩天才算」；手越多次撲越多單。
  - 原話：「灌下來之後兩天有手，我會小試單⋯第三天還是有手，那我再撲一點單」
- 【規則】**離月線近 = 好進場**（停損明確）：「離月線很近、只要跌破就可以直接做停損」。（5/23）
- 【規則】**黑K買、紅K丟「高出低進」循環**：黑K回均線買、紅K拉一兩根丟、**不追紅K**。（5/27）
- 【規則】**外資成本區 + 回檔尾盤切入**：外資成本通常在相對高位（非最低點），策略 = 回檔當天尾盤回到外資成本區附近切入。（5/26）

### 3.4 強勢股「直接切 + 設停損」公式（通則）
- 【規則】右上角強勢股（被動元件等）**不必糾結進場時點、只要停損設好就可直接切**。（5/31，老師明確「就直接切」）
  - 停損基準：**前一根（昨天）的低點**。「你只要設好昨天的低點就可以了」「把你的交易簡單化」
- 【規則】**黑K + 融資大增 → 設前一根低點當停損切入**（嘉晶停損 133.5 示範）。（5/31）

> **演進註記：** 5/19「離均線太遠就堵」（保守過濾）vs 5/31「直接切」並非衝突 —— 前者針對「離均線非常遠／放風箏」的標的、後者針對「右上角沿均線推進但停損可控」的強勢股。兩者錨點不同（位階 vs 停損距離），分流保留。

### 3.5 收復關鍵低點留倉判定（6/9 新增細化框架）
- 【規則】**留倉三層判定（殺盤後）**，越上面越強：
  1. 🟢 收復上週五收盤價 = 可留的「好東西」
  2. 🟡 收復前一殺盤起點收盤價 = 可留
  3. 🔴 沒收復殺盤起點低點 = 都要小心
- 【規則】**收復缺口優先於收復前高**（微星 2377 例）：「收缺口 > 收前高」。
- ⚠️ 通則精神 = 「站回殺盤起點的關鍵低點/收盤價才可留倉」；**6/5、6/6 絕對日期是當週基準、不可套用到其他週**。

### 3.6 進場心法
- 【規則】**忍得越久、回得越多、風報比越好**（與收盤價比的下擋越小）。（4/30）
- 【規則】**試溫哲學**：不一次梭哈、最小單位（3-5 張）插眼觀察、確認後尾盤才壓大部位（變 100-400 張）。（5/23）
- 【規則】**等 K 棒收才下判斷**：不看盤中價、看 K 棒收（核心收盤確認原則）。（4/30，注意此為日 K / 波段原則，當沖判斷亦同此）

## 4. 加碼

### 4.1 加碼通則（全面性規則）
- 【規則】**加碼必先脫離成本 ≥ +10% + 回測支撐才加**。老師 5/29 親口確認**全面性規則**（不限特定股）。
  - 原話（5/25）：「沒脫離成本你加碼是無意義的」「有漲然後下次回到支撐的時候」
  - 原話（5/29）：「全面性規則、沒漲 10% 加碼沒意義」
  - ⚠️ 加碼 +10% 門檻 ≠ 進場 +10% 門檻、不可反向套用。
- 【規則】**回均線黑K才加、不追紅K**（黑K買紅K丟循環的加碼側）。（5/27）

### 4.2 加碼前置（短線/盤中）
- 【規則】**5MA + 10MA 排列清楚（不交叉糾結）+ 連續兩根站穩 5MA → 才加**；未滿足前只做試溫部位。（5/19）
  - 原話：「軍事還是很混亂我沒辦法加碼⋯站上無日線兩根了」

### 4.3 加碼錨點（核心課程）
- 【規則】波段加碼：走出波段後拉回月線（仍在 5% 內）→ 可再加一筆。（核心 Ch3）
- 加碼時機（量價）：股價回測大量高點、下影線在大量高點附近守住 → 可加碼。（核心 Ch2）

## 5. 停損

> 核心紀律：**所有跌破與站回均以收盤價為準、不以盤中價判斷**（核心 Ch2）。例外：當沖分 K 戰場 + 處置股「收高開低盤前出」（見各專章）。

### 5.1 結構性停損（一般 / core 持倉）
- 【規則】關鍵支撐 4 種：① 大量低點 ② 慣性均線 ③ 缺口支撐 ④ 轉強K 棒低點。收盤跌破即出。（核心 Ch7）
- 【規則】**跌破結構支撐 / 頸線 → 停損出清**；大黑K 完整包覆前段漲幅 = 停損訊號；低點越來越高的規律消失 = 停損；跳空缺口當天回補失敗 = 出場。
- 【規則】**有破（低點/支撐）就先出、回 10 日線/月線附近再重新看**。（5/27）

### 5.2 均線停損（多為 10 日線）
- 【規則】跌破關鍵均線停損（順德/成熟製程標：跌破 10 日線即停損，「風報比非常高」）。（5/26）
- 【規則】老師逐檔報停損價（5/31）：嘉晶守 133.5 / 兆赫守月季線 ~70 / 啟碁守 5 日線 / 金豪科守 238-236 → 各標停損設在守住的關鍵均線/前低。

### 5.3 漲停隔日停損（守昨收）
- 【規則】**漲停隔日進場、停損 = 昨日（漲停日）收盤價**，收盤跌破則「往下瀑布」（殺盤）。（5/31，連說三次「守住」）
- 【規則】「收高開低 = 轉弱」（前日出量實紅收最高、次日開平/綠 → 不硬作多、已持有開盤離場）。（核心 Ch7）

### 5.4 非漲停 / 黑K進場停損（守昨低）
- 【規則】**非漲停 / 黑K進場 → 守昨日低點**（與漲停日守昨收分流，5/31 同日兩種基準各有情境）。

> **演進註記（裁決）：** 5/31 同日出現兩種停損基準 —— 漲停隔日守「昨收」、一般/黑K進場守「昨低」。**非矛盾、依情境分流**（見〈規則演進史〉X-5 裁決）。

### 5.5 資金控管取代「無謂停損」（有前提）
- 【規則】**「資金控管做到極致、取代無謂的停損」**——但有嚴格前提：必須「確定是好東西 + 在圈圈內 + 主賽道 + 最小倉位」。（5/21 / 5/23）
  - 原話：「先用最小的單位進去⋯下來你也不怕」
  - **邊界**：「成本太高又買很多 → 絕對扛不住 → 丟」；只有「成本高但買很少」「最小倉位的好東西」可放著震盪不停損。
- ⚠️ **不覆蓋結構底停損鐵則** —— 此規則限「最小倉位 + 圈內好東西」場景；大倉位 / core 持倉仍守結構底停損（見〈規則演進史〉裁決 batch3-4）。

## 6. 停利與出場

### 6.1 波段先停利（分批減碼）
- 【規則】**出量往上 → 減碼一點點、底倉留、回均線再切入**（券商概念股出量要比被動元件更警覺）。（5/27）
- 【規則】**紅K拉一兩根 → 丟丟丟**（高位減碼、不追紅K）。（5/27）
- 【規則】長紅日節奏：往下殺去結好的股票；往上一根長紅賣一半、隔天攻不上去再賣一半。（2026-04）

### 6.2 先停利的反例（core 不亂停利）
- 【規則・反例】**帳面利潤已 30%+ 不必為短線回檔先停利**（一根漲停+一根跌停約 10% 都傷不到成本）。（5/26）
  - 原話：「短線思維 ≠ 中長線崩」
- 【規則・反例】低成本 / 均線附近 / 破線承接到的好股票不要亂丟（會買不回來）、前提是圈圈內。（2026-04）
- ⚠️ **先停利只用於**「短線確認失敗 / 出量見高 / 紅K上影」；core 倉（成本 30%+ / 真心相信的好東西）反對亂停利。

### 6.3 出貨/翻空訊號（出場）
- 【規則】**收高開低 = 態度翻空**：今收漲停隔天開低 → 盤前試撮即跌 → 開盤第一盤跌停價出（處置股 / 漲停隔日專屬、盤前判斷例外）。（2025-11）
- 【規則】漲停隔天開盤價低於前漲停價 = 主力聞風先散、危險。（2025-11）
- 【現象】**兩黑K吞噬一紅K → 隔天可能再出低點**。（6/10）
- 【規則】大紅K 被隔日大綠K 完全吞噬 = 多方被空方吃掉 → 出場。（核心 Ch2）
- 【現象】**漲一根後面就是休息**（每次漲一根 → 休息、漲一根要小心）。（5/26）

### 6.4 三種常見錯誤路徑（自我檢討框架）
- 【規則】錯誤只有三種：① 賣飛（賣太早）② 停對了沒買回 ③ 沒停損一路下去。賣掉的股票要放備用清單、它會給你機會。（5/23）

## 7. 倉位 sizing 與資金梯度

### 7.1 資金配置原則（核心）
- 【規則】依交易屬性調配、沒有標準答案；決定後嚴格遵循不臨機改變。（核心 Ch7）
  - 範例：上班族 = 波段 60% / 短線 20% / 隔日沖 20%；急性子 = 短線 50% / 隔日沖 30% / 波段 20%。

### 7.2 反向操作水位
- 【規則】**回跌往上加碼（水位↑）、過熱長紅減碼（水位↓）**。（2026-04）
  - 原話：「我們只要有回跌就會往上加⋯過熱⋯先減一下」

### 7.3 三分之一滾動法（老師親口、非 AI 發明）
- 【規則】**每漲約 +35% 賣 1/3、進二次處置時用處置策略把那 1/3 買回、底倉 2/3 不動續抱**（可到 +100~130%）。（2026-04）
  - 簡化版：怕累的人第一次 +35-40% 賣 1/3 後就守均線續抱、不再動。
  - ⚠️ 此 1/3 是**老師原話來源**、非 AI 發明的減碼比例 → CLAUDE.md「禁自行發明減碼比例」不適用（見裁決 batch2-C4）。

### 7.4 資金梯度分批（最新框架）
- 【規則 / 戰術】**資金梯度分批、保留現金、依大盤位置加碼**（6/7 通則精神）：
  - 無槓桿/無融資都用現股、風險至上、設好停損。
  - 不確定盤勢時尾盤分 1/3 進場、分三天 run；殺盤日尾盤是進 1/3 時機；扛週末波動用分批稀釋。（5/28 / 5/31）
  - 6/7 當週示範：六成資金分五等份梯度、留一成現金、依大盤位置（41000 / 39000）分批投入。
- ⚠️ **「分幾份」依當下盤勢調整、非固定值**：5/28-5/31 用「分 3 份 / 3 天」、6/7 用「六成分 5 份」。**6/7「五份 = 資金 5 個等份梯度、不是 5 個持倉檔位（不是 5 個倉位）」**（見裁決 X-2）。

### 7.5 sizing 心法
- 【規則】**持股水位要足夠低、讓你能冷靜淡定**（倉位過重就無法客觀看市場）。（6/10）
- 【規則】**好東西最小倉位插眼**：「放個小倉位讓他去震盪、等真的打擊區再咬進去、就會耐得住扛得住波動」（打擊區哲學）。（5/23）
- 【規則】**成本高又買很多 = 扛不住 → 必丟；成本高但買很少 = 可扛**。（5/23）
- 【規則】**特定易洗股性（功率元件等）= 小倉位、耐洗程度要高**。（5/19 / 5/23）

## 9. 當沖專章（Ch5 — 分 K 戰場）

> 【規則】**適用範圍宣告**：Ch5 所有規則只在分 K 戰場（盤中當沖）成立；日 K 戰場回到 Ch2/Ch3/Ch4。老師不鼓勵新手做當沖。
> 術語譯註：軍線 = 均線；守住 = 手助/手築。

### 9.1 當沖前夜篩選
- 【規則】均線 5ma > 10ma > 20ma（多頭排列）且三條皆上彎。
- 【規則】成交量兩天皆 > 兩萬張；三天區間振幅 > 8%；三天周轉率 > 20%；股價離月線 < 30%。
- 【規則】視覺優先過濾（5/19）：右上角已連漲 3 段以上 → 開高紅K都是給你賣的、不切入；離均線太遠 → 直接過濾。

### 9.2 當沖進出場 SOP（Ch5-3）
- 盤中三大判斷工具：① 均價線 ② 5 分 K 高點 + 5 分 K 的 5ma ③ 江波圖。（實戰以昨日收盤價 + 5 分 K 5MA 為主、均價線為輔）
- 【規則】股價必須在均價線之上；突破第一根 5 分 K 高點 = **訊號觸發**（非切入點）。
- 【規則】**前 10 分鐘（09:00-09:10）不切入**、只觀察；9:10 後等回踩 10MA 守住才切。
- 【規則】**前 5 分鐘漲幅 > 5% → 整檔放棄**（一票否決）。
- 【規則】均線（5/10/20ma）發散 = 硬性過濾；第一根 5 分 K 為實體紅K、第二根不跌破第一根。

### 9.3 當沖停損（雙錨點）
- 【規則】**雙錨停損**：主錨 = 第一根 5 分 K 低點；副錨 = 昨日收盤價（反覆強調 ≥ 15 次）；跌破任一即停損。
  - 原話：「跌破昨天的收盤價，所以收盤價超級重要」「死守收盤價」
- 【規則】跌破 5 分 K 的 5ma → 轉弱出場；黑K一路到 10MA/月線 = 今天沒戲、只能做一小段。
- 【規則】**下擋小、上擋大才切**（好邁案例：下擋 0.5、上擋無限、抓 2-3% = 當沖高標）。

### 9.4 當沖達標 / 停利
- 【規則】**當沖達標 1.5%-3% / 每段**（3% 為滿分標準、2-3% 就走、後面漲停不管）。
- 【規則】**量化停利**：2 分 K 拉 ≥3% / 5 分 K 拉 ≥5% = 紅K達標立即出（不等下一根）。
- 【規則】大紅K出量 → 先賣一半、觀察是否快速灌下；**例外：快進漲停則不賣/留倉**。
- 【規則】長上影線 → 全數離開；兩三根 K 不再創高（沒人拉）→ 主動「掀傘」出場、不等停損。

### 9.5 K 棒型態過濾（當沖）
- 【規則】上影線 > 實體 → 不做；下影線 > 實體（掉人線/吊人線）→ 不做（「掉人線很可怕」≥ 6 次）。
- 【規則】紅吞噬後下一根必回踩（「會累」）、不追、等回踩前波支撐再進（不買最低、買 +0.5% 確認位）。
- 【規則】5 分 K 連 2-3 根下影過長 → 即使整體上漲也出場；同時長上影+長下影 = 區間盤整、畫盒等突破再進。

### 9.6 江波圖 / 雙向循環
- 【規則】上漲時一波須比前波高、否則轉弱減碼；下跌時一波沒比前波低 = 止跌。
- 【規則】賣出後若價站回前波高（5MA 收復）→ 可再買回做下一段（同日盤中，強茂案例）、不算破紀律。

### 9.7 當沖留倉思維
- 當沖選到若跳空漲停且解鎖次數少（1-2 次）→ 不賣、轉短線/波段留倉；開開關關 3-4 次 → 考慮減碼。

> **mode 分流提醒：** 當沖 +3% 達標 vs 波段抱長（trend rider）屬不同 strategy_mode、不衝突（見裁決 batch3-5）。

## 10. 隔日沖專章（Ch6）

### 10.1 隔日沖心法
- 【規則】**今天尾盤買、隔天開盤一定要賣**（不管賺賠）；不是每天都能做；目標每日 1-2% 達標。
- 【規則】進場前判斷大盤（最關鍵）：加權 + OTC 兩者皆須當日量增紅K、站上 5ma；尤其 OTC 必須在 5ma 之上。
- 【規則】進場時機：1 點開始選、1:20-1:25 進場（越早勝率越低）；出場：8:30-9:00 預掛跌停賣。
- 進階出場：跌破第一根 5 分 K 低點 → 出場（國際利空時）；2 張 → 1 張開盤賣、1 張用 5 分 K 低點觀察續強。

### 10.2 隔日沖長相分辨（4/30 系統教學）
- 【規則】**A 型（可做）**：昨漲停、今**沒**跳空高開、踩一下守均線慢慢墊上（羽絨/瑞軒）。
- 【規則】**B 型（不做）**：昨漲停、今跳空高開、馬上快速衝高後灌下 = 倒貨（惠特/油田）。判斷核心 = 是否跳空高開 + 開盤後是否快速灌下。
- 【規則】昨漲停今開低 = 沒戲、整天超弱（中華化）；均線向下散 → 轉可做空。

### 10.3 尾盤速篩（Ch6-2 HD vision 精確值）
- 【規則】速篩：布林窄率 < 6% + K 棒突破布林高軌；長紅 > 3.5% + 量 > 1,000 張 + 量增 > 前日；月線斜率 > 0.4；距上軌 < 6%。
- 【規則】進場時機：1 點 20 確認進場（過早會套牢）。

## 11. 處置股專章（2025-11 ~ 2026-06 核心策略）

> 處置股是「政府認證的標股」、約一半市場人默默在做的超額報酬策略；出關當天要高度專注。**自成一套錨點體系（處置前高 / 大量收盤價、非結構底）。**

### 11.1 套用大前提
- 【規則】**OTC（櫃買）指數必須在所有均線之上、且均線上揚**，此策略才成立（進處置的多是櫃買中小型股）。
- 【現象】均線上揚時報酬更好（最香那段）。

### 11.2 標的篩選（標準形態）
- 【規則】**底底高**（低點墊高）；**第一次進處置**最送分（衍生可接受第二次）。
- 【規則】左邊最好「沒人玩過」/ 乾淨（沒大量、沒大漲過、盤底約兩年）；左邊還在土石流的剔除。
- 【規則】進處置前後三天**不出大紅大黑、要含蓄守在進處置前高點上緣**。
- 【規則】處置期間主力要的是**越強越好、不洗盤、均線墊高**；出現開高下影線（洗盤）= 剔除。

### 11.3 進處置前大量錨點
- 【規則】進處置後**前三天收盤價要守在「進處置前大量收盤價」之上**；守住才試單。沒守住、黑黑灌到 10 日線 → 破壞形態、不理它。
- 【現象】約四五成個股進處置前會出現一根大量、記住那根的收盤價當基準。

### 11.4 進場時點（處置中切入）
- 【現象】進處置後前三天是風向球。
- 【規則】**第三、四天**可開始布局（風險高）；**第五、六天（部分第六、七天）最安全** —— 哪條均線離得最近？**10 日線最接近**、風報比最高（經回測）。
- 【規則】**進關後第 4-5 天 + 出關前 2 天 + 第 45 天**為關鍵觀察點（5/31 / 6/3 重申）。
- 【規則】切入是「慢慢試」、第三四天試水、第五六天加大；前面三四檔可能都停損也沒關係（10 檔抓 3 檔就打平）。
- 【規則】處置前/當天爆量低點被跌破 → 處置期間先忍耐不進、等收復約 +35% 區再看。

### 11.5 處置股停損
- 【規則】**跌破 10 日線 → 停損撤退**（守一個 10 日線）。單筆停損約 -10%~-20%。
- 【規則】大黑K / 大灌破平台（跌破前大量收盤價）→ 平台失敗 → 停損。

### 11.6 三種發展型態（依「處置前最高點」錨點）
> 老師沒明確編號、用三類對比案例呈現（解讀檔歸納為 A/B/C）。
- 【現象】錨點慣性：進處置前一天通常是最高點。
- 【規則】**型態 A（主升續攻）**：第一次衝上來漲 35-40% 就進處置 = 最佳時點；處置期間**沒過**處置前高 → 出關有機會過高 → 續攻。
- 【規則】**型態 B（反彈段）**：處置前高回落 25%-30%（過深）→ 只走技術反彈、賺 10%-20% / +15% 達標就閃。進場：處置最後一天尾盤拉上去 = 進貨訊號。
- 【規則】**型態 C（出貨/不能碰）**：處置期間**已過**處置前高（出貨完成）→ 出關很慘；或均線往下排；或已第三次以上進處置。操作：出關前最後一天賣一半、出關當天開低全出、不進場。
- 【規則】處置中漲幅天花板：**處置期間最多漲 20%**（漲跌幅 10% 兩天）；前高設太高、跌下來反彈 20% 也只是回補、無超額報酬。

### 11.7 出關當天
- 【規則】**出關當天氣勢必明確：不是大漲就是大跌、不會小漲小跌**；要高度專注。
- 【規則】**出關當天開在月線（均線）附近 → 容易漲、開盤就過高 → 可小追**。
- 【規則】出關前兩天就先攻過了 → 出關當天可能不攻、要小心（達標先賣）。
- 【現象】出關前一天/最後一天常有人「忍不住」先動 → 加大力道上車的窗口。

### 11.8 二次處置 / 反例
- 【規則】第一次處置期間又宣布第二次處置 → 下車（開盤下車）。

- 【規則】盤中掃個股：碰 10 日線有守、開始打底 → 10 日線地方接一下、尾盤再接一下；跌破 10 日線 = 停損。

## 12. 做空專章

- 【規則】**做空 = 做多形態的鏡像**；不是隨時能做空、須盤中確認空方控盤才啟動。
