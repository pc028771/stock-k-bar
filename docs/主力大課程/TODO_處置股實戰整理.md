### 大前提（前置條件）
- 櫃買 OTC 指數要在所有均線之上 + 均線上揚
- 不符合則策略失效

### 處置股進場 SOP
1. 進處置前的**大量收盤價** = 關鍵起算點
2. 進處置後守住大量收盤價 **3-4 天**
3. **5-6 天時 MA10 最接近** → 風險最低、最舒服進場
4. 守不住 → 黑K灌下十日線 → 不要理
5. 守住 → 試單 → 後面 5-6 天慢慢加碼
