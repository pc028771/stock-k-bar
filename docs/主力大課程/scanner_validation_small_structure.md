## 一、概念定位

依「打擊區哲學」（老師 5/22 公頻）：

```
打擊區 = 老師訊號 × 個股結構準備好

老師訊號 (人判讀) ─┐
                   ├──→ 打擊區（大口吃肉）
個股結構 (機掃) ───┤
   ├── shakeout_strong  → 底部窒息突破
   └── small_structure  → 高位整理末端突破
```

兩個結構 scanner 互補，不衝突：

| 維度 | shakeout_strong | small_structure |
|---|---|---|
| 位置 | N 字底部 | N 字中段 |
| 整理特徵 | 量極縮（窒息）| 高位橫盤 + 量縮趨勢 |
| 突破前 | 連續 5+ 天窒息 | 5-7 天高位橫盤 |
| 案例 | 6207 雷科 5/15 | 群創 5/20、晶技 5/19、力積電 5/13 |

## 四、Scanner 條件（current）

`scripts/zhuli/entry/small_structure.py`

```python
1. 前段攻擊：過去 10-20 天有 +10% 漲幅
2. 高位整理：近 5 天 close range < 10%
3. 量縮趨勢：近 3 天均量比 < 1.5 且當日 < 前 3 天均
4. MA5 追上：MA5/收盤 > 0.93
5. 未跌破 MA10：close ≥ MA10 × 0.95
6. 高位維持：close / 近 10 天高 ≥ 0.85
```

**門檻不需要再調整**（晶技 5/19 已觸發，3/3 案例全中）。

## 五、Production 工作流（高優先權）

### 每日盤後（13:30+）

1. 跑 small_structure scanner 全市場
2. 跑 shakeout_strong scanner 全市場
3. 兩個 scanner 命中清單 ∪ 取聯集
4. 對照老師當日 + 凌晨文章清單 → 找**雙重命中**
5. 雙重命中 = 打擊區候選

### 明早 8:30 試撮

對打擊區候選做：
- 試撮 +5% 內 → 開盤前掛買試單 1/3
- 試撮 +5% 以上 → 等開盤後 5-10 分鐘評估
- 試撮鎖死 → 等回測（隔天回測 5% 內可進）

### 收盤前 13:25

對打擊區但開盤沒進到的：
- 收盤前結構優異（量縮 + 紅K + 上影 < 2%）→ 試單 1/3

## 七、注意事項

- Scanner 是「機器找結構準備好」，**不等於該進場**
- 必須配合老師訊號（打擊區雙重確認）
- 即使雙重命中，依保守策略仍只試單 1/3
- 大進場（正常部位）只在「老師首次明確主推 + 起漲日 + scanner 觸發」三條件成立時
