## 第一部分：課程完整規則

### 1. 投資三大原則（Ch1）

- 勝率組成（主力大自定義拆解）：
  - 選股 50% · 進出場規劃 30% · 運氣 20%
- 課程從頭到尾貫穿三大關鍵：**選股 / 進出場 / 心態觀念**。
- 「選股不對，操作白費」「活下來比什麼都重要」。
- 不使用 MACD / KD / RSI（互相抵觸無法決斷，違反「大道至簡」原則）。

### 2. 選股原則

- **基本面：** 題材（漲價、缺貨、轉單）+ 族群（面板、航運、鋼鐵、記憶體、被動元件…）。
- **籌碼面：** 外資（波段）+ 投信（短線）+ 主力分點（內資、地緣分點）。
- **法人選股法：** 跟著外資/投信買賣超清單篩，因為法人對基本面/題材的研究深度遠超散戶。

### 3. K 棒解讀（Ch2-1）

- 紅K = 收 > 開；綠K = 收 < 開。
- 上影線 = 賣壓；下影線 = 買盤支撐。
- 紅K 永遠優於綠K（當日 > 50% 進場人賺錢 vs 套牢）。
- 突破密集上影線高點 → 賣壓消化完畢，方有進場理由。
- **長上影線頂端連線 = 水平阻力帶**（多根長上影線的頂部 = 賣壓區，可組成矩形阻力）。
  - 案例：4961 天鈺 39.75 / 37.95（vision 比對 ch2-1 16:30 補充）。
  - Scanner：收盤站上「該水平阻力」= 突破訊號。
- **大紅K 被隔日大綠K 完全吞噬 = 多方主力被空方吃掉 → 出場訊號**（ch2-1 vision 補充）。

### 4. 均線實務（Ch2-2）

- 均線 = X 天平均買進成本；用收盤價計算。
- 天期決定交易屬性：
  - 5/10/20/60 → 3-5 天 ~ 3-6 個月
  - 20/60/120/240 → 半年以上的大波段
- **多頭排列（必要 / 理想 / 加分）：**
  - **必要：** 5ma > 10ma > 20ma（價格排序）— 老師 ch2-2 06:04/07:26 藍筆現場更正，60 不是必要（投影片寫錯）。（✅ HD audit Ch2-2 2026-05-19 修正：480p 誤判為紅筆，HD 確認全部為藍色）
  - **理想：** 加上 60ma 形成 5>10>20>60（最高勝率條件，用於窒息量等策略升級）。
  - **長天期版本：** 20 > 60 > 120 > 240（大波段），四條都需上彎。
  - 加分：4 條皆上彎、尚未發散。
- **黃金交叉週期對應**（ch2-2 vision 補充）：
  - 5/20 黃金交叉 → 短期持有 3-5 天
  - 20/60 黃金交叉 → 中期 1-3 個月
  - 60/120 黃金交叉 → 長期 3-6 個月
- **慣性支撐：** 股價多次回測同一均線都不跌破 → 該均線是該股的「慣性買盤」。
  - 收盤站回隔日 → 確認支撐 → 可在均線位置進場；收盤跌破即出場。
  - **可放大到週線**做更長交易週期判斷（ch2-2 vision 補充）。
- 均線上彎 = 支撐；均線下彎 = 壓力（基本原則）。

### 5. 缺口實務（Ch2-3）

- 缺口 = 兩根 K 棒中間真空地帶（不含上下影線）。
- 多方缺口 → 支撐；空方缺口 → 壓力。
- **多方缺口進場：** 量縮回測缺口附近、未跌破**缺口低點** → 切入；收盤跌破缺口低點 → 出場。
  - ⚠️ ch2-3 投影片原寫「缺口高點」是錯字，老師 02:05 手寫「應為缺口低點 更正」（vision 比對補充）。
- **空方缺口封閉的最佳訊號：** **轉強紅K 的低點 ≥ 空方缺口高點**（紅K 完全在缺口高點之上，不只是 close 站上）。
- **多缺口連環處理：** 連續封閉每一個缺口 → 每次封閉都是一次轉強訊號（ch2-3 18:48 完整 SOP 示範）。
- **空方缺口進場（回補轉強）：** 收盤站上缺口高點（封閉缺口）→ 進場；停損 = 封閉紅K 低點。

### 6. 量價判讀（Ch2-4）

- 「量是因，價是果」。
- 健康：價漲量增 / 價跌量縮。
- 不健康：價跌量增（大量黑K）= 籌碼凌亂。
- **量縮定義：** 成交量 ≤ 前一根 K 棒的 1/2。
- 大量 K 棒高點為「第一支撐」、轉強K 低點為「第二支撐」。
- 大量黑K 出現後，跌破其低點即出場；要重新進場必須收盤站上其高點。

### 7. 大波段選股 SOP（Ch3-1 + Ch3-2）

- **三面分析：** 族群/題材 · 籌碼 · 技術。
- **籌碼門檻：** 法人對個股買超 ≥ 當日成交量 1/3，或 ≥ 兩萬張。
- **技術門檻：** 20ma 與 60ma 皆上彎（必要）；當前股價離 20ma ≤ 5%（理想）。
- **停損：** 跌破月線（20ma）→ 離場，或用大量紅K 低點防守。
- **加碼：** 走出波段後拉回月線（仍在 5% 內）→ 可再加。
- **籌碼確認後延遲進場：** 籌碼大買當日 ≠ 隔天馬上進場，須等技術面確認。（⏳ Ch3-2 字幕推導 86–99s，待畫面驗證）
- **日常掃盤優先級：** ① 族群性（同日大量買進同一族群）→ ② 個股 ≥ 2 萬張 → ③ 外資買超 ≥ 1/3 成交量；三條件皆符合勝率最高。（⏳ Ch3-2 字幕推導 640–675s，待畫面驗證）
- **外資族群持續性預期：** 外資大買族群當日 → 後續 2 週～1 個月持續看好該族群。（⏳ Ch3-1 字幕推導 396–405s，待畫面驗證）

### 8. 四大短線起漲形態（Ch4-2）

#### 形態一 — 奇形（旗形 + 旗杆）
- 第一根紅K（旗杆）→ 第二、三根（旗子）紅綠不拘
  - 紅K → 量增
  - 綠K → 量縮
- 收盤條件：兩根收盤在 5ma 之上；最強勢為兩根收盤在旗杆收盤之上。
- 切入：第三天尾盤（積極）或第四天打到 5ma（穩健）。
- 停損：跌破 5 日線。

#### 形態二 — 反轉形態（突破下降趨勢線 + 兩高點連線）
- 5/10/20 均線皆在反轉紅K 之下，不可切到 K 棒。
- 紅K 整根站上所有均線。
- 紅K 底部站上下降趨勢線（兩高點連線）。
- 切入：反轉紅K 下方 1/3 處（隔日操作）。
- 停損：收盤跌破反轉紅K 低點。

#### 形態三 — 布林上軌進出策略
- 通道窄度 < 10%（壓縮通道，主力大定義：(上軌−下軌)/中線 < 10%）。
  - ⚠️ 此 10% 是「短線飆股起漲」情境的理想門檻（「小於 10% 不代表大於就不飆」）。
  - 隔日沖精選設定採更嚴格的 **6%**（HD vision Ch6-2 00:44）。
- **窄度確認時間點：** 看**突破前一天**的帶寬，不是突破當天（HD vision Ch4-2 34:47）。
- 起漲 K：收 > 上軌、必須出量。
- 一買點：突破當天尾盤；二買點：次日上軌附近（無法看盤者用上軌 × 0.99）。
- 出場：收盤實體綠K 跌入上軌之內。
- 雙面刃：股價必須突破上軌；否則上軌變壓力。

#### 形態四 — 布林回測策略（從 PDF 補完）
- **目的：** 抓強勢股第二波攻擊，避開突破前高的獲利賣壓。
- **前置：** 必須先符合形態三（曾跳出布林上軌）。
- **進場：** 急漲後回跌至中軌（20MA）+ 量縮 ≤ 急漲段大量紅K 的 **1/5（20%）以下**。
- **切入：** 積極者當日尾盤；穩健者次日低點比前一日高後尾盤。
- **停損：** 跌破中軌（20MA）。
- **停利：** 沿用形態三 — K 棒離開上軌（實體綠K 跌入上軌之內）。
- **來源：** PDF `1627272942273-主力大全方位操盤教戰守則.pdf` p.127。

### 9. 當沖前夜篩選參數（Ch5-1-1）

- 均線：5/10/20 多頭排列且上彎
- 兩天皆 > 2 萬張
- 三天區間振幅 > 8%
- 三天周轉率 > 20%
- 股價離月線 < 30%

### 10. 當沖兩大精選關鍵（Ch5-2）

- 距離前高 < 10%（準突破）
- 近期量 > 前波高點量

### 11. 當沖盤中 SOP（Ch5-3）

- 進場前：股價 > 均價線 + 突破第一根 5 分 K 高點 → 模擬切入（**雙層確認**，缺一不可）。（🖼 Ch5-3 截圖手寫補充）
- 停損：第一根 5 分 K 低點，或跌破 5 分 K 的 5ma。
- 江波圖：上漲一波 < 前波 → 轉弱；下跌一波 ≥ 前波 → 止跌。
- 開盤 4% 以內、第一根 5 分 K 為實體紅K → 理想條件。
- 達標：1.5%-2.5% / 段。
- **均線型態辨別：** 糾結上升（緊密同步上彎）→ 看多；發散下降（空頭排列、間距擴大）→ 迴避。（🖼 Ch5-3 截圖手寫補充）

### 12. 隔日沖規則（Ch6-1 + Ch6-2）

- 紀律：今天尾盤買、隔天開盤一定要賣（不管賺賠）。
- **進場前判斷大盤：** 加權指數 + OTC 指數兩者皆須符合：
  - 量增紅K（投影片原文「至少要是紅K」，老師手寫補充「加量」，完整條件為量增紅K）
  - 收盤在 5ma 之上（HD vision Ch6-1 04:39 + 05:18）
- 進場時機：1 點開始選、1:20-1:25 進場。
- 速篩三條件（HD vision Ch6-2 00:44 精確數值）：
  1. 布林窄率 < **6%** 且 K 棒突破布林高軌（還原日K）
  2. 長紅棒 > **3.5%**，量 > **1,000 張**，量增 > 前日 **0.1%**
  3. 月線斜率 > **0.4**（老師當場更正：原說「乖離率」，正確是「月線斜率」）
- 篩後人工判斷：整根 K 棒在布林上軌之上（非只收盤站上）為最佳；被上軌切到 = 不理想。
- 出場：盤前預掛跌停賣（以開盤價成交）；或用第一根 5 分 K 低點防守續強。
- 進階：兩張 → 一張開盤賣、一張續留觀察。
- 個股案例：2351 順德 2021/6/22（進場 117，隔天 125，+8 元；HD vision Ch6-2 10:06）。

### 13. 停損規則（Ch7-1）

- 「沒有停損點就沒有進場點」。
- 課程教過的關鍵支撐：
  1. 大量低點
  2. 慣性均線
  3. 缺口支撐
  4. 轉強K 棒低點
- 越靠近關鍵支撐進場 → 風報比越好。
- **進場距離控管：** 進場前需計算股價到停損點的 % 距離，確認在可接受風險範圍內再進場。（⏳ Ch7-1 字幕推導，待畫面驗證）
- **釋單機制：** 拉出紅K 隔一天不追；靠近支撐才開始慢慢釋單（課程進階應用）。（⏳ Ch7-1 字幕推導，待畫面驗證）

### 14. 資金配置（Ch7-2）

- 沒有標準答案，依交易屬性調配。
- 範例：
  - 急性子可盯盤 → 短線 50% / 隔日當沖 30% / 波段 20%
  - 上班族 → 波段 60% / 短線 20% / 隔日沖 20%
- **機械執行原則：** 一旦決定資金配置，須嚴格遵循，不可臨機改變比重。（⏳ Ch7-2 字幕推導，待畫面驗證）

### 15. 主力意圖（Ch7-3）

- **收高開低 = 轉弱：** 前日出量實紅收最高、次日開平/綠 → 出場。
- **收低開高 = 轉強：** 前日出量實黑收最低、次日開平/紅 → 進場。
- **開平盤的恐怖：** 漲停的標的隔天開平盤也是危險訊號。
- 例外：國際性利空導致普遍開低（大盤普跌脈絡）→ 不適用收高開低出場邏輯。（🖼 Ch7-3 截圖補強）
- **出量強勢K = 主力推進訊號：** 亮紅K 收最高、或漲停收盤 = 主力正在積極推進行情，非賣出時機。（⏳ Ch7-3 字幕推導，待畫面驗證）
- **大量黑K 跌停陷阱：** 出量黑K 跌停 ≠ 正常回調，是警訊，須立即評估停損。（⏳ Ch7-3 字幕推導，待畫面驗證）
- **跌幅 30-40% 心理臨界：** 個股自高點跌幅達 30-40% 時散戶心理崩潰，主力常藉此清洗浮額；此階段若出現籌碼進駐需三面確認才介入。（🖼 Ch7-3 截圖手寫補充）

### 16. 窒息量加碼（Ex1-1 ~ Ex1-3）

- 窒息量定義：當日量 < 20 日內最大量的 10%。
- 必要：月線上彎。
- 出量 K 限定：紅K，或下影 > 實體的綠K；不可實體黑K/實體綠K。
- 停損：出量 K 的低點。

### 17. 投信跟單加碼（Ex2-1 ~ Ex2-3）

- 投信習性：中小型、短線（數週至 1 季）、有作帳壓力（6 月底、12 月底最明顯）。
- 警戒線：投信持股 > 12% → 隨時可能結帳。
- **投信策略 SOP（Ex2-2）：** 投信買超 5 天累計 ≥ 1.5% 股本 + 剛上榜 + 技術面排列 + 十日線進出。
- **投信首買（Ex2-3）：** 首日 ≥ 200 張、前 2-3 個月無投信買超、價籌背離（黑K + 投信買）。

## 第二部分：🔴 不可違背的核心主軸

### 進場側

- 🔴 **進場前必須先確認停損點**。沒有停損點就沒有進場點（Ch7-1）。
- 🔴 **越靠近關鍵支撐進場越好**（風報比越好）。
- 🔴 **不在訊號出現的「事件當天」進場，等下一根 K 棒確認**：
  - 窒息量出現當天不進場，等出量 K（Ex1-2、Ex1-3）。
  - 反轉紅K 出現當天不追，等下一根回到下方 1/3（Ch4-2 形態二）。
  - 投信首日上榜不進場（除非積極派可用 5 分 K SOP；穩健派等回十日線）。
- 🔴 **進場必須伴隨「轉強訊號」確認**，不可憑「看好」進場。
  - 轉強訊號 = 收盤站上缺口高點、站上大量 K 棒高點、突破前波高點、突破第一根 5 分 K 高點、站回慣性支撐均線。

### 出場側

- 🔴 **所有跌破與站回皆以「收盤價」為準**，不以盤中價判斷（Ch2-2 line 607-608 明示「收盤跌破」、Ch2-3 line 39-41）。
- 🔴 **收盤跌破關鍵支撐（缺口低點、慣性均線、大量低點、轉強K 低點）→ 出場**。
- 🔴 **大量黑K 出現後跌破其低點 → 出場**；要重新進場必須收盤站上其高點。
- 🔴 **隔日沖紀律：隔天開盤一定要賣**（不管賺賠；除非走進階續留條件）。
- 🔴 **收高開低 = 轉弱訊號**，已持有即開盤離場（國際利空除外）。

### 力量判斷

- 🔴 **量是因、價是果**（Ch2-4）。
- 🔴 **量縮 = ≤ 前一根的 1/2**；量增 = > 前一根（Ch2-4 line 76-81）。
- 🔴 **衝擊前高需要更大量**（右下角的量 > 前波高點的量）（Ch2-4 + Ch5-2）。
- 🔴 **大量 K 棒形成支撐/壓力**：左上方 = 壓力、左下方 = 支撐。
- 🔴 **窒息量必須月線上彎才有效**（Ex1-2）；否則容易真的窒息掉。

### 趨勢與型態

- 🔴 **必須完成「理想多頭排列」才考慮波段進場**（Ch2-2）。
  - 必要：5/10/20/60 價格排序正確
  - 理想：四條均線皆上彎
  - 加分：尚未發散
- 🔴 **缺口必須是真空地帶**（不含上下影線）（Ch2-3 line 114-119）。
- 🔴 **反轉形態的均線不可切到反轉紅K 本身**（Ch4-2 形態二）。
- 🔴 **布林上軌進場必須帶量、收 > 上軌**；通道窄度 < 10%。
- 🔴 **奇形：旗子的收盤價必須在 5ma 之上、下影線在旗杆中間之上**。

### 工具使用紀律

- 🔴 **不使用 MACD / KD / RSI**（互相抵觸無法決斷，Ch1）。
- 🔴 **技術判讀工具只有四件：均線、量價、缺口、型態**（前三件最重要）。
- 🔴 **不是每天都要做股票**（Ch7-1）。
