## 🎯 二、進場（Entry Detectors）

課程內 detector 在 `scripts/zhuli/entry/`，各自獨立、scanner 取聯集。

### 主力三大 detector

**1. 小結構整理末端** `entry/small_structure/detector.py:16-74`
6 條硬性規則全過才算命中：
- 前段攻擊 ≥ +10%（脫離成本）
- 高位整理 range < 10%（沒大跌、只是橫盤）
- 量縮 vol_ratio < 1.5
- MA5 追上（close ≥ MA5 × 0.93）
- close ≥ MA10 × 0.95
- close ≥ 近 10 日高 × 0.85

**2. W 底起漲** `entry/w_bottom_launch.py:47-87`
- 60 天內跌幅 ≥ 20%
- 反彈高 H2 ≥ H1 × 0.85
- 二次拉回不破前低（> L1 × 0.95）
- 接近反彈高（≥ H2 × 0.88）
- 量縮 < 1.5、未破 MA20 × 0.95

**3. 均線全順向** `entry/uniform_ma_above.py:35-100`
- 核心：MA5 > MA10 > MA20 > MA60 且 close ≥ MA5
- 反向工程濾網（排除過熱/冷門）：成交額 ≥ 2500 萬、距 MA20 < 12%、距 MA60 < 40%、20 日漲幅 ≤ 25%

### 其他 detector（`entry/__init__.py:49-59`）
suffocation（窒息量）、institutional_firstbuy（法人首買）、swing_breakout（波段突破）、overnight_swing（隔日沖）、reversal_breakout（反轉突破）、pennant_flag（三角旗）、institutional_swing、intraday（當沖）

## 🚪 三、出場（Exit）

### Production default：**C6 規則** = Rule A + Rule C + 轉倉
`tools/backtest_standard_workflow.py`（backtest +45.44%、目前主用）

**Rule A — MA10 容忍 + 量縮例外**（行 242-260）
- close ≥ MA10 → 不出
- -2% < close < MA10 **且量縮**（vol < 1.0）→ 觀察、不出
- -2% < close < MA10 **且放量**（vol ≥ 1.0）→ 出
- close < MA10 達 -2% → 深破、出
- 連 2 天在容忍區 → 出

**Rule C — 倉位水位 partial exit**（行 315-341）
- 倉位水位 > 15% → trim 到 13%（精準減碼、興櫃強制全出）
- 倉位水位 ≤ 15% → 全出

**C6 轉倉**（行 14-17）
- 條件：倉位滿 + 有 P3 候選 + 有 partial-trimmed 倉
- 動作：找最弱（距 MA10 最負）的 partial 倉、隔日開盤全出、換 P3

> **重點**：C6 沒有硬性 MA5/MA10 停損點、靠 Rule A 漸進評估。Core 持倉停損用**結構底**、不是 MA5（8064 東捷教訓）。

### 4 種 daily 出場 detector（backtest 用）`exit/detectors.py`
- **掀傘**（行 34-113）：連 3 不創新高 + 量縮 × 0.7、浮盈時主動全出
- **高檔長黑 K**（行 182-309）：長黑實體 ≥ 4% + 高檔區 + 2 項意義（缺口回補/包覆創高/吃下前 5 根）
- **分批停利里程碑**（行 323-383）：+10% / +20% / +30% 提示
- **隔日跳空急殺**（行 390-447）：跳空 ≤ -5% 緊急出、-3%~-5% 警示

## ⚡ 五、盤中 Trigger（5 種）

定義在 `intraday_stage_helper.py`、`check_trigger_inline` / `StageTrigger`。

| Trigger | 中文 | 觸發條件（摘要） | 性質 |
|---|---|---|---|
| **Ch5-3** | 首攻 | 連 2 紅 K + 量 × 1.5 + 距開盤 > +1% + 站穩前波高；09:45 後；未太貼日高 | 進場 |
| **T1** | 續攻 | 同首攻邏輯（延續波） | 進場 |
| **T2** | 反彈 | 跌深 ≤ -2.5% + 低點後連 3 紅 K（或 5m diff 由負轉正 early signal）+ 低點反彈 ≥ +1% | 進場（弱） |
| **TC** | 破底 | 跌破前波低 OR 距 MA10 ≤ -3% + 末段量 × 1.5 黑 K | **出場 / 警示** |
| **Closing** | 尾盤 | 13:05-13:25、5 項確認（見下） | 進場（最強） |

### 尾盤 Closing 的 5 項確認（行 1095-1276）
就是 monitor 上「3-4/5」那 5 項：

1. **結構**：close > MA10（守住）← **必過項、失敗直接 skip**
2. **殺盤**：12:00 後最低 < 早盤高 -2%（有經過殺盤考驗）
3. **反彈**：13:00 後連 2 紅 K
4. **量縮**：尾盤 per-bar 量 < 早盤 × 1.2
5. **未追高**：距日高 ≥ -1.5%（沒追在最高點）

**判定**：
- 3-4/5 pass → `confirmed`（Win 82%、標「最佳進場」）
- 5/5 pass → `overheated`（Win 40%、過熱警示）
- < 3/5 或結構失敗 → skip

> ⚠️ 這正是 6/11 4939 假燈的位置：monitor 用 stale 昨收（差一天）算出假跳空 -7.5%、讓「殺盤/量縮/未追高」假性成立 → 4/5 confirmed。**已修（commit `4ac2830`：DB 補日 K 自動失效 cache）。**

## 📌 核心哲學（戰略 > 戰術）

- **課程內邏輯與課程外條件物理隔離**：entry/exit/scoring 只放課程內、自定義條件放 extras/（預設 OFF）。
- **戰略優先**：全方位培訓/法人籌碼課的族群方向 > 個股戰術訊號。
- **收盤確認**：所有跌破/站回均以收盤價為準、不看盤中。
- **寧願錯過、等更好機會**：AI 不主動推進場、Stage 2 不自動執行（8046 -$62k 教訓）。
- **耕耘期 vs 打擊區**：平常小錢耕耘、打擊區（明確訊號）才大口吃肉。
