**關鍵發現:**

1. **原始策略無 edge** — EV ≈ 0 (pre-fee)、加費後負。代表「無過濾的通用綠K進場」本身沒有優勢，需要 pre-filter。
2. **趨勢盤是最佳環境** — 趨勢上漲 EV=+0.35%，其他環境均為負。暗示 v4 在震盪/偏弱盤應收緊入場或降 sizing。
3. **最強族群** — 金融/券商 (EV=+1.34%)、BMC (EV=+1.25%)、光通訊/CPO (EV=+0.67%)、功率元件 (EV=+0.53%)。
4. **最弱族群** — 記憶體 (EV=-1.08%)、BBU (EV=-0.87%)、機器人/特化 (EV≈-0.6%)。
5. **距 MA10 越近越好** — 0~3% 區間 EV=+0.15%、7~10% 區間 EV=-0.51%。
6. **量比高不代表好** — 高量比 (≥1.5) 反而 EV 更負，代表追量爆訊號效果差。
7. **大殺日意外最佳** — EV=+1.24% (n=85)，可能是反彈效應；但樣本少需謹慎。
