## 1. 案例重新分類 (5 大型態、非案例庫原 3 類)

依「下殺後 D+1/D+3 走勢」反推真正可操作的分類：

### T1 — 短期急漲後的真倒貨 (4 案、最危險)
- 條件: **前 5 日累計漲幅 ≥ +5%** + 拉高出貨形態 + 量比 ≥ 1.4x
- 案例: **5/28** (+10.86% / 1.59x), 5/15 (-1.69% 但 5/14 同型, 0.95x), 5/14 (連發, 0.50x), **2/23 (+5.90% / 2.02x — 假訊號但極限)**
- 3 日恢復: 5/15、5/14 = 否；2/23 = 是 (續漲)；5/28 待定
- **特徵**: 5 日累計越高 + 拉高出貨 → 後續越易連續 2-3 日弱
- **5/28 屬此類最 textbook (5d +10.86% 為 20 案最高)**

### T2 — 崩跌中反彈的拉高出貨 (3 案、其實是底部訊號)
- 條件: **前 5 日累計 ≤ -3%** + 跳空 ≥ +3% (恐慌反彈)
- 案例: 3/10 (5d -8.40% / 跳 +3.13% / 量 0.50x), 3/05 (5d -4.79% / 跳 +3.57% / 量 0.76x), 4/23 (5d +2.61% / 跳 +1.51% / 振幅 4.16% / 量 1.84x)
- 3 日恢復: 3/10 是、3/05 是、4/23 是 (+4.17%)
- **特徵**: 崩跌後的「拉高收黑」其實是空頭回補測底、不是真倒貨
- **量縮 (≤ 0.8x) 跳大缺口 (+3%) 反而最有效**

### T3 — 開低殺低 + 高量恐慌底 (3 案、V 反案例)
- 條件: 跳空 ≤ -1.5% + 量比 ≥ 1.9x + 當日收在日高/中段以上
- 案例: 3/09 (-4.81% / 1.92x / 收日高), 3/23 (-3.16% / 2.53x), 2/02 (-1.45% / 2.14x)
- 3 日恢復: 全部「是」
- **特徵**: 開低後拉、收日高/上段 = 主力承接訊號

### T4 — 連續下跌中的弱勢續跌 (5 案、最不該接刀)
- 條件: 前日已跌 + 量縮 (≤ 1x) 或開低殺低收日低
- 案例: 3/19 (0.54x 量縮), 3/12 (前日 +3.99%、隔日量縮開低), 3/04 (-4% 連 2 日連發), 5/15 (連 5/14), 1/30 (連 1/29)
- 3 日恢復: 全部「否」
- **特徵**: 前日已下跌 + 隔日量縮跳低 = 確認向下、不接

### T5 — 全天弱勢但 5 日累計小幅正 (3 案、體質好的拉回)
- 條件: 跌 -1.5%~-2.5% + 5 日累計在 -2%~+5%
- 案例: 5/19 (-2.06% 5d), 4/02 (-0.98% 5d), 3/31 (-0.47% 5d)
- 3 日恢復: 全部「是」(D+2 或 D+3 反彈)
- **特徵**: 量正常、無極端跳空、純技術性回檔

## 2. 真假倒貨判別公式 (ex-ante 可算)

從 9 個拉高出貨案例對比 (T1 + T2)，找出**可在當日收盤前算出**的 3 個 metric：

### Top 3 判別 metric

| Metric | 真倒貨 (T1) | 假倒貨/底部反彈 (T2) | 判別力 |
|---|---|---|---|
| **前 5 日累計報酬** | ≥ +5% | ≤ -3% | ⭐⭐⭐ 最強 |
| **量比** vs 5d avg | 1.0~2.0x (拉抬末段確認量) | ≤ 0.8x (量縮反彈) | ⭐⭐ |
| **跳空缺口** | +1.4% ~ +1.7% (溫和跳高) | ≥ +3% (恐慌反彈跳空) | ⭐⭐ |

### 判別公式 (可寫成 scanner)

```
if 前5日累計 >= +5% and 拉高出貨形態 (上影 >= 2% + 跳空 >= +1%):
    → T1 真倒貨、隔日續弱機率高、不接刀
elif 前5日累計 <= -3% and 跳空 >= +3%:
    → T2 底部反彈、隔日 D+1 大漲機率高
else:
    → 灰色、看量比 + 美股
```

### 量比反向使用 (5/19 課對應)
- 老師原話「拉抬末段爆大量先出一半」(5/19 [L624-635])
- 對應大盤: 拉高出貨日**量比 ≥ 1.5x + 5d 累計 ≥ +5% = 出量見高訊號**
- 5/28 量比 1.59x + 5d +10.86% → **完全符合「拉抬末段大量」格式**

## 4. 進場時機統計推論

### 連續弱勢 N 天後恢復率
- 案例庫 14 組「2 日累計 ≤ -2%」中、D+3 恢復 8 組 = **57%**
- 集中在 3 月 (9/14 組) → **3 月恢復率較差** (3 月有結構性壓力)
- 5 月 14 組中只有 3 組屬連續弱勢、且 D+2/D+3 均恢復 → **5 月反彈體質好**

### 拉高出貨日後進場勝率分布
| 進場時點 | 案例符合 | 勝率推論 |
|---|---|---|
| D+0 收盤 (拉高出貨當日尾盤接) | 9 案 | 6/8 = 75% (整體) |
| D+1 早盤 | 待算 | 3/05、5/15、5/14 開低續跌 → D+1 早盤接 = **危險** |
| D+1 尾盤 / D+2 (老師三天劇本) | 4/23、3/10、3/05 | D+2/D+3 反彈 **3/3** |
| D+3 全押 (替代方案) | 全部 | 12/19 = 63% |

→ **老師三天劇本 (尾盤分批) 歷史勝率最高**

### 量比 ≥ 2x 拉高出貨更危險嗎？
- 量比 ≥ 1.8x: 2/23 (續漲)、4/23 (大漲)、5/28 (待定)
- 量比 ≤ 1.0x: 5/14 (跌)、5/15 (跌)、3/10 (漲)、3/05 (漲)
- **結論: 量比本身不決定方向、要配合 5 日累計**
  - 高量 + 5d 已漲多 = 真倒貨 (但 2/23 反例存在)
  - 低量 + 5d 已跌多 = 底部反彈
  - **5/28 = 高量 (1.59x) + 5d 漲多 (+10.86%) = 真倒貨高機率**
