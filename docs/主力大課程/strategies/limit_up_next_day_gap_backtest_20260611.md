# 漲停隔日跳空門檻 backtest（C-001 驗證）2026-06-11

> **目的**：驗證「鎖漲停隔日跳空 ≥ +3% 禁推」（4526 教訓、n=1）是否成立、+3% vs +5% 哪個對。
> **方法**：standard_daily_bar 全市場、2025-01-01 ~ 2026-06-10、前日漲停（+9.45%~+10.5%、量 ≥ 500 張）→ 隔日開盤無條件進場、量隔日開→收與 D2 收報酬。
> **樣本**：10,693 筆（鎖死漲停 subset 10,329 筆）。

## 結論

1. **H1（+3%）與 H2（+5%）都錯** — 不是門檻選錯、是「門檻」這個概念錯。漲停隔日**任何開高（gap > 0%）無條件進場都是負期望**、+3% 紅線其實太寬鬆。
2. **開低不是禁區、反而是唯一正期望桶**（+1.27%、win 52.7%）— 「漲停隔日開低 = 危險」的直覺不成立。
3. 跳空越大尾部風險越大（p25 從 -3.8% 惡化到 -5.2%）— 原 4526 教訓方向對、但歸因錯（不是 3% 這個數字、是「開高追」這個行為）。
4. 再鎖死（≥+9.5%）買不到、無意義；D2 +1.09% 是動能延續、但不可執行。
5. **與老師 SOP 一致**：`feedback_locked_limit_up_next_day_entry` 本來就教「隔日試撮判讀 + 9:30 後委買量比」—— 即開高需要**額外確認**、不是無條件進。

## Spec 決議

- C-001 結案：**淘汰「+3% 門檻」表述**、升級為 R-DISC-011：
  > 漲停隔日：開高（gap > 0%）一律不做無條件進場（全桶負期望）；要進必須走鎖漲停隔日 SOP（試撮判讀 + 9:30 委買量比確認）。開低非禁區、可依一般 entry 規則評估。
- 未驗證殘留（資料限制）：H3 試撮假價、H4「9:10 後守住開盤價」需要分K/試撮資料、留待後續。

### 結論二（疊加在結論一之上）

1. **跳空方向的影響取決於進場時點**：
   - **開盤進**：開高 = 負期望（早盤 fade 風險全吃）→ R-DISC-011 適用
   - **尾盤進**：開高**且守住** = 最強動能確認（+5% 以上守住 → win 73-82%）；開高沒守住 = skip
2. 完美對應老師框架：**「開盤站賣方、尾盤做事情」** + 「守住」概念 — 尾盤進場時、早盤跳空的 fade 風險已經被「撐過一整天」這個事實過濾掉、剩下的是動能延續。
3. **R-DISC-011 範圍註記**：只規範「開盤/早盤無條件進場」（本來就被 R-DISC-005 禁止）；尾盤窗口進場時、跳空不是否決項、**「守住開盤價」才是關鍵判斷**（已被 Closing panel 結構/守住邏輯涵蓋）。
4. 執行面 caveat：「再鎖」桶若 D+1 收盤仍鎖死、尾盤實際買不到、+5.32% 高估了可實現報酬。

## 限制

- 無條件開盤進場版未模擬課程出場（依 `feedback_backtest_methodology` 本應用課程出場、此處僅做門檻方向性判定）。
- 尾盤版以 D+1 收盤價近似 13:00-13:25 進場價、有 ~30 分鐘誤差。
- 全市場樣本、未分流動性層級與族群；2025-2026 單一 regime。
