### Pre-filter 分析

**重要觀察（全市場 vs 老師族群差異）：**

1. **大盤過濾在老師族群效果有限**（AB 組合改善僅 -0.1pp），因老師族群在大殺日反彈更強
2. **距MA10 ≤+10%** 仍是單一最強 filter（+1.9pp），方向不變
3. **量爆在老師族群仍是風險訊號**（殺破率 32.8%），不因族群資金而改變
4. **全市場「排除邊緣量」改善 +0.3pp → 老師族群 +0.4pp**，效果略微放大
5. **Core tier 加持後殺破率 24.4%、均報酬 4.55%** — 老師點名 core/frequent 的標的確實有超額報酬

## Part 8: 修正版 SOP 建議

### 修正前（全市場版）原 SOP

```
進場條件: 紅K + 收≥昨 + 大盤非大殺/拉積 + 距MA10≤10%
停損: 結構底 / 連續紅K≥4根 / 大盤偏弱
```

### 修正後（老師族群版）SOP

#### 一、Universe 前置篩選（最關鍵）

```
1. 只做 teacher_sector_tickers.json 或 teacher_picks_2026.json 內的 ticker
   → 比任何技術條件更重要，單獨篩選即可降低殺破率 25.9pp
2. 或擴展到老師說的「~500 支」範圍（AI 相關 + 流動性 ≥5000萬日成交）
```

#### 二、優先推薦族群（按勝率排序）

```
🥇 一線（殺破率<25%、勝率>60%、均報酬>4%）:
   光通訊/CPO (5.21%均報、24.6%殺破)
   封測 (4.83%均報、23.8%殺破)
   ABF載板 (4.35%均報、24.6%殺破)
   功率元件 (3.78%均報、24.7%殺破)
   被動元件 (6.79%均報、27.9%殺破、均報最高)

🥈 二線（正報酬、殺破率<28%）:
   矽晶圓、CCL材料、IC設計、主流設備、PCB

🥉 三線（謹慎操作、降 sizing）:
   散熱、機器人

❌ 高風險（需加嚴門檻或 skip）:
   廠務設備（殺破率 35.1%、勝率 46.7%）
```

#### 三、大盤環境 SOP（老師族群版）

```
✅ 最佳進場環境（由強到弱）:
   1. 偏弱盤 / 其他 (均報 3.69-4.78%, 殺破 19-27%)
   2. 大殺日後（適合觀察族群反彈，5.48%均報）
   3. 趨勢上漲（仍可，但非最佳 2.77%）
   
⚠️ 謹慎：
   4. 拉積盤（1.64%均報，勝率 50.2%）
   
❌ 避開：
   5. 震盪盤（殺破率最高 33.0%，勝率僅 50.6%）
   → 注意：全市場版本結論在這裡相反，震盪盤在老師族群是最危險環境
```

#### 四、技術條件（pre-filter，按重要性）

```
必要條件:
  ✅ 紅K + 收盤≥昨收（基本）
  ✅ 距MA10 ≤+10%（改善殺破率 +1.9pp）
  ✅ 族群在 AI 主軸 or 擴散AI（非廠務設備）
```
