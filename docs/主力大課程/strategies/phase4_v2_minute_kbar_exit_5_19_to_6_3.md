## 2. Detector 改良說明

### 掀傘 — Daily → 5分K
| 版本 | 條件 |
|------|------|
| v1 Daily | 日線連3根不創高 + vol_ratio < 0.7 (日均量比) |
| **v2 5K** | 當日5分K連3根不創高 + vol_ratio < 0.7 (5K均量比) |

改進點: 日線1根 = 整天，5K 1根 = 5分鐘，能偵測**當日尾盤主力收手**訊號，不必等到次日才確認。

### 高檔長黑 — Daily + intraday 確認
| 版本 | 條件 |
|------|------|
| v1 Daily | high_zone_ratio ≥ 1.3 + 3種意義 ≥ 2 |
| **v2 intraday** | high_zone_ratio 放寬至 1.2 + 加 intraday 波動確認 (range ≥ ATR5 × 1.2) |

改進點: high_zone_ratio 放寬以提高靈敏度，用 intraday 真實波動率補強確認。

### 急殺 — Daily open → 1分K 09:00 第一根 open
| 版本 | 條件 |
|------|------|
| v1 Daily | 當日 daily open vs 前日收 |
| **v2 1min** | 1分K 09:00~09:05 第一根 open (真實開盤價) |

改進點: daily open 可能是加權平均或快取，1分K第一根才是**真實撮合開盤價**。

## 3. Backtest 結果對比

**User 總損益:** +$232,534 (15 筆)

### 掀傘 — 核心改善案例

| 版本 | 觸發次數 | 觸發標的 | 平均 Δ PNL vs User |
|------|---------|---------|-------------------|
| v1 Daily | 1/15 | 2885 元大金 (6/2) | **-40,861** (早出損失) |
| **v2 5K** | 4/15 | 3149/2464/6282/3481 | -25,473 |

**2885 元大金 核心案例:**
- v1 Daily 掀傘: 觸發 @ 2026-06-02 $68.9 → PNL +13,743  
- v2 5K 掀傘: **未觸發** ✓  
- User 仍持有 6/3 @63.7 → PNL +54,603  
- **改善: +40,860 (避免 daily 版誤觸早出)**

v2 5K 版觸發的 4 筆:
- 3149 正達: 5/25 @65.0 PNL +22,622 (User 5/28 出 +23,036 → Δ -414, 幾乎相同)
- 2464 盟立: 5/25 @158.0 PNL +41,717 (User 5/28 出 +85,185 → Δ -43,468, 早出)
- 6282 康舒: 5/25 @63.1 PNL +15,488 (User 5/28 出 ±0 → Δ +15,488, **5K版更好**)
- 3481 星宇航: 5/22 @44.65 PNL +19,410 (User 仍持有 +92,909 → Δ -73,499, 大幅早出)

> **結論:** 5K 掀傘觸發4次但誤觸率仍高 (2464/3481 都是強勢股被誤觸)。需加「連 N 日趨勢中」過濾條件。v1的2885誤觸問題確實改善，但5K觸發條件對「波段主升段」股票太敏感。

### 高檔長黑 — 0/15 兩版均無命中

原因分析:
- high_zone_ratio 1.2 放寬後仍需同時滿足 M2 嚴格包覆條件
- 2026-05-19~06-03 這段行情主要是上升趨勢，高檔反轉 case 少
- 唯一候選: 3016 嘉晶 6/2 大黑K，但進場當天即出場無觀察期

### 急殺 — 0/15 兩版均無命中

原因分析:
- 1分K版讀取 09:00~09:05 真實開盤，但樣本期間無跳空 ≥ -3% case
- 3016 嘉晶 6/2 是試撮違紀進場，合法進場後當天損失屬「進場失誤」非「急殺訊號」
- 8027 鈦昇 5/19→5/20 跌幅雖大但為結構跌破，非跳空急殺型態

### 分批停利 — 兩版相同

| +10%   | 4       | 2476, 3149, 3481, 2464(5/25) |

## 5. Monitor 是否切換到 1 分 K 為主

**現況:** WSCache 已 push 1分K資料，monitor 基礎設施就緒

**建議:** 是，應切換

理由:
1. 2885 案例証明 daily 掀傘會誤觸 (-$40k)，5K 版已修正
2. 急殺 detector 用 1分K 09:00 第一根才是真實開盤 (daily open 有延遲)
3. 5K 版缺點是強勢股誤觸率偏高 (3481 5/22 就觸發)，需加 filter:
   - 趨勢過濾: MA5 仍上升中不觸發掀傘
   - 或僅在「浮盈 ≥ +15%」後才啟動掀傘偵測
