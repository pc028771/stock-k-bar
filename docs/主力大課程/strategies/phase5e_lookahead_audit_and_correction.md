# Phase5 Lookahead Bias Audit — Universe 來源分析與修正

**審查日期:** 2026-06-04  
**審查對象:** phase5_weekly_report_only, phase5b, phase5c, phase5d  
**核心問題:** 週報「段1 崩盤筆記本」= 回顧性資料 = Lookahead Bias

---

## 1. 問題定義

本週市場資訊報的結構分為兩段：

| 段落 | 標題 | 性質 | Backtest 可用性 |
|------|------|------|----------------|
| **段 1** | 「回顧本週單日漲幅 6%+ 族群 (老師崩盤筆記本)」| **回顧性**：老師列出這週已漲的標的觀察清單 | ❌ Lookahead Bias |
| **段 2** | 「本週新聞題材」| **前瞻性**：老師整理下週關注的新聞/籌碼邏輯 | ✅ Forward-looking |

**段 1 是 retrospective**：發佈時這些標的已經跑了整週。會員「讀完再進場」等同於已知過去一週的強勢族群才選股 — 這是 lookahead bias。

## 5. 修正後的誠實結論

### 對 User 的 Honest Assessment

| 問題 | 答案 |
|------|------|
| Phase5d +$2.37M 真實嗎？ | **部分不真實**：約 69%（~$1.64M）來自 lookahead bias 段1標的 |
| 純 forward-only 的 P&L？ | **Phase5b forward: ~+$4k（持平）**；Phase5d forward 估~+$740k |
| 哪個版本 forward 100%？ | **沒有任何版本是 100% forward**，但培訓影片+Line 來源的部分是 |
| 真實會員可達報酬？ | Phase5e（待跑）：用週報段2 + 培訓影片 universe 估算 |

### 主要發現

1. **Phase5b 的 +$110,670 是假象**：3 檔段 1 lookahead 標的（3481/3149/3317）貢獻 +$205,425，排除後剩 -$94,755。

2. **Phase5d 的 +$2.37M 虛增估計 ~$1.6M**：最大贏家（3481群創+123k、6239力成+132k、2327國巨+129k、6284佳邦+146k、2492華新科+126k）全是段 1 崩盤筆記本標的。

3. **培訓影片/Line 來源的標的是 forward**：6182合晶+113k、3675德微+71k、2481強茂+78k 都來自 5/22-5/25 的 Line 群/直播。

4. **週報段 2 的 forward P&L 相對保守**：2303聯電+122k（phase5d，但 phase5b 沒有）、5347+55k 是最好的結果。
