# Stage 1+1+1 SOP 歷史模擬 — 2026-06-02

> 對 user 過往 7 檔操作做反事實 (counterfactual) 模擬、驗證 Stage SOP 是否優於 all-in。

## 模擬規則

### 樂觀 all-in
- Stage 1 訊號出現當日、進 3 張
- 抱到模擬期最後一天 (2026-06-02 或 user 出場日)
- 不做任何停損

### 真實 all-in + 恐慌出
- Stage 1 訊號出現當日、進 3 張
- 任何 -5% drawdown 觸發、賣在 -5% 線
- 對應 user 過往實際行為（panic exit）

### 紀律 Stage 1+1+1
- Stage 1: 訊號日進 1 張（30-40% sizing）
- Stage 2: 回測守住 (-5%+ drawdown + 紅 K + 收高於前日) 進 1 張
- Stage 3: 突破前波高 + 量 ≥ 1.5x 5 日均量 進 1 張
- 抱到最後一天

## User 自述心理特性 (2026-06-02 親口)

> 「這個 SOP 對我來說就是最大利潤、因為除非是我自己很有把握、或是有人說得很清楚、我才能放心 all-in、不然就算是老師報的我現在也會怕。」

**對應 AI 行為**：
- 即使老師明示、預設 Stage 1 試水 30-40% sizing

- 「恐慌 -5% 出場」是 user 真實行為、SOP 設計就是為此
