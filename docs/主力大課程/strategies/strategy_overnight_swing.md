# 隔日沖策略 (Overnight T+1) — D 策略

## 策略屬性

- **strategy_mode**: `overnight`
- **持有期**: 進場到隔日 9:00 開盤（~19 小時）
- **推薦度**: ⭐ 主推（backtest +36.91%）

---

## 進場

同當沖、使用相同 trigger：

- **Ch5-3 confirmed**：第一根 5K 6/6 條件 + 9:10 後過高 + 回踩 MA10 守住（弱勢盤）/ 過高即進（強勢盤）
- **T1 confirmed**：強勢延續（Stage 2 加碼場景）
- **T2 confirmed**：跌深反彈、3 紅 K 確認

每日 top 2（用 score 排序）：

```python
score = trigger_type_base          # Ch5-3=30, T2=25, T1=20
      - minutes_after_910 * 0.5   # 越早越好
      + tier_bonus                 # core=+25, frequent=+15
      + dist_ma10_bonus            # 距 MA10 ≤5% → +10
      + vol_ratio_bonus            # vol_ratio ≥3 → +10
      + regime_bonus               # strong → +5
```

---

## 持有期（~19 小時）

進場後到「隔日 9:00 開盤」= 當日剩餘時間 + 隔日試撮觀察 + 開盤第一筆

**持有期紀律（不可違反）：**

- ❌ 不殺尾（13:30 不出）
- ❌ 不加 % 停損（除非老師結構底跌破）
- ❌ 不加碼（除非真的觸發 Stage 2、不在 D 策略範圍內）

---

## 出場

**主規則：隔日 9:00 開盤價、無條件出。**

例外狀況：

| 情況 | 處理 |
|---|---|
| 開盤跳空 ≤ -5% | 立即出（避免進一步擴大損失） |
| 開盤跳空 ≥ +5% | 重新評估、可考慮續抱（但要重跑 trigger） |
| 其他（-5% ~ +5%） | 隔日 9:00 開盤價出、無條件 |

## Backtest 結論

**期間**：2026-05-19 → 2026-06-03（11 個交易日 × top 2、共 20 筆）

| 策略 | Win% | 平均淨報酬 | 累計淨 |
|---|---|---|---|
| 純當沖 13:30 | 40.9% | -0.798% | -17.57% |
| 純當沖 掀傘 | 31.8% | -0.487% | -10.71% |
| Stop -1.5% | 22.7% | -1.067% | -23.48% |
| **D 隔日 9:00 出** | **65.0%** | **+1.845%** | **+36.91%** |

D 完全壓倒所有純當沖組合。

**注意**：樣本量 20 筆（11 個交易日）尚不夠多、需持續驗證。

## 適用情境

- 大盤強勢 / 正常盤（非重大利空期）
- Trigger confirmed（Ch5-3 / T1 / T2）
- 有過夜風險承受度（持有期含收盤後 + 次日開盤前）

**不適用情境：**

- Core 持倉（用結構底停損 + 波段策略、不是 D 策略）
- 颱風假前夕、重大財報前夕
- 主力有明確倒貨跡象

## 關係說明

D 策略（隔日沖）與波段策略（swing）的差異：

| 比較項目 | D 策略（overnight） | 波段（swing） |
|---|---|---|
| 持有期 | ~1 個交易日 | 3-5 天 |
| 出場規則 | 隔日 9:00 無條件 | 結構底 / MA5 跌破 |
| 適用場景 | 短打題材 / 當日 trigger | 有趨勢的主流股 |
| 停損 | 持有期短、不設 % 停損 | 結構底停損 |
| Backtest | +1.85% / 筆 | +2.35% / 筆（持 3d） |
