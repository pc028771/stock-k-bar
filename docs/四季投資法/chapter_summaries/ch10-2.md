# ch10-2 — 10.2 波動市場下怎麼活？建立能穿越牛熊的投資體質 feat. 股魚

> 時長：36:25 | cues：1362 | triggers：131

## 核心主題（1-2 句）

對談章節（feat. 股魚），價值投資與四季投資法的互補：如何用四季框架精準找到「立夏起漲點」，解決價值投資「等太久資金卡死」的問題；籌碼面+技術面的雙重確認 SOP。

## 講師說了什麼（重點段落，含 timestamp）

- **00:26 — 問題設定**：股魚的基本面投資者學員反映，找到好股票但等待半年甚至一年以上，資金卡死，無法靈活調配，這跟四季投資法的「資金效率」有什麼解法？（@01:00）
- **02:01 — 四季法的精準起漲訊號**：四季法非常強調資金效率，要放在「正在夏天炙熱的股票」。精準起漲訊號：布林通道上軌斜率 >3（上通道打開 = 技術面發動）。（@02:38）
- **02:44 — 立夏判斷**：「漲停板開布林那一天」或「大漲長紅開布林那一天」就是立夏；布林上軌斜率 >3 可以用系統偵測。（@02:38）
- **03:20 — 籌碼面+技術面雙重確認 SOP**：春天時先看關鍵分點大買 / 投信連買 / 主力連買 → 籌碼面過濾 → 等技術面發動（立夏訊號出現）→ 進場。（@03:34）
- **08:00 — 基本面不等於漲的邏輯**：主力籌碼是領先指標；EPS 高、本益比低只是「過去」的數據，未來不一定這麼好；秋天的高 EPS 是主力出貨的掩護道具。（呼應 ch7-2）
- **15:00 — 可轉債作為春季工具**：春天等待期用可轉債（利息）讓資金有最基本的回報，等到正股起漲再賺股票漲幅，解決「資金卡死」問題。（@15:00）
- **20:00 — 動能交易思維**：夏天要相信趨勢的慣性，不要因為漲多就不敢買，告別逢低佈局。

## 量化條件 / 具體數字

- 立夏技術訊號：布林上軌斜率 **>3**（@02:51）
- 春天等待解法：可轉債賣回殖利率 >4%，同時等正股立夏
