# ch6-3 — 6.3 出貨行為實戰案例：從分點籌碼識別主力撤退訊號

> 時長：13:03 | cues：424 | triggers：61

## 核心主題（1-2 句）

高檔分點大賣的具體個股案例（台中銀、虹光、佳大、鑫龍騰等）；2021 年航運股申報轉讓案例；高檔大賣訊號辨識與應對。

## 講師說了什麼（重點段落，含 timestamp）

- **00:07 — 高檔分點出貨邏輯**：分點出的張數很多就是大主力出貨，通常對股價有很大傷害。「當股票在高檔你看到主力賣、有分點大賣，記得趕快離場。」（@00:20）
- **00:28 — 台中銀 2812 案例**：國泰分點在高檔（23.3 附近）一路大賣到 20 塊，高檔大賣主力大賣後跌。（@00:31）
- **00:46 — 虹光 2380 案例**：夏天來得又快又急又短，漲上來後永豐金新竹大賣，股價很快回檔，四季循環很快結束；提醒每天晚上要研究籌碼有無大賣。（@00:46）
- **01:21 — 佳大 2033 案例**：漲上去後永豐金敦北在高檔大賣，對股價有影響。（@01:23）
- **01:33 — 鑫龍騰 3188 案例**：元大博愛在 4 月份低檔大買，反彈至高位後元大博愛也大量出貨，後面容易修正。（@01:36）
- **01:53 — 官田鋼 2017 案例**：雖然沒大漲，稍微反彈後元大府城大賣，在地人大賣要多小心。（@01:53）
- **02:13 — 2021 年航運股申報轉讓**：航運類股（航海王時代），高點時有人做申報轉讓（名字：郭人豪），大量申報轉讓是重要出貨訊號；高點漲到後主力呈現賣超、集保戶數大增。（@02:16）
- **03:18 — 高檔主力賣超籌碼訊號**：集保戶數大增＋申報轉讓＋主力賣超，這三種訊號同時出現在高點是最強的出貨訊號。（@03:22）

## 量化條件 / 具體數字

- 高檔出貨訊號（定性複合條件）：分點大賣 + 主力賣超 + 集保戶數大增 + 申報轉讓（@03:22）
- 每天研究時機：「每天晚上要研究籌碼有沒有大賣」（@01:19）
