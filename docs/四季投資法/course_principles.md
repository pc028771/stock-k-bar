## 一、不可違反的基本原則

1. 四季 = 主力資金運作的四個階段（吃貨→拉抬→出貨→離場）✅（@ch1-1 05:00 / @ch6-1 01:52 / @ch8-1 00:37 投影片三重確認）
2. 個股四季是獨立的，不能完全依大盤週期推斷個股位置（2026 年台股：大盤強但 2/3 公司在年線以下）
3. 在對的季節做對的事；不要在冬天播種，不要在秋天貪婪
4. 四季投資法是動態的決策框架，非死板規則；有些股票四季不明顯
5. 主力籌碼是領先指標；財報（EPS、營收）是落後指標，不可反向依財報買賣 ✅（@ch8-1 + @ch10-2 嘉賓確認；秋天高 EPS 是主力出貨的掩護工具）
6. 「量價是表象，籌碼才是內核」
7. 股市最大的風險不是虧損，而是沒有紀律；絕對不能大賠
8. 3 步 SOP：**定位（Identify）→ 策略（Execute）→ 觀察（Monitor）**
9. 主力 vs. 散戶平行時空（@ch6-1 01:52 投影片 vision 確認）✅：春=主力偷偷吃貨/散戶無聊；夏=主力點火拉抬/散戶追高；秋=主力用利多倒貨/散戶看財報加碼；冬=主力早已離場/散戶套牢攤平
10. **情緒週期英文對應**（@ch8-1 00:37 投影片 vision 確認，僅投影片出現）✅：春=Accumulation、夏=Markup（Euphoria 頂峰）、秋=Distribution、冬=Abandonment（Despair 低點）

## 二、四季定義（識別條件）

### 春 — 低檔吃貨（Accumulation）

**量能條件**
- 成交量通常低於 20 日均量的 **0.8 倍**（ch2-1 投影片+字幕確認；PDF CH2 P1 寫 0.5，截圖投影片更正為 0.8）⚠️ PDF vs 影片不一致，以影片為準
- 量低價穩，量能窒息，週轉率很低
- 成交量「連續多日不見放量，但股價守穩特定區間不破」

**價格條件**
- 股價不再大幅破底，從下跌轉向築底，橫盤整理
- 股價在年線附近盤整
- 目的是為吃貨而非拉抬

**均線條件**
- 5MA、10MA 開始由下走平並逐漸靠攏
- 選股參數月線斜率介於 **-1～1**（月線既沒有強烈上揚也沒有快速下跌）
- 帶寬（布林帶寬）**<10**

**籌碼條件**
- 主力 5/10/20 日籌碼 **>0**（近期籌碼漂亮）
- 可觀察到特定分點籌碼持續穩定流入，但股價無明顯漲幅

**基本面條件**
- 產業穩健，獲利能力平穩，殖利率穩定，每年配息不差太多
- 殖利率大於 **5%** ✅（@ch2-2 05:31~08:26 軟體篩選截圖確認；摘要先前誤記為 4%，@vision 更正）
  - 備注：PDF CH2 P4 寫 4%，軟體截圖條件列與後製說明框均顯示 5%，字幕@ch2-2「我希望殖利率大於5%」，以截圖+字幕為準
- 配息穩定性要求：選「歷年配息穩定」的；「今年特別高但往年普通」的要謹慎（@monthly_202604 字幕）
- 股價淨值比建議不超過 2-3 倍（@ch2-1 01:04）✅
- 排除：營建股（殖利率不穩定，常大好大壞）

**時間軸**：持續到市場「冬盡春來」三大訊號出現前（定性；可能長達半年至一年以上）

**進場訊號**（冬→春轉折）：三個訊號同時出現：
1. 低檔止跌結構確認（股價不再破底）
2. 成交量從窒息轉為緩步增加（人氣回籠）
3. 產業趨勢開始轉強（新題材徵兆）

**滿倉定義**（@ch2-2 02:40 截圖確認）：
- 一檔股票最多佔可操作資金的 **10%** 即滿倉，建議不超過 10%
- 換算：該股跌停時對總資金影響 < 1%

**操作心法**：農夫心態，耐心與紀律；分批進場（三步驟：觀察期輕倉→確認期逐步加碼→完成核心持股）；禁止一次性重押（All-in）

### 夏 — 主升段噴發（Markup）

**量能條件**
- 爆量上漲（條件 A）；成交量 **>500**（立夏參數）✅
- 持續性爆量且量價同步創下近期新高

**價格條件**
- 突破關鍵壓力（條件 B）；股價爆發力最強
- 有效站穩關鍵壓力位，沿均線陡峭上漲
- 乖離年線 **<30%**（立夏時，代表剛起漲未漲太多）；盛夏階段乖離可更大 ✅

**均線條件**
- 立夏：月線斜率 **<0.5**（剛起漲）✅、上軌斜率 **>1**（布林上通道打開）✅、位階 **>8** ✅
- 盛夏：月線斜率 **>0.5**、上軌斜率 **>1**
- 立夏技術訊號進階：布林上軌斜率 **>3**（ch10-2）

**籌碼條件**
- 立夏：主力 1/5/10/20 日籌碼 **>0**（全部大於 0）✅
- 盛夏：主力 5/10/20 日籌碼 **>0**（不要求主力 1 日）✅
- 龍頭股帶動族群，法人報告開始推薦

**族群條件**
- 族群集體發動（同產業/題材多檔同步轉強）
- 主流資金集中（從防禦型股票流向該題材）
- 高度市場想像空間（題材有成長願景）

**進場時機**：
- 「漲停板開布林那一天」或「大漲長紅開布林那一天」= 立夏
- 確認突破關鍵壓力且爆量後進場，告別低接思維，勇敢追價

**春轉夏轉折點兩個關鍵條件**（@ch2-2 03:43 截圖確認）✅：
1. **帶量長紅**（K 棒帶量收長紅）
2. **突破區間**（突破盤整期壓力位）

**夏季實戰操作要點**（@ch3-2 35:00 投影片 vision 確認）✅：
1. 選股策略：鎖定市場主流題材、資金高度集中的標的
2. 進場時機：等待股價以「爆量」方式突破關鍵壓力，達成價量共振訊號
3. 執行果斷：確認趨勢後果斷進場，不要為了等待深回檔（深跌）而錯失良機
4. 加碼節奏：利用股價上漲過程中的短暫整理期進行加碼
5. 處置警訊：進入處置股可以先減碼，等待處置回檔後加碼
6. 市場訊號：密切觀察市場是否過熱、監控龍頭股走勢，一旦出現警訊（主力大賣、乖離過大、月線殺兩次等）應果斷收割獲利

**加碼節奏**：利用上漲過程中的短暫整理期加碼；進入處置股可先減碼，出關回檔後再加碼

**停損條件**：（CB 操作）破月線停損；重新進場條件：創新高即重新進場

**處置股操作要點**（@ch3-2 多個時間戳 vision 確認）✅：
- 快處置前（上通道）→ 找賣點
- 處置後回檔（下通道）→ 找買點
- 注意股票：連續 3 日注意 → 成為處置股，每 5 分鐘成交一筆（@08:26 軟體確認）
- 創高後第一次回檔 = 最安全的進場點

**大暑識別**（@ch3-2 12:00 附近 vision 確認）✅：
- 股票可以回檔，但**月線不能下彎**（下彎=換季警訊）
- 月線殺兩次 = 秋天形態，非洗盤
- 反彈沒力（只漲一點點後又再往下）= 大暑訊號
- 乖離年線 **200%+** 要小心（晶豪科案例：年線73元、股價196元）

**操作心法**：決斷力、相信趨勢慣性；「克服懼高症」；只要趨勢未變（未跌破重要支撐）就續抱

### 秋 — 高檔出貨（Distribution）

**量能條件**
- 量價背離：成交量依然巨大但股價漲幅縮小，換手不積極，賣壓湧現
- 高檔爆量長黑，或一紅一黑震盪

**價格條件**
- 股價在高位，波動加劇，趨勢由向上轉橫盤或緩跌
- 一紅一黑節奏（今日大漲隔日大跌）
- 月線「兩次跌破」（創高後殺一次月線，反彈後再殺一次）= 明確出貨訊號
- 乖離年線 **>30%**（選股參數，代表已漲多）✅

**均線條件**
- 5MA/10MA/20MA 開始走平甚至下穿長均線（初步結構改變）
- 秋季選股參數月線斜率 **介於 -1～0**（非「<0」，下彎但未死）✅
- 月線下彎或兩次跌破 = 出貨訊號

**籌碼條件**
- 主力 5/10/20 日籌碼 **<0**（選股參數，**無**主力 1 日條件）✅
- 主力大賣、散戶大買、集保戶數大增、大戶持股減少
- 法人忽然撤出
- 申報轉讓大量出現
- 買賣家數差從負值轉正值（主力從買轉賣）

**秋天常見型態**（@ch4-2 13:36 投影片 vision 確認）✅：
> 長紅後有大量隔日沖 → 賣壓大

- 長紅隔天容易黑K
- 漲停板隔天容易黑K
- 主力一買一賣（分點隔日沖大量進，隔天賣）

**秋季檢查表**（@ch4-2 06:29 投影片 vision 確認）✅：
- ✓ 診斷：K線是否出現「一紅一黑」且高檔震盪？
- ✓ 真相：是否出現「利多不漲」的背離現象？
- ✓ 行動：是否已經開始執行「分批減碼」？
- ✓ 警覺：是否隨時準備應對「斷崖式下跌」？

**CB 出貨訊號**：高檔融券大增 → 融券現償大增（CB 轉換）→ 主力轉賣 → 股價從秋轉冬

**入秋三大警訊**（ch7-1）：
1. 月線跌破兩次（多頭防守無力）
2. 月線從往上→走平→開始下彎（均線結構變差）
3. 龍頭股/指標股頻繁出現警示或處置，無法再創新高（「處置癌」）

**秋季操作策略**：
- **進攻停止，轉為防禦**
- 分批減碼（不要一次賣光）；利用反彈降低持股水位
- 嚴禁追高與過度交易；**停止逢低買進**（秋天低點越來越低）
- 降低曝險部位，增加現金比例

**秋季高檔出貨模擬交易 SOP**：
1. 第一天反彈紅 K **>3%** 時，研究買超分點有無大量隔日沖
2. 有隔日沖 → **10%** 資金於第二天買盤竭盡時模擬短空，急殺回補 ✅（ch4-1_03-46.jpg 投影片確認）
3. 第二天若又有大量隔日沖大買 → **20%** 資金於第三天再做一次 ✅（同上）

**人工核對說明（2026-05-20）**：
- 截圖投影片 @03:46 清楚寫：第二天 10%、第三天 20%
- 字幕（VTT）在此段落**完全未提及任何 %**（搜全章確認）
- PDF CH4 P2 寫「50%」— PDF 為誤記，以投影片截圖為準

**不適用秋季出貨法的狀況**：(1) 籌碼轉成主投買，(2) 近期有波段單主力，(3) 創新高

**短空停損**：設創新高停損；遇漲停板一律停損隔天再戰

**操作心法**：警覺；「不懂行情時，不動作就是最好的動作」

### 冬 — 量縮下跌（Mark-down）

**量能條件**
- 成交量窒息（市場乏人問津，想賣也賣不掉）
- 均量比 **<0.8**（量縮，選股參數）
- 下跌出量可以搶反彈；下跌沒量不要接

**價格條件**
- 緩跌（溫水煮青蛙，每日小跌累積成大傷）
- 「看似便宜，實則會更便宜」（價值陷阱）

**均線條件**
- 選股參數：月線斜率 **<-0.5**、上軌斜率 **<-0.5**、下軌斜率 **<-0.5**（三線都往下）
- 空頭排列：股價 < 10MA < 20MA < 60MA（所有均線皆下彎）

**籌碼條件**
- 主力 5/10/20 日籌碼 **<0**（主力長期往下）
- 籌碼鬆動，主力完全離場
- 利空消息反應劇烈

**操作策略**：
- **策略一（主要）**：防守優先、現金為王；避開弱勢股，不要試圖接刀子；「在熊市冬季，最佳操作往往是零操作」
- **策略二（進階）**：反手波段放空；條件：空頭排列 + 主力完全離場 + 產業基本面無支撐；反彈空（股價反彈到壓力線時放空）；不追空（急跌不進場，容易反彈）

**操作心法**：紀律；保存資本，等待是主動的策略

## 三、操作 SOP（每季）

### 春季操作

- **進場條件**：三大轉折訊號同時出現（止跌+量能回溫+趨勢轉強）；股價在年線附近盤整+量創波段新低+基本面無惡化
- **加碼條件**：分三步：輕倉觀察→逐步加碼→完成核心持股
- **停損條件**：基本面出現重大惡化（一般不設嚴格停損，因為位階低）；或使用可轉債替代
- **停利條件**：等春轉夏的立夏訊號出現後，切換到夏季策略；或等配息（持有中）

### 夏季操作

- **進場條件**：立夏訊號（漲停開布林/大漲長紅開布林）；爆量突破關鍵壓力＋族群連動確認
- **加碼條件**：上漲過程中的短暫整理期；乖離年線仍在合理範圍
- **停損條件**：跌破重要支撐（如月線）；CB 操作設破月線停損
- **停利條件**：見夏轉秋警訊（量價背離/領頭羊力竭/情緒極度樂觀）果斷停利；分批獲利了結

### 秋季操作

- **進場條件（短空）**：反彈紅 K >3% 後，確認有大量隔日沖；用 10% 資金第二天短空，第三天再用 20%（截圖確認；PDF 50% 誤記）
- **加碼條件（短空）**：第二天若再有大量隔日沖，第三天再做一次
- **停損條件（短空）**：設創新高停損；遇漲停板一律停損
- **停利條件**：急殺後回補；波段空單在大盤轉空時可留少量

### 冬季操作

- **進場條件**：原則上「零操作」；進階放空條件：空頭排列+主力完全離場+基本面無支撐
- **加碼條件**：不追空
- **停損條件**：反彈空時，股價反彈超過壓力點即停損
- **停利條件**：等待冬盡春來三大轉折訊號出現，轉換操作模式

## 四、可轉債應用（春/夏 case）

### 春季可轉債

- 春季是佈局可轉債「最具效率的時刻」：價位低檔、賣回殖利率漂亮、安全性高
- **核心優勢**：下檔有支撐（債權保底）、上檔無極限（換股獲利空間）→ 不對稱的獲利與風險
- **優選條件**：
  - 可轉債價格 **<100**（處於低檔）✅（@ch_trial 07:00 截圖確認）
  - 賣回殖利率 **>4%**（漂亮的標準）；軟體「領債息/套利」分頁篩選門檻 **>3%**（@ch_trial 03:00 截圖確認）
  - 負債比 **<60%**（倒閉機率低）；截圖彈出框（@ch_trial 03:55）顯示嚴格版 **<50%**（兩個標準並存，課程兩處各有不同門檻）
  - 速動比 **>100%**（快速變賣資產還債能力）
- **費用結構（康和新金鑽）**：手續費千分之 **0.5**（來回），庫存成本利息 **2.5%**（六倍槓桿時借貸利息 0.42%），賣回利率 **>0.42%** 就划算（@ch2-3 07:00）
- **停損**：CB 持有中若正股走夏天，破月線停損；重新進場條件：一創新高就進場（@ch3-3 02:51/03:03 截圖確認）✅

### CB 轉換四步驟（@ch4-2 01:00 投影片 vision 確認）✅

CB 概念股秋季型態：
1. **高檔融券大增**（空方介入，CB 轉換套利）
2. **融券大減 + 融券現償大增**（CB 轉換，空方平倉）
3. **主力轉賣**（主力開始出貨）
4. **股價從秋轉冬**（轉換完成後股價下跌）

> 折價訊號 = 秋天/出場訊號（@ch9-1 07:34 vision 確認）✅：
> - 定義：CB 委賣價 < 轉換後市值 = 折價

## 八、跨季節轉換訊號

### 冬→春

1. 低檔止跌（股價不再破底，W 底/箱型底/圓弧底；低點一個月內不再跌破）
2. 量能回溫（成交量脫離窒息，買盤進駐，主力資金重新進場）
3. 產業趨勢轉強（特定族群強於大盤並領先上漲，龍頭股率先回穩，基本面傳好消息）

### 春→夏（立夏）

1. 成交量明顯放大（爆量）
2. 突破關鍵壓力點（股價創階段新高，脫離底部區間）
3. 族群發動（同產業齊漲訊號）；技術面：布林上軌斜率 >1（立夏參數）/ >3（精準版）

### 夏→秋

1. 量價背離（股價創新高但量萎縮；或爆量但漲不動）
2. 領頭羊力竭（族群最強股率先跌破月線或季線）
3. 情緒極度樂觀（散戶參與度達到頂峰，擦鞋童理論）
4. 月線兩次跌破 + 龍頭股頻繁處置（ch7-1 補充）✅
5. **下一次反彈沒有再創高** → 準備出場（@ch10-1 / ch10-2 / ch10-3 嘉賓視角補充）

### 秋→冬

1. 成交量窒息（量縮）
2. 所有均線下彎（空頭排列）
3. 利空消息反應劇烈
4. CB 大量轉換後（融券現償大增）主力大賣

## 十一、盤中異動 7 大訊號（@ch9-2 13:47 + 15:00 vision 確認）✅

> 軟體：**全方位獨門監控** → 本方位（盤中流水牆）

**偏多訊號（3 種）：**
- **股買**：股票買超
- **買購**：買進認購權證
- **創高**：股價創近期高點

**偏空訊號（4 種）：**
- **股賣**：股票賣超
- **賣購**：賣出認購權證
- **創低**：股價創近期低點
- **期賣**：期貨賣超

**程式交易（隔日沖）識別：**
- 同一秒鐘買進多檔不同代號 = 一定是程式交易（「吃豆腐」）
- 流水牆顯示「股買」後突然一排 = 在吃發行商豆腐
- 識別：發行商變盤約 100 毫秒，程式在 **20 毫秒**內搶先

**正/逆價差判斷：**
- **正價差 = 期貨 > 現股 = 偏多**（藍色線在白色線上方）
- 景碩案例：期貨 272.5、現股 270.5 正價差 → 當日漲停（字幕確認）

**做多/做空前 15 名排行榜欄位：**
- 排名 / 股票代號 / 股票名稱 / 券商 / **買賣超金額（萬）** / **1 日** / **5 日** / **日報比**

**券商排行大表欄位（@ch9-2 10:47）✅：**
- 股票代號 / 股票名稱 / 券商代號 / 券商名稱 / 認購/賣 / **買賣超金額（萬）** / **主力買賣超(%)** / **主力連買 N 日** / **外資買賣超(%)** / **外資連買 N 日** / **投信買賣超(%)** / **投信連買 N 日** / **自營買賣超(%)** / **自營商連買 N 日** / **月線斜率(%)** / **月線來績(%)** / **季斜率**

**分點類型識別：**
- **造市商（忽略）**：元大、群益、凱基（總公司）等
- **隔日沖特徵**：同分點一天買進隔天賣出；分散買別家券商的權證
- **周一+周三+周五出現做多排行 = 炎熱夏天（波段多方）**
