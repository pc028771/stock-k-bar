# 需要手動處理圖片的文章清單

用途：列出若要把課程觀念轉成股票分析系統規則，最值得手動截圖或保存圖例的文章。  
注意：此清單不保存課程圖片，只標記人工處理優先順序。

## 高優先

1. `📝【單一K線】十字線蘊含的轉折與延續意義`
   - 原因：十字線長短、連續十字線、隔日確認、區間高低點都需要圖例輔助定義。
   - 建議標記：短十字線、長十字線、連續十字線高低點、隔日突破/跌破。

2. `📝【單一K線】上影線與下影線的正確判斷`
   - 原因：影線不能直接等同壓力/支撐，需用圖例區分位置與盤中路徑。
   - 建議標記：創新高上影線、低檔下影線、上影線後續續攻/失敗。

3. `📝【單一K線】十字線與上影線在攻擊階段的意義(上)`
   - 原因：攻擊階段中的長十字線低點、防守位置與合併十字線需要圖形確認。
   - 建議標記：攻擊段、長十字線低點、突破長十字線高點、合併十字線。

4. `📝【單一K線】十字線與上影線在攻擊階段的意義(下)`
   - 原因：同樣是上影線，剛攻擊、攻擊中、拉抬後遇壓三種狀態需要圖例對照。
   - 建議標記：攻擊初期上影線、攻擊中上影線、拉抬後壓力區上影線。

5. `📝【單一K線】紅色誤解：連續紅K的判斷要點`
   - 原因：連續紅K是否真的攻擊，需觀察開盤位置、是否反覆開低、是否快速拉抬。
   - 建議標記：連續紅K但開低、真正攻擊連續紅K、突破K低點失守。

6. `📝【突破跌破】假性跌破的實務意義`
   - 原因：假跌破的判定高度依賴區間低點、跌破幅度、收回速度。
   - 建議標記：區間低點、跌破K、收回K、後續突破。

7. `📝【突破跌破】整理趨勢進入型態判斷的關鍵-假性跌破之後`
   - 原因：從整理到型態轉換需要連續圖形，適合建立 pattern label。
   - 建議標記：整理區間、假跌破、回到區間、突破點。

8. `📝【賣壓化解】K線圖的第一個研判要點`
   - 原因：賣壓區與過往套牢區需要在圖上標示，適合轉成 volume profile/前高區規則。
   - 建議標記：前高壓力、套牢區、穿越後是否守住。

9. `【成本原理】層層套牢的行進研判`
   - 原因：層層套牢需要辨識多個壓力層，文字摘要不足以定義區間。
   - 建議標記：每一層前高/成交密集區、突破與回測。

10. `【成本原理】賣壓中空的結構判斷`
    - 原因：賣壓中空最適合用圖形與籌碼/成交密集區輔助建立量化規則。
    - 建議標記：中空區、下方成本、上方壓力、突破後推升效率。

## 中優先

11. `📝【單一K線】跳空在K線的特殊地位`
    - 建議標記：攻擊缺口、一般缺口、事件缺口。

12. `📝【單一K線】跳空漲停一條線`
    - 建議標記：跳空漲停一條線、隔日續攻/失敗。

13. `📝【單一K線】高檔區域的長黑K`
    - 建議標記：高檔位置、長黑吞噬、跌破攻擊段低點。

14. `📝【關鍵K線】趨勢改變的關鍵K線`
    - 建議標記：趨勢改變前後、關鍵K高低點、防守位置。

15. `📝【關鍵K線】關鍵K線的意義與用途`
    - 建議標記：關鍵K線類型、進場/出場觸發。

16. `📝【型態判斷】頭部底部型態合併要點(一)`
    - 建議標記：頭部/底部區間、失敗型態、確認K。

17. `📝【型態判斷】頭部底部型態合併要點(二)`
    - 建議標記：合併型態、突破/跌破確認。

18. `📝【型態判斷】築底的應對與實務意義`
    - 建議標記：築底區、假跌破、攻擊起點。

19. `📝【型態判斷】區間整理走勢應有的認知`
    - 建議標記：區間上下緣、量縮、突破與失敗。

20. `📝【型態判斷】鑷頂型態的短線判斷解說`
    - 建議標記：相同高點、隔日確認、短線停損。

21. `📝【型態判斷】鑷頂與鑷底的特殊價位意義`
    - 建議標記：相同高/低點與特殊價位。

22. `📝【買點賣點】買點與攻擊研判`
    - 建議標記：攻擊買點、突破買點、攻擊失敗。

23. `📝【買點賣點】股價的買點決策(二)盤整買在突破`
    - 建議標記：盤整區間、突破K、突破後守住。

24. `📝【買點賣點】股價的買點決策(三)多頭買在攻擊`
    - 建議標記：多頭攻擊段、攻擊延續、停損點。

25. `📝【停損】停損位置的設定準則(一)`
    - 建議標記：停損位置類型、關鍵K低點、區間低點。

26. `📝【停損】停損位置的設定準則(二)`
    - 建議標記：停損移動、攻擊失敗、趨勢破壞。
