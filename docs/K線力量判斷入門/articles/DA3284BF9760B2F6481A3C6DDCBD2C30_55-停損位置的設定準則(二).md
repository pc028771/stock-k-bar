# 📝【停損】停損位置的設定準則(二)

![](https://go-image.pressplay.cc/image/D69A015978EA4FDE38E84A0CF2314532)

---

講到停損人人討厭，你可以看到大多數寫給投資人看的文章，都不喜歡討論停損這個話題，只會說一下這有多重要而已。明明很重要，卻只是輕輕帶過，因為人們習慣報喜不報憂，不喜歡看不想看的話題。但你必須要相信，這二十年來我看過實際在股市中倒下的人，都是因為該停損的時候沒有執行。台股三年以上多頭了，還屢創新高，人們漸漸地遺忘空頭市場的可怕，屢屢不停損股價又衝上去，就以為只有耐心與耐力才是股市的重要準則。不過請相信，中期角度只有你自己的正確觀念可以保護你自己，美股不行、利多也不行、樂觀的希望更不行。

上一篇談到了前兩種停損邏輯判斷，請記得複習。

今天我們來談第三與第四種**：****多空力量上的變化、攻擊力量研判。**

多空力量上的變化基於轉折與力竭的原理，要運用這個邏輯得要先學習轉折組合K線。不過我們這邊以空頭吞噬與大敵當前為例，當股價出現的力竭現象的時候，是一種停損(停利)的設置標準。空頭吞噬的定義是創新高的紅K，隔天被長黑完全包覆，請先看以下的說明圖片。

**107-06-14嘉晶(3016)走勢圖**

![](https://go-image.pressplay.cc/image/2819C70F6B8879B5893486E45D8888DC)

**空頭吞噬的圖片說明**

![](https://go-image.pressplay.cc/image/BB6BDFC81ED966D211E8E3309AC75DC2)

從上圖的走勢中，請大家留意最高價93.4元的那一天，為什麼討論時我用「停損」二字？這天開盤是開高，然後小小的創新高之後往下回檔補了當日跳空，也跌破了前一天紅K的低點、還連向上跳空缺口都回補**(三個現象同一根發生)**，不只如此，吞噬的隔天還繼續開低。基於短線交易的角度我才用停損這兩個字，對於當初在五月份買突破的人來說，是停利。不管停損或者停利的名詞，都是一樣的，意義是出場。基於多空力量變化的原理，也就是原本的攻擊資金結束了攻擊力量，還轉變為賣出的力量，這個多空力量上的變化向來是最強烈的一種改變，所以停損的設定基於這個原理是很重要的，當然，其他的轉折組合K線例如**大敵當前、雙鴉躍空、夜星棄嬰、暗夜雙星**都算是多空力量的變化形成的轉折組合，都可以做為停損點的設定標準。

**遇到轉折組合不出場就無法避開跌勢**

![](https://go-image.pressplay.cc/image/8098B34820E16220EC55E411C6F512BF)

攻擊力量的研判對於停損來說，主要就是連續走勢低點的判斷，不過這個有比較複雜的變化，被歸類於攻擊失敗與攻擊結束的範圍。[](https://www.pressplay.cc/project/vippPage/108-11-01%E5%AF%A6%E6%88%B0K%E7%B7%9A%E6%94%BB%E6%93%8A%E7%A0%94%E5%88%A4%E4%BA%8C%E6%94%BB%E6%93%8A%E5%A4%B1%E6%95%97%E8%88%87~/A7A3E66B7603C5B5DDB3C71F5D77F7AF)

簡易的判斷要從一根K線的低點開始，一根K線的高點叫做企圖、低點是意圖，有了意圖但不一定會執行，企圖心的持續展現從有沒有突破高點看。換言之，攻擊的角度會創新高，低點只是防守而已，如果前一天的低點破了的攻擊停止，當然是停利的位置。

**短線攻擊失敗的停損**

以上都是單純在短線交易中遇到攻擊失敗或者攻擊停止的例子，其中有許多單獨更細微的變化需要理解。透過大學光的走勢我們來說明特殊狀況的判斷，大學光(3218)在九月五日跌停板，前一天還在攻擊的階段，隔日卻開低跌停，主要是受了利空消息的影響，因為短線漲太多被警示為注意股之後公告七月營收，僅有0.18元股價就跌停板了。

**曾因利空消息而跌停的大學光(3218)**

![](https://go-image.pressplay.cc/image/3047E8512F036AD34CF85D0ACC2E3FAF)

原本是攻擊的走勢，**遇到突如其來的利空，**攻擊資金肯定是來不及出脫了，那麼表示看起來股價雖然拉回，攻擊股價的資金依然在場內，還是一種醞釀與否的判斷，屬於攻擊K線的範圍。依照日出與攻擊的意義，短線交易者依照低點被跌破而停利出場是正確的做法，從波段的考量才去考慮中期的力量判斷，但是必須知道，大資金如果無法出脫，未來就會自己想辦法解決這個問題，從大學光的K線就可以看出後來資金是怎麼處理這個問題的。

**108-10-31大學光K線圖**

![](https://go-image.pressplay.cc/image/AEA7C6F9C641557D6F5CF268F2EFB50C)

停損與停利其實是同一個標準，只不過差別在於獲利叫做停利、虧損叫做停損。與投資人買進的價位無關，所以停損沒有那種多少百分比停損的設定模式，不管進場的價位如何，停損的設置應該透過K線上來判斷，不是買進成本當作標準。如果讀者有感覺到停損的設定也得懂攻擊、趨勢、趨勢改變、關鍵K線，沒錯，技術分析本來就是一種標準，不會因為買進者的成本不一樣、心態的不同而有變化，這樣才是能夠執行的準則。

人的個性都很傾向於眼見為憑，剛剛我所提到的例子，很容易就事後的走勢再來評估一下標準的對與錯，但其實這樣是不妥當的，停損對於操作者來說是一種股價進行當下的標準，不能用最後你是賺錢還是虧損來判斷標準是否正確，而是要思考如果你沒有標準的話，真正不如預期的走勢你又能如何避開呢？台股昨日上漲157點，再創29年來新高，如果短線交易者沒有依照準則來執行停損，那麼同一天遇到光群雷、或是上週楠梓電這類型的狀況肯定避不開，更別說投入的資金能買什麼會飆漲的股票了，都卡在這了。
