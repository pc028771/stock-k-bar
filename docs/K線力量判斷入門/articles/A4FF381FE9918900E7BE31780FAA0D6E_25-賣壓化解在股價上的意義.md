# 📝【賣壓化解】賣壓化解在股價上的意義

![](https://go-image.pressplay.cc/image/471DC35FA519458E064450455FD05683)

---

理論上，K線的技術面意義不管用在哪一個商品似乎都是一樣的，實務上還是有不小的差別。型態學讓很多人以為是交易技巧的基本邏輯，其實每一樣「基本」其中都蘊含了深切的意義，只是大部份投資人並沒有在這些表面簡單的圖形中細細思考。以型態的頸線突破為例，既然是突破，就表示漲上去才有辦法突破，知道**「突破」才是買點**，又為什麼看盤的時候想的總是買低呢？

這個心態上的問題要解決才行。

**型態學的簡易說明圖**

![](https://go-image.pressplay.cc/image/6D078EC6FCA0E8A60EA53BA328E49C1F)

用這張圖我們先來說明股價，頸線(定義是季線上揚之後的前一個高點)突破之後，最簡易的認知並不是前面的底部型態，而是自此已經不存在過去套牢籌碼壓力(型態學中你明知道應該要買B突破，心裡總想著找A低點進場)。也就是說從A到B的這一段過程，拉抬者會面臨到過去套牢的賣壓，然而明知道拉抬過程有解套賣壓，依然有資金願意往上拉，這段過程就是**賣壓化解**。賣壓區段，是攻擊資金面臨的第一個問題，賣壓化解的意圖，是攻擊前必經的關卡。

**圖片中最後一根長紅還沒有出現之前，拉抬者肯定知道，過去套牢區段(黃框)的套牢籌碼解套就有可能會賣出，既然知道，資金又為什麼願意往上拉抬呢？這就稱之為賣壓化解的意圖。**

![](https://go-image.pressplay.cc/image/E1DB6F2A6F58B0601DDEED135934EA70)

與成交量無關，最後這根紅K解決了黃框中所有曾經套牢的賣壓，稱之為賣壓化解的意圖，形態上也是突破。

**了解賣壓化解的意義，對於這個根紅K的出現就不會感到意外。**

![](https://go-image.pressplay.cc/image/15B4831B53DEBCD26EE9D34A8363BD66)

型態學中的頸線突破，是基於賣壓化解的原理，這讓股價的走勢判斷，越過了賣壓化解的區段，表示上檔已經沒有套牢的籌碼，籌碼都到了攻擊者的手中，這樣就能明白為什麼股價K線中最強的一段，通常都在突破創新高之後出現。下圖是前年最具技術分析代表性的泰碩(3338)，橘色線下的那一段就是賣壓化解，能在買進之後有最舒服的感覺就是在賣壓化解之後才進場，是否對於選股選強勢、汰弱留強、買點的重要性都有更進一步的認識呢？

**後來股價突破百元的泰碩(3338)在8月16日K線圖**

![](https://go-image.pressplay.cc/image/4058F338921BBB66FEB3C65E26D1527F)

頸線之下賣壓化解的意義，是因為過往的套牢籌碼只要解套就會賣出來，拉抬股價的資金肯定需要面對的障礙，這是股票股價的技術分析使用的判斷。

**大盤的K線型態卻沒有這個問題**，因為大盤是加權股價，表示過去的指數高點，如果要越過，並不需要拉抬同樣的股票才行，也就是說「上一波跌的如果是大立光，那這次可以用台積電的上漲突破」(以上為舉例)，沒有一定需要面臨賣壓化解的問題。這表示大盤(包括期貨)與股票的技術分析技巧依然有細微的不同之處，使用的時候得要分辨技巧上的差異，這也是大盤K線更加重視**趨勢判斷**的原因。

**再次突破的賣壓化解意義**

當多頭格局持續，往往有很多股票不只是一次的出現賣壓化解，這是技術分析中重要的判斷項目。我們不能探討為什麼投資人總是有著追高的恐懼，跟著想要找尋低點買進，而是要透過股價的賣壓結構被資金有意圖的化解過往的套牢賣壓。

![](https://go-image.pressplay.cc/image/595AB9D9A0E1D719ACEFEC7FBD844FE1)

**小結語**

賣壓化解是攻擊前的意圖，有了賣壓化解不一定就會攻擊，但是，不能不理解賣壓化解在股價上的意義，也就是市場上有人用資金來為過去套牢的人解套，在高價股與低價股的表現會有差異，是因為低價股籌碼容易凌亂。但這不影響我們對於賣壓化解在型態學上的意義應有的認識。
