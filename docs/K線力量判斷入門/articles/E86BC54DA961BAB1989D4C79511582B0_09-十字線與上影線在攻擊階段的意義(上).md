# 📝【單一K線】十字線與上影線在攻擊階段的意義(上)

**前言**

講解K線的時候，總感覺有很多細項K線的原理可以更加詳細說明，尤其是遇到震盪的大盤，個股的走勢判斷就需要對K線有足夠的認識才能應對。不過世界上的事情就是這樣，股價如果是平凡無奇的單向發展並沒有很多變化範例可談，整理時期的時候個股多空分歧都需要技巧，卻無法將所有的技巧一篇之內講完，只能逐一解說日起才能有功。

期望讀者能明白：**「你需要持續累積自己對K線的判斷實力」**，不能想著關鍵時刻來了再等別人提醒，而這個K線研判與決策的能力是操作必備的功課。

當然我理解，有的人會覺得我講的這些K線教學很深，其實並不是如此。我覺得這些教學內容都很實務，交易上你都會需要的。因為市場上的投資人長期以來很習慣想要用簡單的方法就來快速判斷漲跌，所以明知道不能用，卻還是抱持著參考一下也好的觀念。

例如有人說：「缺口回補三日內要漲上去表態」，這就是一種很好笑的說法，卻有人相信，那萬一三天內有反彈、第五天才跌破怎麼辦？萬一缺口一補就開始狂跌，你有辦法等三日嗎？或是有人解盤說：「拉回不破支撐是買點」，那我想問的是如果你買了之後，隔天才破了怎麼辦？有人這時候又會說：「那就要善設停損。」

好吧，既然不管用怎樣的邏輯操作都得要善設停損，那你為什麼不選擇股價有攻擊力量正在發揮的股票呢？反而要選弱勢沒人要的時候進場呢？

這篇文就是要來講解這樣的邏輯，攻擊階段的K線意義你不能發生錯誤的判斷，因為當你判斷有誤、加上人性想要有高就先賣一賣獲利了結的心理，不小心就會把正在攻擊的股票出掉，卻留下還在套牢的持股，怎麼看都很難避過未來的空頭市場。

**十字線的基本定義簡述**

十字線是一個多空力量對決的現象，需要研判的位置通常在漲多或跌深之處，簡單區分有兩種：上下影線都短、上下影線都長，屬於中繼休息或轉折意義。
**(一)上下影線都短的，具備較強的轉折意味，因為多空雙方都沒用力出手，顯示兩股力量當天都保持觀望。
(二)上下影線都長的十字線，最不容易判斷，因為多空雙方都在當天有大量出手，最後勢均力敵，未來看哪一方有力才能判斷。**

📌所以處在攻擊階段的十字線，如果是短十字線又遇到以往的壓力區，就不宜再回檔。
📌如果是長十字線就以十字線的低點為防守標準，以下會用詳細文字說明。

**從攻擊角度看待十字線**

以攻擊的角度來說，當股價來到了高檔區域，出現了短十字線，表示這一天多方並沒有強烈的拉抬，這個多方也就是過去的拉抬者的力量。

必須先理解，未來的獲利了結賣壓就會是來自這個多方。

短十字線屬於多空觀望的型態，明天不能往下走，這個往下走會變成是獲利了結賣壓出籠的型態。相對的，長十字線是一種當日多空激戰，判斷時必須思考，現在會是誰最在乎股價不能跌呢？當然是已經持有很多籌碼的攻擊方。這股力量在長十字線的這一天還很刻意的花錢維護了讓股價不跌，那麼花了更多錢之後，接下來繼續攻擊的可能性不就更高嗎？

唯一要留意的一點是長十字線的當天，如果成交量沒有明顯的放大，那麼就得要考慮這個長十字線的高低點了。股市裡，價格才是最真實的存在(有拉抬的意願不會先回檔很深給大家低買)。所以在這樣的狀況下，高點就得盡快突破，震盪中十字線的低點不能跌破。

我們透過這張K線圖可以發現，突破剛剛創新高也在這一天出現，所以**突破的角度、長十字線的角度**都可以看得出來這還是攻擊的態勢，基於買在攻擊的交易準則，已持有者就不用等待明天有沒有跳空出現，而是未來的停損位置也因為長十字線的意義，有了明確的價位，沒跌破之前什麼都不要想太多。

**合併十字線**

**108-11-05慶豐富(9935)的K線走勢**

![](https://go-image.pressplay.cc/image/977A0FD2C135A5A875DE93B4C924FD73)

除了單一根K線的變化之外，有另外一種變形需要考慮，也就是由一根上影線、一根下影線兩根K線合併起來的走勢等於一根長十字線的狀況。長十字線需要單獨判別的狀況通常都在已經多頭創新高之後、或者是空頭創下新低之後才需要這麼認真的去研究其中的意義，尤其是多方攻擊階段更是好用，因為可以辨別多方力量的前進或者停止點。不過，這個合併變形本質上有點不同，雖然合併看是一樣的，畢竟還是經過多了一天的走勢，就停利點的角度來說用法相同，但就攻擊力量的角度來說，積極性略低，有可能轉為橫向震盪。

不過我們並不需要因此事先預設股價太多，依然使用攻擊的角度來判斷股價的變化即可。也就是如果「**低點依然保持推升**」就沒有太大的問題，直到這個慣性消失再來討論。如果沒有持續出現攻擊的跡象，那麼這個合併起來視為長十字線的低點還是目前的停利點。

其餘的要件，就要依照攻擊的判斷原理來決定即可。
