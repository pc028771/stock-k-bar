## 9) Agent Handoff Minimal Flow

Run in order:

1. Syntax check
2. Nightly warmup (or verify cache exists)
3. Run breakout scanner with `balanced`
4. Review:
   - `breakout_daily_scanner_topn_summary.csv`
   - latest rows in `breakout_daily_scanner_recent20d.csv`
5. If ranking quality degrades, rerun with:
   - `--strict-filter-profile aggressive`
   - or increase warmup coverage (`--max-per-date`, `--days-back`)
